# 视觉风格圣经 · The Academy of Curiosities

This skill is **read-only canon**. Image prompts, generated art, and reviews must all
conform to what is written here. When the canon and a one-off request conflict, the
canon wins unless the 主编 explicitly overrides it in writing.

Baseline: 项目正典 V1.0（2026-06-04）。废弃角色「阿文」不存在；其职能并入葛西泉藏与陸珣。

---

## 1. 美术风格（锁定）

- **媒介**：柔和水彩（soft watercolor / 日本绘本 ehon 风），有可见的水彩纹理与晕染、柔和的边缘。
- **写实度**：半写实童书风。**不是** anime/萌系/chibi，**不是** 3D/CG，**不是** 厚涂厚重油画。气质接近高质量日本绘本与吉卜力背景的"温柔写实"。
- **光线**：自然光为主，温暖、柔和；清晨、午后斜光、黄昏暖光皆可，避免硬阴影与戏剧化高反差。
- **色调**：温暖、克制、低饱和的柔和色；米白/原木/淡蓝/嫩绿/樱粉为基底。避免霓虹、荧光、强对比的赛博感。
- **情绪**：安心、好奇、温柔、日常。**零恐怖**：无鬼怪、无血腥、无惊吓、无超自然。
- **构图**：留白舒适，视线引导清晰；人物表情含蓄真实（不夸张瞪眼/不浮夸）。

## 2. 世界设定细节（必须出现的"真"）

地点恒为**名古屋的日本市立小学**。画面里应自然出现这些文化细节（按场景取用）：

- 上履き（室内鞋，白底彩色脚尖）、玄关下足箱（げた箱）
- 引き戸（横拉教室门，金属拉手）、窓側一列靠窗课桌、金属窗框
- ランドセル / 帆布书包、木地板、绿色黑板与平假名板书
- 樱花（4月）、季节物候要与卷次月份吻合（见各案件卡）
- 给食、扫除、运动会等日常校园元素

**禁止**：中式校园元素（红领巾、中式校服、简体标语牌）、西式教室、与设定不符的服饰。

## 3. 角色锚点（逐人锁定）

每个角色有：**身份 / 外貌 / 服装 / 招牌道具 / 招牌动作 / 专属色**。专属色用于背景色光、封面、周边的统一识别，不一定是衣服颜色。出图时**招牌道具或动作至少出现一项**，以保证可辨识。

### 観察社 · 创始四人

**伊藤光（いとう あきら）** — 专属色：暖橙
- 5年2組。观察社发起人，外向、行动力强、温暖的连接者。
- 短而蓬松的栗棕色头发，眼神明亮，常带笑。
- 招牌道具：**橙色围巾**。招牌动作：发现东西时眼睛一亮、招呼同伴。
- 口头禅：「え、何か見つけた？」

**加藤慧美（かとう けいみ）** — 专属色：青绿
- 5年1組。实验验证型，冷静、慢半拍、信"做出来给我看"。
- 整齐的深色齐肩短发（bob）。
- 招牌道具：**银框眼镜**、烧杯/试管或记录本。招牌动作：**用食指推眼镜**。
- 口头禅：「まだ検証してない。」

**松本志郎（まつもと しろう）** — 专属色：草绿
- 5年3組。结构还原型，风风火火、爱动手。
- 短而带刺的深棕/黑发，常卷起袖子。
- 招牌道具：**扳手/螺丝刀**等工具。招牌动作：**专注拧螺丝时舌尖微微伸出**。
- 口头禅：「見に行こう！」

**陸珣（りく しゅん）** — 专属色：靛蓝
- 5年2組。微观领域专家型（昆虫/自然知识），一周前**从大阪转学**而来，沉默寡言，对外台词极少。
- 深色头发、刘海略压眉，神情安静内敛；常穿浅蓝衬衫。
- 招牌动作：**左手轻搭在书包侧袋上**（自我安抚的"安全锚点"）。招牌道具/母题：**身旁一只被他注视的小蚂蚁**。
- 关键台词：「そこにある。」 / 「……くっついてる。」
- 成长弧：从"不被注意"到"被需要"；动作上从"摸侧袋"到"松手伸出"。

### 后续登场 & 长辈

**陸瑆（りく ひかる）** — 专属色：暖金
- 4年2組。陸珣的妹妹，记录者→后期人际逻辑型侦探（双层叙事的"原理层"承担者）。
- 活泼好奇，常扎双马尾或戴发箍。
- 招牌道具：**翻开的笔记本 + 铅笔**（几乎不离手）。招牌动作：边看边记、把重要的一页折角。
- 口头禅：「ちょっと待って、それは逆かも。」

**山本理紗（やまもと りさ）** — 专属色：薰衣草紫｜**11卷起登场**
- 感知解析型，对光、色、错觉敏感。
- 较长的深色头发，别一枚发夹。
- 招牌道具：**举起的色卡/颜料**。招牌动作：眯眼比对颜色与光。
- 口头禅（提示）：「目が、だまされてる。」

**中谷琦（なかたに …）** — 专属色：暖灰褐｜**11卷起登场**
- 证据固定/记录者，靠照片说话。
- 干净的短发。
- 招牌道具：**挂在脖子上的相机**。招牌动作：举相机、翻看拍过的照片。
- 口头禅（提示）：「あの時、撮ったよ。」

**葛西泉藏（かさい せんぞう）** — 专属色：赭褐｜**社区长辈（不定期出场，历史/旧物案关键）**
- 街区里开杂货铺的和善老人，溯源考据型（旧地图、旧物、城市记忆）。
- 灰白短发，温和；**圆框眼镜**；常系**工作围裙**。
- 招牌道具：**旧地图/旧物/卷起的纸**。招牌动作：戴上眼镜端详旧物、缓缓讲过去的事。
- 红线：成人**永不替孩子破案**，只提供场景、旧物与有限线索。

## 4. 红线（出任何图都不得违反）

1. **零恐怖**：无鬼怪、血腥、惊吓、超自然。氛围始终温暖安全。
2. **儿童安全**：所有未成年人形象健康、得体、着装完整，**绝不性化**；自然童真，不卖萌媚态。
3. **大人不破案**：成人可出现、可提供旧物/场景，但推理由孩子完成。
4. **文化校准**：日本名古屋校园细节准确；禁止中式/西式错置。日文板书/招牌如出现需正确（不确定就留白或不放文字）。
5. **谜面线索可视**：若该图对应某案，**三条公平线索中可入画者应被画出**（如"唯独这把椅子湿""窗玻璃内侧有雾""金属贴窗框无缝"），但不直接剧透答案。

## 5. 一致性要点（跨图统一）

- 同一角色在不同图里：发型、脸型、专属色、招牌道具/动作必须一致。
- 生成新场景时，**优先以该角色的"主参考图"作为输入参考**（见 `illustration-nano-banana` 技能），再叠加场景描述。
- 服装可随季节/场景变化，但**招牌元素**（光的围巾、慧美的眼镜、珣的侧袋手势+蚂蚁、瑆的笔记本、理紗的色卡、中谷的相机、葛西的圆镜+围裙）应保留可辨识。
