# 学堂奇事録 · The Curious Logbook

> *The Academy of Curiosities · A Curiosity Hall Mystery Series*  
> 名古屋市立小学校 · 儿童图文校园推理章节书

**首市场**：日本  
**核心读者**：10–12岁，宣传覆盖9–12岁  
**当前阶段**：V2.0结构候选与内部MVP原型，非出版定稿

---

## 项目定位

《学堂奇事録》（中文：**学堂趣事录**）发生在名古屋市立小学。孩子们组成校园趣事观察社，用采访、观察、实验和公平线索解释校园中的异常事件。

V2.0核心：

> **怪事不会伤人，误会会。**

每案以强异常和真实儿童压力进入，用跨学科机制澄清事实；每卷五案形成关系网与学科网，最终区分错误责任与真实责任，并完成可见的关系修复。
