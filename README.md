## 项目简介

《学堂奇事録》（中文品牌：**学堂趣事录**）是面向日本小学生的跨学科推理系列。

故事发生在名古屋市立小学校。孩子们组成 **学校おもしろ観察クラブ**（校园趣事观察社），用物理、化学、生物等可验证知识，破解日常里的「有点奇怪」——**没有鬼怪，没有超能力**。

**Vol1（正典）**：《觉得奇怪，就先观察》— 转学生陸珣入社 + 5 个小案（非单卷湿椅子案；湿椅子见素材库）。

> **年级正典 V1.0（2026-06-04）**：观察社 **5 年跨班**（珣·光 5-2 · 慧美 5-1 · 志郎 5-3）· 瑆 **4年2組非社员**。  
> **废止**：旧「4年2組 四人同班 / 四人团」结构 — 见 [`01_角色设定/00_年级班级关系表_V1.0.md`](01_角色设定/00_年级班级关系表_V1.0.md)
