## 项目简介

《学堂奇事録》（中文：**学堂趣事录**）发生在名古屋市立小学校。孩子们组成 **学校おもしろ観察クラブ**，用可验证的观察与实验破解日常里的「有点奇怪」——**没有鬼怪，没有超能力**。

**Vol1（正典）**：《觉得奇怪，就先观察》— Plan B **五案** + 序（非湿椅子单卷；湿椅子见 C001 素材库）。

> **年级 V2**：观察社 **5 年跨班**（珣·光 5-2 · 慧美 5-1 · 志郎 5-3）· 瑆 **4年2組 · 非社员**  
> **废止**：4年2組 四人同班 — [`01_角色设定/00_年级班级关系表_V1.0.md`](01_角色设定/00_年级班级关系表_V1.0.md)
