---

## 项目简介

《学堂趣事录》是一套面向7-12岁儿童的跨学科推理绘本系列。

故事发生在日本名古屋的一所小学。孩子们组成「怪事研究社」，用物理、化学、生物、工程、心理学等跨学科知识，破解校园里发生的日常谜题。团队随卷逐步成长（卷1 起 4 人，最多 6 人）。

**没有鬼怪，没有超能力。每一件"怪事"，都是科学在和你打招呼。**

## 核心差异化

| 维度 | 学堂趣事录 | 传统同类产品 |
|------|-----------|-------------|
| 推理方式 | 跨学科硬知识破案（3+原理/卷） | 单一学科或纯逻辑 |
| 恐怖元素 | **零恐怖**，睡前放心读 | 常有悬疑恐怖包装 |
| 实验验证 | 每卷末附家庭可复现实验 | 少数附带 |
| 叙事结构 | 第三人称限知 + 妹妹第一人称笔记 | 单视角 |
| 知识融入 | 角色对话自然带出，无说教 | 知识卡片/旁白解释 |
| 原理管理 | AI管理50卷不重复原理清单 | 常有原理复用 |
