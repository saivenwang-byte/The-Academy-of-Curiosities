## 1. Q1 定案 · 案首版式（A）

**每案开头 = 半页内标题条**，占版心上半或上 1/3，**不另起整页扉页**。

```
┌─────────────────────────────┐
│  [半页标题条 · 约 50% 版心高]  │
│  第 N 起 · めくれたポスター     │
│  （可选：一行异常句 · 无剧透）   │
├─────────────────────────────┤
│  正文 §1 开端 …               │
│  …                           │
└─────────────────────────────┘
```

| 规则 | 说明 |
|------|------|
| 位置 | 该案 **第一 spread 上半** 或 **单页顶 1/3** |
| 内容 | 案号（第 N 起）+ **日文案名**（正典）+ 可选 **中文副题**（内页小字） |
| 禁止 | 整页空白扉 · 每案重复封面级大图 · 案内 `### 1–4` 升格为独立章首页 |
| 与序关系 | **序** 仍占卷首独立篇；**案①** 从半页标题条起，不二次「卷首」 |

**PRODUCT-GATE 对齐**：P-11 目录可扫读 · P-12 版心/字号/行距 · 案内节拍 **不进目录**（见 §4）。

## 8. 明确排除（Vol1 不做）

| 排除项 | 说明 |
|--------|------|
| **语音包** | 无朗读音频 · 无 QR 配音 · 无 TTS 附件 |
| **实体道具包** | 无印刷可撕线索卡实物 · 无实验材料盒 · 无线圈本赠品 |
| **线圈装帧设计** | 内文 **胶装/无线胶** 桥梁书 · 不设计螺旋线圈交互 |
| **每案扉页** | Q1=A · 半页标题条即可 |
| **PPT 16:9 正文** | 路演用 PPT ≠ 儿童阅读产品（见出厂标准 Q3） |

**数字道具**：仅 **8 张** 手绘电子 PNG 草图 · 见 [`Vol1_观察社数字道具草图_最小包_V1.0.md`](../07_设计原档/Vol1_观察社数字道具草图_最小包_V1.0.md) · **嵌入 PDF/EPUB** · 非实体。
