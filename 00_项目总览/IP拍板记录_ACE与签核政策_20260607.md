## 4. 签核政策覆写（IP Owner override）

> **风险声明**：商业推广、合同、侵权与人格权风险 **仍由 IP Owner 承担**；E25 onboard 后可修订本覆写。

### 4.1 蒸馏复合专家体可签 Gate A 的角色

以下 Gate A 签核位，在 **对应蒸馏 Officer + 能力原子就绪并 lint PASS** 后，可由 **蒸馏复合专家体** 签字，**不要求**另寻真人单独签字：

| E 编号 | 环节 |
|--------|------|
| **E04** | 日文正文 / 语言表记 |
| **E07** | 文化 / 公平线索 |
| **E09** | 资料 / 校园素材 |
| **G1** | 线稿 PNG 验收 brief 对齐（产出仍须 G1 画师 PNG；ACE 签 brief/PASS 条件） |
| **E20** | 试读组织者 / 年级 band（Gate A 口径） |
| **E25** | Gate A 前合规扫描（送社前合同仍 👤） |

签核包标注：`signoff_type: ace_distilled` · 附蒸馏卡 ID + lint 报告。

### 4.2 不变项

- **E06 终签** · **IP Owner PASS/HOLD** — 不可代签  
- ACE 输出 = `STATUS: DRAFT · NOT SIGNOFF` 直至签核位明确 PASS  
- 禁止对外暗示真人 endorsement
