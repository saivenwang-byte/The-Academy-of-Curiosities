L1  硬指标确认   00_…/创作硬性指标确认书.md

## 二、标准创作流水线（强制顺序）

### 阶段 0 · 策划（先讨论，后写稿）

| 步骤 | 执行 | 产出 |
|------|------|------|
| 0.1 | `academy-series-architect` | `卷任务包_总策划.txt` |
| 0.2 | 对照 `名古屋探索圈层地图.txt` 选地点格 | 任务包内 B/M/K… 标注 |
| 0.3 | `academy-research-editor` | 素材调度单 + 补 `09_/` |
| 0.4 | 用户确认任务包 | 状态：可写 |

**未确认任务包不得写正文。**

### 阶段 1 · 写作前阅读（30–60 分钟）

- [ ] `名古屋写作硬指标_本格科学参考.md` — **当月**气候/日出日落/物候
- [ ] `docs/world_reference/00_MASTER_ENVIRONMENT_INDEX.md` — 填 **Scene Card**（07 模板）
- [ ] `docs/volume_planning/` 该卷 Case Card（08 模板）
- [ ] Writer Quick Ref 01–05 + maintenance 细则（按线索类型）
- [ ] `日本背景知识库` 相关章节（学期/空间/当番…）
- [ ] `11_写作素材速查/按卷可用的元素清单.txt` 该卷段
- [ ] `00_项目总览/人物名称定稿.txt`
- [ ] 卷任务包全文
- [ ] 若涉怪谈结构：`07_/` + **零恐怖改写表**

### 阶段 2 · 初稿

| 项 | 标准 |
|----|------|
| 执行 | `academy-engine` |
| 视角 | 第三人称限知 · 陸珣 |
| 字数 | 6000–8000 汉字（阶段 I） |
| 结构 | 发现→假设→验证→突破→结论→意象收束 |
| 原理 | ≥3 学科，列于卷任务包 |
| 范式 | 7 选 1–2，见 `七种推理范式.txt` |
| 笔法 | 5 类，见 `写作技巧与叙事笔法.md` |

### 阶段 4 · L1 三并列门禁

1. **Project World Metrics** — Startup Gate + 05 红线清单 + 06 矩阵核对  
2. **硬指标七检** — `名古屋写作硬指标` §十二  
3. **田中みどり** — `japan_campus_consultant_agent.html` 五维消红  
4. **本格公平** — `04_MYSTERY_SCIENCE_CASE_STANDARD.md`  

### 阶段 4.5 · L2 质量评分卡

- **日常门禁**：`docs/00_VOLUME_QUALITY_SCORECARD.md`（24 项 + §7 文学语感）  
- **全维度深评**：`docs/00_VOLUME_QUALITY_SCORECARD_APPENDIX_44.md`（44 项 · 两版对比 / 出版前）  
- **产出**：`docs/volume_planning/volume_XX_scorecard.yaml`  
- **语感决策**：`03_…/语感决策记录_VolX.md`（Vol1 已建 · 防绘本体）  
- 门槛：L2 原始分 ≥180、7.3≥7、无一票否决 → `READY_FOR_SAMPLE`  

### 阶段 5 · 红线与质量自检

使用 `academy-engine` 质量清单 + 本节「验收清单 B」。

### 阶段 5 · 附录

- [ ] 陸瑆笔记 800–1200 字（第一人称，克制）
- [ ] 「你可以这样做」实验页 ≤5 步，家庭材料

### 阶段 6 · 定稿入库（中文版）

- 路径：`03_故事内容/第X卷_*/完整文字稿.txt`
- [ ] `名称扫描` 无旧名
- [ ] 卷任务包状态改为「中文定稿 · 待日译」

### 阶段 7 · 日文版翻译与全文督查（不可跳过）

> **门禁全文**：`docs/00_JP_TRANSLATION_REVIEW_GATE.md`

| 步骤 | 产出 |
|------|------|
| 7.1 中→日翻译 | `完整文字稿_日本語.txt` |
| 7.2 **田中みどり五维全文督查** | `文化校准报告_日本語.txt`（按章粘贴 `japan_campus_consultant_agent.html`，五维全选） |
| 7.3 日文加检 J1–J7 | 直译腔 / 敬语 / 科学术语 / 儿童可读 等 |
| 7.4 归档 | `09_CONSULTANT_REVIEW_LOG.md` §日文版 |

**未过阶段 7，不得标 `READY_FOR_TRANSLATION` 或对外送日文样章。**

中文版 L1 通过 ≠ 日文版可发布。

## 三、验收清单 A · 红线（一项不过 = 打回）

| # | 检查项 |
|---|--------|
| R1 | 零恐怖：无鬼怪、无不可解超自然、无血腥暴力 |
| R2 | 大人永不破案：核心推理与验证在孩子 |
| R3 | 无霸凌、无政治敏感 |
| R4 | 谜面与竞品无高度重复 |
| R5 | 日本校园文化已校准，无中国校园模板（引き戸/給食/係） |
| R6 | **无阿文**；团队最多 6 人；葛西非第 7 成员 |
| R7 | 人名与 `人物名称定稿.txt` 一致 |

## 四、验收清单 B · 内容与结构

| # | 检查项 |
|---|--------|
| S1 | 核心原理 ≥3，且与已出版卷不重复 |
| S2 | 谜底完全科学可解释 |
| S3 | 伊藤光 = 第一发现者；陸珣 = 限知视角 |
| S4 | 各角色功能与专属台词风格有展示 |
| S5 | 结尾一个意象收束，无说教感叹 |
| S6 | 实验可家庭复现，标注课标年级 |
| S7 | 季节/月份与 `50卷大纲`/任务包一致 |
| S8 | 地点在 `探索圈层地图` 已登记 |

## 五、验收清单 C · 日本文化（抽样）

| 检查项 | 正确 |
|--------|------|
| 教室门 | 引き戸，非「推开」 |
| 鞋 | 玄关上履き |
| 午餐 | 給食/配膳，非食堂打饭 |
| 职务 | 係/学級委員，非班长 |
