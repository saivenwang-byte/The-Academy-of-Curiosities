## 2. 物理与字符规格

### 2.1 整卷

| 项 | Vol1（入门） | Vol5+（稳定） |
|----|-------------|---------------|
| 开本 | 新书判 / 略大于文库 | 同左 |
| 总页数 | **144–160** | **152–176** |
| 正文页 | 110–125 | 110–135 |
| 图+瑆+线索页 | 25–35 | 25–35 |
| 正文日文字符 | **32k–38k** | **38k–45k** |

### 2.2 版式比例

| 内容 | 比例 |
|------|------|
| 正文 | 65–70% |
| 插图 | 15–18% |
| 陸瑆日记 | 7–10% |
| 线索/复盘 | 7–10% |

### 2.3 阅读阻力（E03 · E04）

- 每 **4–6 页** 视觉变化 · 每 **8–12 页** 完整插图  
- 连续纯文字 **≤ 6 页** · 每案 **6–10 页**（排版页）  
- 每 **1,000–1,500 日文字符** 一次情节推进  
- 每 **20 分钟** 至少 **2 项** 回报（线索/笑/冲突/错推/图/情绪/答案/钩）

## 5. 日文句段 · 读音标注（E04 硬门禁）

> **读音标注专档**：[`日文读音标注策略_V1.0.md`](./日文读音标注策略_V1.0.md)  
> 对标《放课后ミステリクラブ》全汉字注音 · 本 IP **不全书全注音** · 取 **年级适配 + 首次标音**。

| 项 | 标准 |
|----|------|
| 普通句 | 20–35 字符 |
| 复杂句 | ≤ 45–50 字符 |
| 段落 | 1–4 句 |
| 对话 | 35–45% |
| 读音标注 | **年级适配 + 首次标音** · 人名/地名/术语 **案首复现** · 常用汉字不重复标 |
| 密度 | 明显低于全注音儿童推理 · E04 以「小五独立读不卡」终检 |
| 文字语法 | **J10 标日对齐** · 仅假名+日汉字 · 见 [`日文正文文字与语法标准_标日对齐_V1.0.md`](./日文正文文字与语法标准_标日对齐_V1.0.md) |
