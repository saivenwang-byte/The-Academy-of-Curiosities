# IP新定位与价值承诺 · LOCK候选

> 状态：`CANDIDATE_FOR_LOCK`  
> 适用：主编、商业、视觉、出版提案、全体创作Agent

## 1. 品类定义

内部：**校园涟漪洗冤本格 / 儿童社会判断冒险小说**。

核心承诺：

> 每卷五件校园怪事；每案一条正在失控的传言、压力或错误判断；观察社在倒计时结束前，用公平线索与跨学科机制澄清事实、责任与关系。

## 2. 三层市场表达

### 给儿童

- 全校都说是他。你相信吗？
- 怪事已经发生，倒计时已经开始。
- 五件怪事，最后可能是同一张错误故事网。

### 给家长

- 让孩子被怀疑时会说清楚，怀疑别人时会先停一下。
- 用故事练习事实、推测、责任与修复。
- 高刺激但零血腥、零重犯罪、零真实超自然。

### 给出版社/学校

> 以校园怪谈和不可能画面为入口，以真实儿童压力为情绪动力，以公平推理和可验证机制为事实基础，以责任澄清和关系修复为社会价值的系列儿童小说。

## 3. 不再主打的表述

禁止单独宣称：

- “世界首创双层叙事”；
- “独一无二跨学科推理”；
- “科学知识最多”；
- “儿童心理治疗”；
- “完全根据真实事件改编”；
- “适合所有年龄儿童”。

以上只能作为结构或候选优势，不得当未经验证的市场事实。

## 4. 情绪用途

每案至少兑现两项：

1. 被误会者获得被理解的机会；
2. 读者体验一次“不急着站队”；
3. 朋友关系经历选择与代价；
4. 错误责任被洗清；
5. 真实责任被承认；
6. 关系以一个可见动作得到修复。

## 5. 系列价值阶梯

| 层 | 读者得到什么 |
|---|---|
| 入口 | 强画面怪事、倒计时、反转 |
| 中层 | 观察、比较、证据、实验、表达 |
| 深层 | 群体判断、责任边界、友情与归属 |
| 长线 | 五案关系网、校园记忆、人物成长 |

## 6. 品牌总句

- **怪事不会伤人，误会会。**
- **真相不是找到谁该被责怪，而是弄清每一步到底发生了什么。**
- **学校里没有坏人，只有没被看见的原因。**
