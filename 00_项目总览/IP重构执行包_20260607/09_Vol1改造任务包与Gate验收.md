# Vol1 改造任务包与 Gate 验收

> 状态：`EXECUTION_PLAN`  
> 原则：先改A001做证据，再决定是否重构A002—A005；禁止一次性全卷推倒重写。

## 1. 项目目标

把现有Vol1从“五个并列校园小案”升级为：

- 每案独立成立；
- 每案有强异常、压力、洗清错误责任与真实责任；
- 五案组成关系网；
- 五案共同形成学科编织结构；
- A005完成卷级事件、学科、人物三重回收。

## 2. Phase 0 · 基线审计

### 任务

- [ ] 读取A001—A005最新正文、Case Card、文学审计和科学审核；
- [ ] 建立旧版字段→V2字段差距表；
- [ ] 标记哪些内容可保留、加强、移动或删除；
- [ ] 确认现有五案每案主机制、人物功能、情感锤；
- [ ] 确认哪些正典不得修改。

### 交付

- `Vol1_V2迁移差距矩阵.md`
- `A001-A005_旧新字段映射.csv`
- `Vol1_不可改正典清单.md`

### Gate `MIG-G0`

- 唯一正文源明确；
- 五案版本号明确；
- 无未来日期/状态误覆盖；
- 无“现稿已出版定稿”误报。

## 3. Phase 1 · 卷级终局反推

先设计A005新终局，不写正文。

### 必答问题

1. A005最大异常是什么？
2. 谁会在倒计时结束时失去什么？
3. 大家错误地把哪几件事编成一个故事？
4. 前四案留下哪些3—4个变量？
5. 五案使用哪3—5种关系？
6. 五案学科如何互证、冲突、误用、汇流？
7. 卷级元问题是什么？
8. A005怎样回收人物与陸瑆情感线？

### 交付

- `Vol1_五案关系网_V0.1.md`
- `Vol1_学科编织图_V0.1.md`
- `Vol1_A005回收矩阵_V0.1.md`
- `Vol1_卷级变量台账_V0.1.md`

### Gate `MIG-G1A`

- 前四案变量≤4；
- 关系类型3—5种；
- 有明确元问题；
- A005不靠新证据硬解；
- 五案仍能独立闭合。

## 4. Phase 2 · A001增强设计

### 必须新增或强化

- 8秒可复述的强画面；
- 被错误指向的人；
- 具体倒计时；
- 主压力与副压力；
- 1主机制+1交叉机制；
- 错误责任与真实责任；
- 朋友/班级立场冲突；
- 可见修复动作；
- 陸瑆的第二真相；
- 1个卷级变量。

### 交付

- `A001_CaseCard_V2.0.md`
- `A001_分场表_V2.0.md`
- `A001_钩子与压力测试稿.md`
- `A001_科学公平线索表_V2.0.md`
- `A001_陸瑆笔记三稿.md`

### Gate `MIG-G1B`

- 认知预算未超限；
- 科学答案独立成立；
- 人际故事未替换机制；
- 开篇500日文字符内出现异常、时间、场所、站队句；
- 结尾无说教。

## 5. Phase 3 · A001增强原型正文

### 写作要求

- 保留现有人物灵魂与日本校园真实感；
- 不为新定位强加成人阴谋；
- 高压与幽默交替；
- 所有核心证据在揭晓前公平出现；
- 原理解释控制在正文3%—5%；
- 情绪通过动作、空间和物件表达。

### 交付

- 中文文学稿；
- 日文母语编辑稿；
- Reader测试版；
- 32p以内视觉测试样段；
- 旧版/新版对照表。

### Gate `MIG-G2`

主编、科学、文化、文学、角色五栏内部PASS后进入真实试读。

## 6. Phase 4 · 真实测试

### 样本建议

- 日本10—12岁儿童：最低12，建议18—24；
- 家长：最低8，建议10—15；
- 教师/司书/童书编辑：最低4，建议6—8。

### 核心指标

| 指标 | 目标 |
|---|---:|
| 3秒指出怪事 | ≥75% |
| 8秒复述给同伴 | ≥60% |
| 想立即知道答案 | ≥70% |
| 完整读毕 | ≥80% |
| 担心被误会者 | ≥60% |
| 区分事实与猜测 | ≥60% |
| 复述3—4步主因果 | ≥60% |
| 认为陸瑆笔记属于故事 | ≥60% |
| 说出笔记新增的人物信息 | ≥50% |
| 24小时后记得标题/画面 | ≥50% |
| 家长愿意或考虑购买 | ≥60% |
| 专业者判断不说教 | 多数PASS |

### Gate `MIG-G3`

- 核心硬门槛全部达成；
- 无多数儿童反馈“人物太多、知识太多、解释太长”；
- 无日本文化P0错误；
- 无科学P0错误。

- A005按回收矩阵重构；
- 五案逐案走V2 Gate；
- 重做卷级连读和复杂度审计；
- 日文母语终审前锁定五案关系网。

### Gate `MIG-G4`

- 五案独立闭合；
- 五案关系网成立；
- 学科编织成立；
- A005回收公平且可理解；
- 角色成长和陸瑆线成立；
- 决定更新旧正典与Schema。

## 8. 停止条件

遇到以下任一情况，停止扩展五案并返工A001：

- 儿童只记得知识，不记得人物；
- 家长觉得像心理教材；
- 8秒无法复述；
- 被误会者完全工具化；
- 日本学校情境不自然；
- A005必须依靠大量新解释才能成立；
- 卷网让单案阅读明显变差。
