4. 任何迁移必须保留旧版本、来源和commit。
5. 真实试读前不得批量重写A002—A005。

## 5. 建议机器规则优先级

| 级别 | 例子 | 行为 |
|---|---|---|
| P0 FAIL | A005靠未提前出现的新关键证据；真实未成年人隐私；超自然兜底 | 阻止合并 |
| P1 FAIL | 无倒计时、无错误责任、无修复动作、线索少于3 | 阻止V2 Gate |
| P2 WARN | 知识解释偏多、压力偏多、关系边过多 | 允许提交但需说明 |
| INFO | 可选字段、传播素材缺失 | 仅提示 |

## 6. 人工审核不可自动替代

程序不得自动宣称：

- 日文自然；
- 情绪有效；
- 儿童一定理解；
- 文化使用安全；
- 真实事件权利清晰；
- 公平推理完全成立；
- 陸瑆笔记具有文学价值。

这些必须由母语编辑、科学顾问、文化顾问、主编和真实读者确认。
