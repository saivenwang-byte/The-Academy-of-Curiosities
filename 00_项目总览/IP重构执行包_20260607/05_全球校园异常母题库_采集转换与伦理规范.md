# 全球校园异常母题库 · 采集、转换与伦理规范

> 状态：`MIGRATION_SPEC`

## 1. 目标

建立长期内部素材库：从全球校园真实异常、学校口述、地方怪谈、公开档案和学校生活记忆中提取“异常结构、人性压力与传播规律”，再经日本学校制度、名古屋环境和儿童口语重构为原创故事。

原则：

> **取其人性，不偷其外衣；取其异常结构，不消费其真实伤害。**

## 2. 来源分级

| 等级 | 来源 | 用法 |
|---|---|---|
| R1 | 官方记录、档案、可靠新闻、多源核实 | 可研究机制；审查后可称真实取材 |
| R2 | 多地流传的民间/校园传说 | 标注传说母题，不声称真实发生 |
| R3 | 经同意的个人口述 | 去身份化，提取情绪和关系结构 |
| R4 | 小说、影视、单一匿名帖 | 仅作类型研究，不进入真实素材宣称 |

## 3. 禁采与禁用

原则上不进入儿童商业改编：

- 可识别未成年人死亡、自杀、性侵、严重欺凌；
- 未解决犯罪、真实受害学校与家庭；
- 神圣仪式、特定族群禁忌且无当地咨询；
- 仅靠猎奇伤害成立的故事；
- 版权作品的具体情节、角色、视觉符号。

## 4. 母题卡字段

- `seed_id`
- 国家/地区
- 来源与URL/档案号
- 证据等级
- 学校类型
- 一句话怪事
- 强画面/声音
- 发生规则（时间、地点、动作）
- 儿童为何相信
- 谁容易被指向
- 真实情绪风险
- 异常结构
- 文化依赖度
- 权利/隐私/伦理风险
- 日本化可行性
- 适合单案/卷网/番外

## 5. 机制转换卡字段

- 核心机制候选
- 交叉机制候选
- 可见痕迹
- 安全复现方式
- 可公平呈现的线索
- 日本学校自然对应场景
- 名古屋季节/建筑条件
- 主压力/副压力候选
- 错误责任/真实责任候选
- 可连接的关系类型

## 6. 本土化比例

建议：`20%全球母题 + 80%日本再生`。

必须重写：

- 日本小学制度；
- 班级、值日、给食、广播、活动流程；
- 日语儿童口语；
- 名古屋天气、季节、建筑；
- 教师/家长介入边界；
- 情绪后果与修复方式。

## 7. 三道升格门

1. **人性可迁移**：核心焦虑在日本儿童情境中真实。
2. **机制可验证**：存在自然、物理、生命、信息或制度机制。
3. **校园可自然发生**：日本顾问和母语儿童判断无明显违和。

三门全过后，才可进入候选题材池；仍不等于正典。

## 8. 首轮研究计划

- 6个文化区；
- 每区10条；
- 共60张母题卡；
- AI聚类后筛12条；
- 选择3条制作300字开场与Case Card；
- 仅在儿童/家长测试通过后进入正典候选。

## 9. 隐私与儿童投稿

未来征集必须通过家长、教师、学校或图书馆；不得收集：

- 真实学校全名；
- 班级、精确位置；
- 未成年人面部；
- 可识别同学姓名；
- 实时位置；
- 未经监护同意的私人经历。

## 10. 对外表述

可用：

- “受世界各地校园口述母题启发”；
- “取材于不同学校中反复出现的生活现象”；
- “以真实可验证的校园机制重新创作”。

禁止：

- “全球真实事件改编”作为笼统宣传；
- 对R2/R3/R4素材声称真实发生；
- 隐去文化来源后直接商业包装。
