# 标题、钩子、传播与视觉符号规范

> 状态：`MIGRATION_SPEC`

## 1. 标题目标

标题必须尽量压入以下五类信息中的三类以上：

1. 强画面或声音；
2. 集体确信或传言；
3. 被指向者；
4. 机制入口；
5. 情感风险。

标题不是解释案情，而是制造一个可复述的缺口。

## 2. 标题公式

可选结构：

- 《全班都听见了他的声音》
- 《没有人写过的道歉》
- 《每个人都记得的海报》
- 《只出现在她抽屉里的失物》
- 《午休后消失的影子》
- 《全校都在等她承认》
- 《比赛开始前，所有人的号码都变了》
- 《没有人按过的广播按钮》

禁止：

- 纯知识型标题，如《空气对流的秘密》；
- 只说地点，如《音乐教室怪声》；
- 过度抽象，如《友情的考验》；
- 直接泄底，如《风吹起的海报》。

## 3. 八秒复述测试

儿童看标题和一句简介后，应能在8秒内向同伴复述：

- 发生了什么不可能的事；
- 大家怀疑谁；
- 为什么必须继续看。

不能复述则钩子FAIL。

## 4. 开篇钩子

前300—500日文字符内必须出现：

- 异常事实；
- 至少一个具体场所；
- 一个时间或任务限制；
- 一句会引发站队的话。

禁止先用一页介绍人物、天气或学校历史。

## 5. 章节钩子密度

每个小案至少设置：

- 开篇异常钩；
- 中段证据矛盾钩；
- 关系冲突钩；
- 倒计时升级钩；
- 揭晓前反证钩；
- 结尾情感余震。

钩子不是每段都反转。连续强钩之间必须有信息兑现和情绪喘息。

## 6. 一句话传播模板

```text
全校/全班都说____。
可有一条线索证明____不对。
如果在____之前查不清，____将失去____。
```

示例：

> 全班都听见广播里是他的声音，可那天他根本说不出话。午休结束前查不清，他会被撤掉广播委员。

## 7. 社交传播单元

每案至少可以拆出：

- 1条15秒怪事短内容；
- 1张强异常静态图；
- 1条事实/猜测对照；
- 1个“你现在相信谁”互动；
- 1句结尾余震。

短内容不得完整泄底；必须释放一块证据，而不是空问“你猜为什么”。

## 8. 固定品牌口令

候选口令：

- **等等，先观察。**
- **这还不能证明是他。**
- **先别站队。**

建议动作：三根手指，对应：

1. 我们真的看见了什么？
2. 哪些只是猜的？
3. 现在还缺什么？

## 9. 视觉图腾

候选核心符号：**不对劲的翘角**。

定义：一个表面正常的方形/纸张，其中一个角轻微翘起，代表“这里有一件事还没被看清”。

应用：

- Logo辅助图形；
- 书脊；
- 社团徽章；
- 线索标记；
- 证物卡；
- 视频结尾；
- 校园地图；
- 书店陈列。

视觉符号必须在单色、120px和书脊尺寸下成立。

## 10. 标题与传播Gate

| Gate | PASS标准 |
|---|---|
| T1 强画面 | 3秒能指出异常 |
| T2 可复述 | 8秒能讲给同伴 |
| T3 情绪 | 知道谁可能失去什么 |
| T4 缺口 | 有一条明确矛盾，不能直接猜完 |
| T5 不泄底 | 标题不直接给机制答案 |
| T6 品类 | 可识别为校园悬疑/推理，不误判为恐怖犯罪 |
| T7 记忆 | 24小时后能回忆标题或核心画面 |
