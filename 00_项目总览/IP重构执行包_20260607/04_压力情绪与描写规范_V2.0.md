## 1. 双倒计时（卷/案通用）

### 1.1 定义

| 钟 | 叙事功能 | 描写要求 |
|----|----------|----------|
| **事件钟** | 自然/流程节点 | 具名时间（「还有二十分钟」）· 可感知变化（雨、风、张贴） |
| **社交钟** | 名誉/制度后果 | 可验证制度（委员、壁报、班长钥匙）· **禁** 网络暴力写实 |

两钟须在 **Step④** 同时点亮；Step⑨ 后允许 **余波**（道歉/沉默/空栏），不许 **反转吓人**。

### 1.2 Vol1 各案压力锚

| 案 | 事件钟 | 社交钟（pressure_primary） |
|:--:|--------|---------------------------|
| A001 | 报名截止前 20 分 | 撤水野委员 + 「破坏活动」舆论 |
| A002 | 早自习前泥印消失 | 空栏投稿引发「鞋柜搞事」 |
| A003 | 壁报张贴前 | 「社员偷懒/隐瞒」内部压力 |
| A004 | 早会前灰圈再现 | 「半夜写字/闹鬼」班级议论 |
| A005 | 活动展前 | 全校「系列破坏」+ 褪色名单 |

## 2. 情绪弧线（单案）

```
好奇 → 误判紧张 → 观察节奏（幽默喘息）→ 误指峰值 → 机制释然 → 温柔澄清 → 余波/空栏
```

| 阶段 | 情绪 | 幽默 relief 点 |
|------|------|----------------|
| 误判 | 轻紧张 | 志郎「查监控/犯人」被慧美拦 |
| 验证 | 专注 | 志郎换贴习惯自嘲 |
| 误指峰值 | 同情（非虐） | 光一句冷笑话破局 |
| 澄清 | 释然 | 校工/大人 **只划边界** 不代破 |

**禁**: 羞辱性公开审判 · 长期社交孤立 · 成人道德说教长篇。

## 3. wrong / true 责任的情绪写法

| 层 | 写法 | 示例（A001） |
|----|------|--------------|
| **wrong** | 群体 **误读** 行为动机 | 「水野故意撕海报搅局」 |
| **true** | 机制 + 怯懦/善意 | 风侧翘边+他换贴旧通知；想参加活动却不敢说 |
| **收束** | 观察社 **不写审判词** | 壁报写机制；空栏仍等核实 |

**瑆第二真相**（Step⑪）: 补 **情感**（「他把海报换回去」）· 不补 **新机制**。

## 4. 描写规范（继承+追加）

### 4.1 继承 V1 Hybrid Voice

- 短句留白 · 陸珣限知 · 删说明文 · 场级变奏  
- 详见 `academy-voice-editor` · `Vol1_文学MVP最终成果_20260614.md`

### 4.2 V2 追加

| 规则 | 说明 |
|------|------|
| **压力可见** | 倒计时须 **角色可感知**（看钟/听广播/看委员表） |
| **舆论一句** | 每案 ≥1 句 **班级版传言**（可错） |
| **幽默不稀释** | relief 段 ≤ 全场 15% · 不插科打诨毁 fair-play |
| **瑆笔记密度** | 卷级 1 篇总笔记；单案 **第二真相** ≤ 80 字/案 |
| **水野弧线** | 出场 ≤3 案主帧 · 禁 villain 特写 |

|----|-----|

## 6. 场级自检（写作后）

- [ ] 双倒计时是否在正文 **前两场** 出现  
- [ ] 误指峰值是否在机制闭合 **之后**（或交叉但机制仍 fair）  
- [ ] 是否有 ≥1 处幽默 relief  
- [ ] 瑆第二真相是否 **零机制信息**  
- [ ] 舆论句是否可被后文 **反驳**
