## 3. 十五秒传播单元

每案拆 **1 单元** 供试读/运营 · 结构：

```
[0–3s] 异常画面（喇叭响/黑板字/空海报位/抽屉/无影合照）
[3–8s] 误指一句（「又是他干的？」）
[8–12s] 观察社反动作（「先观察。」）
[12–15s] 机制半句（「文件是旧的。」/「膜边在反光。」）· 不剧透全解
```

| 案 | 15s 文案 seed |
|:--:|---------------|
| A001 | 广播里是他的声音——可他就站在全班面前。 |
| A002 | 黑板上每天一句「对不起」，没人写过。 |
| A003 | 全班都记得那张海报，学校却找不到。 |
| A004 | 失物全在她抽屉里，她却没有钥匙。 |
| A005 | 合照里五个人，脚下都没有影子。 |
