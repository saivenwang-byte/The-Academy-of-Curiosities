## 1. 标题层级

| 层 | Vol1 定案 | 规则 |
|----|-----------|------|
| **系列** | 学堂奇事録 / 学堂趣事录 | 品牌 LOCK |
| **卷** | 觉得奇怪，就先观察 | 不变 |
| **单元** | 五件怪事都指向同一个人 | V2 卷级叙事题 |
| **单元副** | 全校都在等他承认 | 信息污染钩 · 禁剧透真凶 |
| **单案** | 保留 V1 题名（可副标题加注 V2 钩） | 例：翘边的海报 |

---

## 2. 单案标题规则

| 规则 | 说明 |
|------|------|
| **物件+异常** | 翘边的海报 · 错位的泥印 · … |
| **禁吓人词** | 鬼/犯人/恐怖/阴谋 |
| **可加副题** | ≤12 字 · 点压力（「报名截止前」） |
| **A005 V2** | 主题名建议：**褪色的名单**（机制钩）· 副题保留橡皮屑方向 **仅 V1 对照** |

---

## 3. 十五秒传播单元

每案拆 **1 单元** 供试读/运营 · 结构：

```
[0–3s] 异常画面（海报翘边/泥印/空栏）
[3–8s] 误指一句（「又是他干的？」）
[8–12s] 观察社反动作（「先观察。」）
[12–15s] 机制半句（「风侧先开口。」）· 不剧透全解
```

| 案 | 15s 文案 seed |
|:--:|---------------|
| A001 | 报名截止前，海报翘了；全校等他承认——等等，先观察。 |
| A002 | 没人用的鞋柜格，却有泥印。 |
| A003 | 壁报有一格，永远空着。 |
| A004 | 讲台灰圈，每天早会前出现。 |
| A005 | 名单上的名字，遇热就淡。 |
