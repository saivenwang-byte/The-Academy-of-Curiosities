# Vol1 · 全卷出版验证总作战图

> **目标**：完成《学堂奇事録》Vol1 的 **全卷日文母语终审、商业视觉、印前样书、出版社验证**。
> 
> **项目基线**：当前仓库仍属于 **Gate A / 序+A001 样张 MVP**；本文件定义 Gate A 之后到“可送出版社验证”的完整生产路径。
> 
> **原则**：不把“样张可读”误报为“全卷出版就绪”；不把 AI 审校、屏幕 PDF、读者试看或出版社联系人意见，单独当作最终出版 PASS。

---

## 0. 最终出口定义

只有同时满足以下四个出口，Vol1 才可标记为：

`VOL1_PUBLISHER_VALIDATION_READY`

| 出口码 | 出口 | 核心判断 |
|---|---|---|
| **J-FINAL** | 全卷日文母语终审 PASS | 32,000–38,000 日文字符全卷由合格日文母语编辑完成三轮审校，修改记录闭环 |
| **V-COMM** | 商业视觉 PASS | 封面、书脊、人物、内页、系列识别在书店缩略图、货架与目标读者测试中成立 |
| **P-PROOF** | 印前样书 PASS | 32p 先导实体样 + 144–160p 全卷数码样书完成印前、装订、色彩、可读性检查 |
| **PUB-VAL** | 出版社验证完成 | 至少获得目标出版社/资深童书编辑的结构化反馈，并形成 Go / Revise / Reposition 结论 |

> **禁止状态误报**：
> - `E06 PASS` ≠ 全卷出版 PASS
> - `Reader PDF` ≠ 印前文件
> - `母语者读过` ≠ 母语终审
> - `出版社看过` ≠ 出版社验证完成

# 工作流 A · 全卷日文母语终审（J-FINAL）

## A0. 先统一五案生产等级

每案必须具备：

- 日文正文唯一真源
- Case Card / 科学原理与公平线索表
- 角色声音检查
- 名古屋学校文化检查
- 读音索引与表记表
- 插图点位与不可改动证据点
- 版本号、字数、签核状态

A002–A005 必须补齐与 A001 同级的生产包，禁止只把 prose-only 文本直接送终审。

## A1. 母语终审必须拆成三轮

### 第一轮：日文母语责编审（構成・文体編集）

**人员要求**：日本儿童文学、児童書或教育出版经验；不能只由翻译人员承担。

**检查范围**：

- 整体故事节奏、案间重复、角色弧线
- 10–12 岁自然口语和叙述距离
- “翻译腔”、中文逻辑残留、解释过度
- 笑点、悬念、误推、揭晓是否自然
- 每个角色的语尾、词汇、反应模式是否可辨
- 科学说明是否像故事而非教辅

**交付物**：

- `Vol1_JP_母语责编修订稿_Vx.x`
- `Vol1_JP_责编修改说明`
- `角色声音偏差表`
- `逐案结构裁决表`

### 第二轮：年龄与文化审（児童読者・学校文化）

**人员组合**：

- 日本小学高年级语文/阅读相关教师 1 名
- 日本 10–12 岁儿童试读 6–10 名
- 儿童书编辑或学校图书馆司书 1 名

**检查范围**：

- 独立阅读是否卡顿
- 学校制度、社团称呼、班级行动、师生边界是否自然
- 名古屋地域表达是否必要且不过量
- ルビ、汉字、科学词难度
- 是否像“外国人写的日本学校”

**交付物**：

- `J-Age_Readability_Report`
- `J-Culture_Authenticity_Report`
- 儿童逐页停顿/跳读记录
- P0/P1/P2 修改清单

### 第三轮：出版校对审（校正・校閲）

**人员要求**：与第一轮责编不同的人；执行独立校对。

**检查范围**：

- 错字、脱字、标点、数字、单位
- 人名、读音、班级、时间线、道具连续性
- 全卷首次标音/案首复现规则
- 图文一致、页码、目录、标题、奥付
- 禁则、行首行末、ルビ溢出与断行

**交付物**：

- 初校 PDF + 赤字
- 再校 PDF + 赤字闭环
- 念校/终校签字页
- `Vol1_JP_表记统一表_FINAL`
- `Vol1_读音索引_FINAL`

## A2. J-FINAL 验收阈值

- P0 语言、文化、科学错误：**0**
- 角色姓名、读音、年级、班级错误：**0**
- 未闭环赤字：**0**
- 10–12 岁试读者完整读毕率：建议 **≥80%**
- “像翻译作品/不自然日语”强烈反馈：**0**
- 至少 2 名相互独立的日文母语专业人员签字

# 工作流 B · 商业视觉（V-COMM）

## B0. 商业视觉不是“把插图画完”

必须解决三个商业问题：

1. 在日本儿童书货架上，3 秒内是否能看懂“校园+趣事+推理”。
2. 角色是否具有系列长期辨识度，而不是普通日系小学生群像。
3. Vol1 与未来 Vol2–20 是否能形成稳定书脊、卷色、徽章和封面语法。

## B1. 先建立《商业视觉 Bible》

至少包含：

- 品牌关键词与禁止词
- 目标读者和购买者双画像
- 系列 Logo 日文正字与安全区
- 四级书名层级
- 五名 Vol1 核心角色正侧背、表情、身高比、轮廓差异
- 脸型、眼睛、眉形、鼻口、发型、服装、配色差异矩阵
- 观察社徽章、笔记、工具、校园空间的视觉语法
- 插图线条、阴影、背景密度、表演字规范
- 书脊系统、卷号系统、卷色系统
- 封面/封底/勒口/腰封/电商缩略图规范
- 黑白或双色印刷时的降级规则

## B2. 封面必须做三方向商业测试

建议三稿不是“同构图换颜色”，而是三种营销命题：

| 方向 | 核心卖点 | 应验证的问题 |
|---|---|---|
| **A · 异常事件型** | 翘边海报/风/线索成为第一视觉 | 谜题感是否强 |
| **B · 角色行动型** | 四人观察社正在调查 | 角色亲和力是否强 |
| **C · 系列徽章型** | 社团徽章+校园地图+事件符号 | 系列品牌感是否强 |

测试场景：

- 手机电商 120–180px 缩略图
- 实体书架 1.5–2m 观看
- 只露书脊
- 与 8–12 本真实同类书并排
- 儿童选书、家长选书、编辑选书三组分开记录

## B3. 内页商业样张

先完成 **32p 实体先导样**，应覆盖：

- 封面/书脊/封底
- 人物介绍
- 案首半页标题条
- 正文纯文字页
- 正文+半页图
- 线索页
- 实验/验证页
- 陸瑆ノート
- 跨页或强视觉页
- 次卷预告与系列页

## B4. V-COMM 验收

- 目标读者在无说明条件下能说出“校园、推理/谜、好玩”中的至少两项
- 主要角色黑色剪影或灰阶头像仍可辨
- 三稿测试有明确优胜方向，不以 IP Owner 个人偏好替代数据
- 书脊与电商缩略图均能识别系列名与卷号
- 视觉 Bible 可让第二位设计师继续生产，不依赖原画师口头解释

# 工作流 C · 印前样书（P-PROOF）

## C1. 两级样书，不直接跳全卷

### P1 · 32p 实体先导样

目的：验证字号、ルビ、留白、图片密度、纸张、装订、封面识别。

必须采用真实预期尺寸、装订方向与接近量产的纸张；不能只打印 A4 办公纸。

### P2 · 144–160p 全卷数码样书

目的：验证整卷厚度、书脊、翻阅节奏、跨页、色差、疲劳度与装订安全。

## C2. 印前文件要求

最终母版：**InDesign 或出版社/印厂接受的等价专业排版文件**。

文件包至少包括：

- 可编辑排版源文件
- 印刷 PDF
- 屏幕审阅 PDF
- 字体与授权清单（不随意外传字体文件）
- 链接图像清单
- 图片分辨率检查报告
- CMYK/专色/黑版规则
- 出血、裁切、版心、装订侧安全区
- 预检报告
- 页码/目录/奥付/ISBN 占位状态
- 封面展开尺寸计算表

> 出血、色彩配置、墨量、书脊厚度等具体数值，必须由最终印厂模板确认；仓库只保存“印厂确认值”，不凭经验写死。

## C3. 实体样验收动作

- 日光、室内暖光、冷光三环境看色
- 儿童自然握持、翻页、平摊测试
- 连续阅读 20–30 分钟疲劳观察
- ルビ清晰度和小字号糊版检查
- 跨页关键信息是否吃进装订沟
- 深色块透印、蹭脏、掉粉、色偏检查
- 封面耐磨与书脊开裂检查
- 实测重量、厚度、开合与书架站立

## C4. P-PROOF 验收

- 印前预检 P0 错误：0
- 页面缺图、低分辨率、字体替换：0
- 装订沟吞噬关键线索：0
- 儿童试读中字号/ルビ造成的阅读阻断：0
- 初样问题全部形成赤字清单，再样闭环后签字

# 工作流 D · 出版社验证（PUB-VAL）

## D0. 验证目的

出版社验证不是立即“求签约”，而是验证四件事：

1. 品类定位是否被日本编辑理解。
2. 年龄带、篇幅、五案结构是否适合其产品线。
3. 视觉与制造成本是否符合商业模型。
4. 作者/IP 方是否具备持续交付 20 卷系列的能力。

## D1. 目标对象分三层

| 层级 | 对象 | 目标 |
|---|---|---|
| **S** | 具有儿童章节书、校园推理、学习娱乐产品线的出版社 | 获取正式编辑判断或立项讨论 |
| **A** | 童书编辑、版权代理、出版顾问、书店儿童书采购 | 验证市场位置、包装和渠道 |
| **B** | 教师、司书、家长、儿童阅读推广人 | 验证使用场景、推荐理由和阻力 |

## D2. 出版提案包

### 10 页以内核心提案 PDF

1. 一句话定位
2. 日本 10–12 岁读者问题
3. Vol1 卖点与五案结构
4. A001 样章钩子
5. 角色与社团关系
6. 科学验证和零恐怖差异化
7. 商业视觉与书架缩略图
8. 20 卷持续性与案例库
9. 读者测试数据
10. 制作状态、团队、希望合作方式

### 附件

- 日文母语终审后的完整稿或出版社要求篇幅
- 32p 实体/电子样章
- 全卷梗概与逐案 synopsis
- 角色 Bible 摘要
- 商业视觉 Bible 摘要
- E20/E21 读者验证报告
- 科学与文化审核声明
- 权利声明、AI 使用披露、素材来源表

## D3. 统一反馈问题

每位出版社/编辑都回答同一组核心问题：

1. 你会把本书归入哪个书架/产品线？
2. 目标年龄是否准确？
3. 最大卖点是什么？最大阻力是什么？
4. 五案合卷是否成立？篇幅是否合适？
5. 封面是否能驱动拿起/点击？
6. 与现有同类相比，差异化是否足够？
7. 哪一项会阻止进入选题会？
8. 需要先改稿、改视觉、改定位还是改成本？
9. 是否愿意看完整稿/参加第二次会议？

## D4. 出版社验证样本与裁决

最低建议：

- S 层有效编辑反馈：3 份
- A 层专业市场反馈：5 份
- B 层教师/司书/推广反馈：8 份

裁决：

| 结果 | 条件 | 动作 |
|---|---|---|
| **GO** | 至少 2 家 S 层愿意看全稿或进入二次讨论；无共识性致命定位问题 | 进入定向投稿/商务谈判 |
| **REVISE** | 多方认可核心，但同一问题被 3 位以上专业人士独立指出 | 一轮定向整改后复验 |
| **REPOSITION** | 出版社无法归类、年龄/篇幅/视觉出现系统性冲突 | 重做产品定位，不继续堆制作成本 |
| **HOLD** | 反馈样本不足或对象不匹配 | 继续验证，不得宣称出版社认可 |

## 3. 读者验证升级

现有 E20 的 8–12 份反馈适合作为 A001 Gate A 验证；全卷商业验证应升级为：

| 组别 | 建议样本 | 核心指标 |
|---|---:|---|
| 日本 10–12 岁儿童 | 24–36 | 读毕、续读、角色偏好、谜题理解、跳读点 |
| 家长 | 12–18 | 购买意愿、安全、价格敏感、教育感过强风险 |
| 教师/司书 | 6–10 | 年龄适配、课堂/图书馆推荐、文化自然度 |
| 编辑/出版专业 | 5–8 | 品类、篇幅、包装、成本、系列化 |

分两轮：

- **R1**：32p 样章与三封面测试
- **R2**：全卷样书与价格/书架测试

不得混合计算儿童、家长、编辑评分。

## 4. 组织与权责

| 角色 | 决策权 | 不可替代职责 |
|---|---|---|
| IP Owner | 最终 Go/Hold | 正典、预算、合作方式、最终签核 |
| 日文母语责编 | J-FINAL 内容裁决 | 文体、结构、年龄语言自然度 |
| 独立校对者 | 校正签字 | 错字、表记、连续性、版面文字 |
| 科学顾问 | 科学 P0 否决 | 原理、实验、安全、可复现性 |
| 日本文化/学校顾问 | 文化 P0 否决 | 学校制度、地域与行为自然度 |
| 艺术总监 | V-COMM 视觉裁决 | 系列视觉 Bible、封面与角色统一 |
| 书籍设计师 | 印前执行 | InDesign、网格、图文、输出与预检 |
| 印厂/制作担当 | 生产参数确认 | 纸张、装订、色彩、模板、样书 |
| 出版顾问/代理 | PUB-VAL 组织 | 目标社、提案、会议和反馈归档 |

**AI/Agent 的边界**：可做 lint、比对、清单、模拟审稿、版本治理；不得替代母语责编、校对、科学签字、印厂确认或出版社反馈。
