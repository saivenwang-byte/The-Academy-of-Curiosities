## 1. 为何调整（相对 V1.1 CN 参考列）

V1.1 §3「CN 创作参考」为 E04 日译前的 **粗算对照** · 非 PRODUCT 硬门禁。  
V1 文学重写案⑤已达 **~7,200 汉字** 且阅读节奏合格 · 证明 **7k–8k/案** 更符合：

- 日本小五小六 **15–20 分/日** 桥梁推理竞品（放课后系 · 科学ミステリー短卷）  
- 五案 + 瑆 + 卷终汇流 **信息密度** · 非注水扩写  
- 中文=创作稿 · 日译阶段 **删增至 JP 预算** · CN 不必削到 2k–3k

**废止执行口径**（非废止 V1.1 文件本身）：

- ❌ 「未达 CN 参考下限 = G-CN 不过」  
- ❌ 为凑字数重复解释/总结  
- ✅ 「规范全绿 + 读者群/专家体感 OK = G-CN 可过」

## 4. 读者群 + 专家组评估流程（新增）

### 4.1 读者群（试读）

| 项 | 建议 |
|----|------|
| 样本 | 目标读者 **10–12 岁** · 每案 **3–5 人** · 可加家长伴读组 |
| 任务 | 读 **1 案/次** · 15–20 分钟 · 复述「怪事是什么」「谁被误会」 |
| 采集 | 哪里想跳过 · 哪里没看懂 · 哪里最喜欢 |
| 输出 | `薄样张_试读/读者群反馈_A00X_YYYYMMDD.md` |

### 4.2 专家组复评（与读者群并行）

| 维度 | 权重 | 说明 |
|------|:----:|------|
| LOCK/机制 | 25% | 09 + 10 字段表 |
| 文学/Voice | 20% | 短句 · 瑆笔记 |
| 科学公平 | 15% | P06 + 客座签核 |
| 卷结构/钩子 | 15% | 关系网 · 汇流 |
| **读者群体感** | **15%** | 复述率 · 跳读点 |
| 插画就绪 | 10% | SC 链 · 空间资产复用 |

**综合 ≥9.5 / 单项 ≥9.8**：在 **规范全绿** 前提下 · **读者群无 P0 跳读** · 专家签字。

### 4.3 与 MVP 顺序

```
G-CN（规范+体感）→ 读者群小样本（可选并行）
    → G-SHOT-T → G-JP → 修订分镜字 → G-IMG → G-MVP
```
