## 1. 单位命名（禁止混用）

| 术语 | 定义 | 英文/编号 | Vol1 例 |
|------|------|-----------|---------|
| **系列** | IP 总品牌 | Academy / 学堂奇事録 | 学堂趣事录 |
| **卷** | **1 本可发行的单行本** | Vol N / 第 N 巻 | **Vol1 · 第 1 卷** |
| **册** | **= 卷**（同指一本书） | — | 第 1 册 = 第 1 卷 |
| **案 / 篇 / 话** | **1 个完整校园趣事谜题** | A001… | **A001** |
| **单元** | **出厂验收单位** | PRODUCT-UNIT | = 1 案的全套交付 |
| **章 / § / 节拍** | 案内叙事段（发现→误导→线索→收束） | — | **不是**独立故事 |

```
系列
 └── 卷（Vol1 … Vol20）          ← 1 卷 = 1 册 = 1 本书
      ├── 序（1 篇）
      ├── 案 ×5（A001–A005 …）
      └── 卷末陸瑆总笔记 + 附录
```

**素材库**: `C001` 湿椅子 = **C 线番外素材** · **不进** Vol1 目录。

## 2. 篇幅与物理规格（专家组 + 出厂标准）

### 2.1 整卷（1 本书）

| 项 | LOCKED |
|----|--------|
| 形态 | 竖版 **A5** 桥梁书 PDF/纸书 · 单页纵向翻页 |
| 页数 | **96–128p**（目标中位 **~112p**） |
| 结构 | 外封 · 版权 · 目录 · 序 · **5 案** · 卷末瑆笔记 · 实验汇总（可选）· 封底 |
| 首印语言 | **日本語**（`学堂奇事録`） |
| 图/文 | 图 **~30%** · 文 **~70%** |

### 2.2 单案（1 单元 / A00x）

| 项 | LOCKED |
|----|--------|
| 阅读 | **12–18 min** / 案（合订本一口气读完） |
| 册内占位 | **~18–22p** / 案（含前后置分摊） |
| 正文 | 中文定稿 **~2500–4500 字**（B2/E02 试读级）→ E04 日译 |
| 插图 | **A 场景 4–6 + B 总结/笔记/原理 1–2** ≈ **6–8 张/案** |
| 尾钩 | **L1 轻钩** · 本案科学/人际 **必须闭合**（`00_趣事尾钩规范`） |
| 瑆 | **全卷 1 组总笔记**；单案样章可暂 1 页（E02 年龄适配） |

**密度参照**: `C001` 湿椅子 ≈ 48p/单案 · 12 图 = **形态上限样张** · Vol1 为 **5 案合订**，不按 48p×5 排页。

> **PRODUCT 字数** → 见 **[`篇幅与单位构架_V1.1.md`](./篇幅与单位构架_V1.1.md)** · 废止 subagent 临时 8000–10000/案 口径。

### 2.3 连载节奏（产品层 · E03）

| 层 | 节奏 |
|----|------|
| 日 | ~20 min · **1 阅读片段**（推进当周案） |
| 周 | 5 天 · **完成 1 案** |
| 月 | **1 卷**（5 案） |
