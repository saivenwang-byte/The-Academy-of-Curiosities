## 0. 日文书写体系（强制 · IP 锁）

| 合法类别 | 说明 |
|----------|------|
| **平假名（ひらがな）** | 和语·语法·口语自然段 |
| **片假名（カタカナ）** | 外来语·拟声强调·专名音译 |
| **汉字（漢字）** | 配当内叙事·人名地名·必要 disambiguation |

### 废止术语（全员 · 文档/口播/外包 brief）

| 废止 | 原因 |
|------|------|
| **「振假名」** | 非日文书写类别 |
| **「全振假名」** | 同上 · 且易与「全书注音」混写为错误概念 |
| 任何暗示「第四种假名体系」的说法 | 日语只有上三类 |

### 正确术语

| 场景 | 用法 |
|------|------|
| 中文文档 | **读音标注** · **注音** · **ルビ** · **振り仮名（ふりがな）** |
| 工作稿 | 括号 `漢字（ひらがな）` |
| 出版排版 | **ルビ**（汉字上方标读音） |

**一句话**：控制 **哪些漢字出现、哪些漢字要标读音** — 不是发明新书写体系。

## 1. 对齐《标准日本语》（标日）规范

> **标日** = 国内通行的标准日语教材体系（初/中级）· 本 IP 日文的 **书写密度与汉字引入节奏** 对齐其教学逻辑，**不是**把正文改成课文。

| 标日原则 | 本 IP 执行 |
|----------|------------|
| 汉字 **按级引入** · 未学字需读音 | **小学五·六年级配当汉字** = 默读基线 · 表外字 **必须** ルビ 或改写法 |
| 语法·接续 **以假名自然呈现** | 禁止为「文学感」把可假名表达 **强行汉字化** 再注音 |
| 课文 **汉字密度低于** 报纸小说 | Vol1 正文 **汉字率低于** 一般児童文学 · E04 终检 |
| 外来语 **片假名** | クラブ · ポスター · ノート 等保持カタカナ |
| 注音 **服务未学字** · 非装饰 | **选择性ルビ** · 非竞品式全书每字注音 |

### 1.1 尽量减少汉字使用频率（强制）

**优先顺序**（E04 日译 / 润色时逐句检查）：

1. **能ひらがな/カタカナ清晰表达** → 不强行用汉字（尤其副词、接续、口语）  
2. **必须用汉字** → 人名 · 地名 · 制度专名 · 科学术语 · 同音区分  
3. **表外/难读汉字** → **ルビ** 或 **改写为假名**（标日式「换一种写法」）  
4. **禁止** 为提升「大人感」堆叠 **文语汉字** · 禁止 **汉诗式** 密度  

| 指标 | 目标（heuristic） |
|------|-------------------|
| 正文汉字占全字符 | **低于** 同篇幅一般児童文学 · E04 案级抽检 |
| 相对放课后竞品 | **更低汉字率 + 更低注音率**（双低 · 不同方向降门槛） |
| 工作稿带注音汉字 | 约 **15–25%**（见 §4） |

**示例**：

| 不推荐 | 推荐 |
|--------|------|
| 彼は確認した。 | 彼は **かくにん** した。／彼は確認（かくにん）した。← 案内首次 |
| 異常な現象 | **いじょう** な **げんしょう** ← 或首次ルビ后省略 |
| 全段汉字名词化 | 保留对话 **假名口语** + 关键名词汉字 |

## 3. 总原则

**标日规范 + 年级适配 + 首次标音 + 减汉字频率**

- **年级适配**：五·六年级配当 = 默读基线  
- **首次标音**：全卷首次 · **案首复现**  
- **减汉字**：§1.1 优先顺序  
- **出版**：工作稿括号 → 印前 **ルビ**

---

## 4. 三级规则（E04 执行表）

### A · 必须表记 + 必须标音（首次 + 案首复现）

人名 · 地名/设施 · 科学/机制术语 · 同音 disambiguation（陸瑆/伊藤光）· 表外难读汉字

### B · 汉字可用 + 首次标音后省略

五·六年级配当内叙事词 · 案内重复词 · 社团专名

### C · 优先假名 / 不标音

四年级前高频词（学校·先生等）· 已标同案内 · **假名本体** · **口语自然段**

## 5. 密度与质检

| 禁止 | 说明 |
|------|------|
| **全书每字注音** | 与 IP 年龄带冲突 |
| **「全振假名」式表述** | 术语违规 |
| 为降龄 **盲目加注音** | 应用 **减汉字+选择性ルビ** |
| 科学机制词 **零注音** | 推理可读性 |

| 终检 | 标准 |
|------|------|
| E04 | 小五生 **独立读不卡** · 汉字率/注音率双检 |
| E07 | 人名/制度/名古屋词 **0 误读** · 标日式改写 **0 违和** |
