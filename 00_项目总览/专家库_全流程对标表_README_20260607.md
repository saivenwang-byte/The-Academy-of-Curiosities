## 拟虚拟专家名 列 → 已废止

2026-06-07 IP Owner 批准 **ACE 路线** 后，CSV/XLSX 中 **「拟虚拟专家名」** 列（如佐藤健一、吉田文子、渡辺理等）**不再代表待建 persona Agent**。

| 旧列含义 | 新用法 |
|----------|--------|
| 拟虚拟专家名 | **历史占位** — 能力已映射至 ACE 能力原子 |
| 蒸馏 Agent 名 | 改用 **ACE Officer** 名，如 `ACE-B_Sample_Visual_Production_Officer` |
| 缺口 / 拟增补方式 | 查 [`ACE全球IP专家蒸馏计划_执行路线图_V0.1_20260607.md`](./ACE全球IP专家蒸馏计划_执行路线图_V0.1_20260607.md) Phase 2 原子库 |

**允许**：在蒸馏卡 `knowledge_sources[]` 列出公开人物/机构作 **知识出处**（非 persona）。
