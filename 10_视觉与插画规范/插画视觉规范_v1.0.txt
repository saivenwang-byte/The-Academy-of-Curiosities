========================================================================
《学堂趣事录》视觉与插画规范 v1.0
The Academy of Curiosities — Visual & Illustration Specifications
========================================================================

适用范围：绘本内页插画（每卷 48-56 页，约每 2 页 1 幅插画）
目标读者：7-12 岁中日双语儿童
插画功能：讲故事、传递情绪、隐藏线索


========================================================================
一、核心风格定位
========================================================================

【一句话风格】
日本写实校园风格 × 儿童绘本温暖色调 × 本格推理的观察细节

【不属于本 IP 的风格】❌
  ❌ 欧美卡通风（大眼睛、夸张肢体、迪士尼式变形）
  ❌ 国漫古风/水墨风
  ❌ 日系萌系/Q版二次元（大头小身体）
  ❌ 极简扁平矢量风
  ❌ 暗黑哥特风

【属于本 IP 的风格】✅
  ✅ 写实比例的人物（头身比约 1:5，符合小学中年级体型）
  ✅ 温暖明亮的色调（见下文色板）
  ✅ 有质感的线稿 + 水彩/彩铅式上色
  ✅ 日本校园的真实空间感（推拉门、鞋柜、给食、扫除）
  ✅ 表情克制但精准——日式含蓄
  ✅ 画面中藏有可"二次阅读"的线索细节


========================================================================
二、参考插画师风格对标（Influence）
========================================================================

以下插画师风格可作为方向参考，不应直接复制：

1. 杉田比呂美（Sugita Hiromi）
   - 特点：日常场景、温暖色调、人物比例真实
   - 参考：校园生活绘本、城市街景
   - 适合：日常校园场景、上下学路线

2. 黒井健（Kuroi Ken）
   - 特点：柔和彩铅风、情感氛围强
   - 参考：《手ぶくろを買いに》
   - 适合：结尾场景、陸瑆笔记氛围页

3. いわさきちひろ（Iwasaki Chihiro）*
   - 特点：水彩晕染、儿童神态精髓
   - 参考：大量儿童绘本
   - 适合：情感场景参考（但本 IP 需要更硬朗的线稿）
   *注意：ちひろ风格过于柔和，仅作情感氛围参考

4. 日本の児童文学挿絵傳統
   - 特点：线稿 + 淡彩 / 彩铅
   - 参考：福音館書店、童心社出版的儿童书
   - 适合：整体方向锚定

5. 現代绘本参考（海外）
   - Jon Klassen：简约但有叙事张力
   - Carson Ellis：场景氛围构建
   - Isabelle Arsenault：线稿+有限色板
   注意：仅参考构图和用色逻辑，不可照搬欧美风格

【风格总结】
  线稿：黑色或深褐色细线，有手绘感，不一味追求"干净"
  上色：透明水彩或彩铅质感，避免数码感过强的渐变色
  光影：日本校园的自然光（片廊下走廊的斜阳、阴天的柔和散射光）
  留白：每页至少保留 20% 留白区域，不填满

========================================================================
四、角色视觉统一规范
========================================================================

【角色比例标准】
  五年级（伊藤光、加藤慧美、松本志郎、山本理紗）：
    头身比约 1:5，身高约 135-145cm
  四年级（陸珣）：
    头身比约 1:4.8，身高约 130-135cm
  三年级（陸瑆）：
    头身比约 1:4.5，身高约 125-130cm
  成年人（山崎老师、父母）：
    头身比约 1:6.5-7

【角色辨识度——每个角色至少 2 个不可变特征】

伊藤光：
  ✓ 圆脸 + 浅褐色笑眼（眯成月牙）
  ✓ 亮色围巾（橙/黄，秋冬季）
  ✓ 手势丰富（说话时比划）
  ✗ 不可画成瘦脸或尖下巴

加藤慧美：
  ✓ 银框圆眼镜（不可换款式）
  ✓ 低马尾，黑色直发，刘海偏分+银色发夹
  ✓ 食指推眼镜的动作
  ✗ 不可画成披肩散发

松本志郎：
  ✓ 根根竖起的黑短发（像静电打过）
  ✓ 圆脸但有咬肌，小麦色皮肤
  ✓ 书包侧袋露出螺丝刀/金属零件
  ✗ 不可画成瘦削体型

山本理紗：
  ✓ 茶褐色丹凤眼（眼尾上挑）
  ✓ 莫兰迪色系衣物（灰粉/雾霾蓝/燕麦色）
  ✓ 随身布面速写本
  ✗ 不可画成大眼萌系

陸珣：
  ✓ 深褐色单眼皮，眼神"钝感"（日常）
  ✓ 素色衣物（灰/深蓝/墨绿，无Logo无图案）
  ✓ 蹲下观察昆虫的姿态
  ✓ 深蓝尼龙双肩包 + 放大镜挂坠
  ✗ 不可画成帅气型或表情丰富

陸瑆：
  ✓ 短发齐肩，刘海整齐
  ✓ 牛皮纸封面线圈笔记本（几乎从不离手）
  ✓ 比同龄人矮一头
  ✗ 不可画成甜美卖萌型

葛西泉蔵：
  ✓ 花白头发向后梳 + 深麦色皮肤
  ✓ 作务服（深蓝/深灰）+ 木屐
  ✓ 老花镜用细绳挂在脖子上
  ✗ 不可画成"圣诞老人式"慈祥夸张

【人物一致性铁则】
  - 所有插画师开工前必须拿到以上角色"不可变特征"清单
  - 每个角色制作标准三视图（正面/侧面/背面）+ 5 个常用表情
  - 同一卷内，同一角色的身高、脸型、发型、眼镜款式不允许任何偏差
  - 跨卷角色成长（如陸珣六年级时身高增长）需提前规划


========================================================================
五、服饰规范
========================================================================

【日常服装】★ 重申：公立小学不强制校服，穿私服 ★
  参考各角色四季穿搭档案（01_角色设定/ 目录下的角色文件）
  关键原则：
    - 每个角色有固定的色彩偏好（伊藤：亮色 / 加藤：学院深色 / 松本：运动色 / 陸珣：素色 / 山本：莫兰迪）
    - 同角色不同场景可换衣服，但色系要保持一致
    - 衣物有被使用过的痕迹（磨损、褪色），不是全新的

【必须统一的少数物品】
  ✓ 室内鞋（上履き）：全角色统一白色布面+橡胶底
  ✓ 体育服（体操着）：全角色统一白色短袖+藏蓝短裤
  ✓ 红白帽（紅白帽子）：运动会统一佩戴
  ✓ 给食白衣：给食当番时统一穿戴（仅当番角色）
  ✓ ランドセル：仅入学式场景和上學场景出现（日常上课放教室ロッカー）

【书包】ランドセル是家长的自主选择，不是校规
  - 传统：男生黑/女生红；现代：多样色
  - 一年级崭新光亮 → 六年级磨损褪色
  - 女生常挂钥匙扣/挂饰；男生侧袋藏玩具
  - 陸珣用深蓝尼龙双肩包（不是ランドセル）——这是他转学生的特征之一

【鞋】
  - 室外鞋：自己的运动鞋（各角色不同品牌/颜色均可）
  - 室内鞋：统一的白色バレーシューズ型
  - 厕所拖鞋：场景需要时使用（トイレのスリッパ）


========================================================================
六、主要场景视觉模板
========================================================================

【教室（四年二組）】
  - 推拉门（引き戸）在东侧——门口视角
  - 靠窗座位在西侧——早上背光
  - 黑板在北侧
  - 后墙ロッカー在南侧
  - 地板：淡绿色ビニール床タイル
  - 椅子：钢管腿 + 木纹合板座面
  - 窗外：操场 + 远山（名古屋东山方向）
  ★ 全卷教室视角必须一致，不可出现门窗位置矛盾

【玄関/鞋柜】
  - 大块地砖，两侧整面墙下駄箱
  - 木制格子，每格标签 + 室内鞋
  - 光线：晨光从东侧窗户射入

【走廊（片廊下）】
  - 一侧教室推拉门，一侧敞开窗户
  - 能看到中庭的银杏树
  - 地面：木板或水磨石

【操场】
  - 土质地面（不是草坪也不是塑胶跑道）
  - 樱花树在操场南侧
  - 单杠、攀登架、砂场
  - 校舍在北侧做背景

【中庭】
  - 银杏树（秋季金黄色——时间锚点）
  - 几丛杜鹃（春季开花）
  - 长凳，学生午休可坐

【陸家外观】
  - 二层独栋，浅米色外墙
  - 大庭院（400+平米），几棵果树
  - 小鱼池，石板小径
  - 位于富裕住宅区与老街区交界处

【葛西杂货铺】
  - 老式木造店铺，瓦顶
  - 门口摆着零食和文具
  - 葛西爷爷常坐在门口看报纸

七、每卷插画分配标准
========================================================================

每卷标准 48-56 页，插画分配：

  A 类（场景设定）：每卷 1-2 幅
     - 校舍外景、教室全景、操场、上下学路线
     - 功能：建立世界观

  B 类（关键情节）：每卷 8-12 幅
     - 发现异常、推理讨论、实验过程、真相揭晓
     - 功能：推动叙事

  C 类（人物情绪）：每卷 3-5 幅
     - 角色特写、关键时刻的表情
     - 功能：传递情感

  D 类（实验图解）：每卷 2-3 幅
     - 实验步骤图解（铁片vs玻璃片等）
     - 功能：知识可视化

  E 类（陸瑆笔记）：每卷 2-3 幅
     - 手写笔记本跨页
     - 风格：区别于主插画——用铅笔淡彩或纯线稿
     - 功能：制造双重视角的叙事层次

【第一卷具体插画分配建议】
  01：校舍外景（晨光，校门前）———— A 类
  02：玄関换鞋（陸珣第一次走进）—— B 类
  03：教室全景（陸珣找到座位）——— A 类
  04：伊藤光转身打招呼———————— B+ C 类
  05：始業式体育馆—————————— B 类
  06：陸珣画樱花（伊藤路过）———— C 类
  07：发现椅子湿了（俯拍椅面）—— B 类
  08：三人围着椅子讨论——————— B 类
  09：加藤慧美拿出温度计测窗框—— B 类
  10：陸珣说"挨着窗框"（特写）—— C 类
  11：放学后放铝片和玻璃片————— B 类
  12：第二天早上铝片上全是水珠—— B 类
  13：陸珣翻开座位图笔记本———— B+ C 类
  14：椅子挪开后，光用温度计复测—— B 类
  15：伊藤光在笔记本上写字———— B 类
  16：走廊里三人走远的背影————— C 类
  17：陸瑆笔记本跨页———————— E 类
  18-19：实验页图解—————————— D 类（2幅）


========================================================================
八、插画技术规范
========================================================================

【格式】
  文件格式：PSD（分层源文件）+ PNG（交付预览）
  分辨率：600 DPI（印刷标准）
  色彩模式：CMYK（印刷）
  出血：四边各 5mm

【构图原则】
  ✓ 每页至少 20% 留白
  ✓ 避免所有角色面向同一方向（呆板）
  ✓ 俯视/仰视交替使用（不全是平视）
  ✓ 重要的科学线索用近景/特写
  ✓ 关键道具（温度计、铝片、笔记本）必须清晰可辨
  ✗ 禁止鱼眼镜头式夸张透视
  ✗ 禁止前景虚化/背景虚化的浅景深效果（绘本不是摄影）

【人物比例一致性检查】
  插画师需提交：
    1. 场景比例网格（标注桌椅、门窗、人物的标准尺寸）
    2. 角色身高对照表（所有角色站在一起的对比图）
    3. 同一把椅子在不同角度下的标准尺寸参考


========================================================================
九、隐藏线索植入规范
========================================================================

本 IP 的核心差异化之一：插画中埋藏跨卷线索。

【规则】
  - 每卷至少 1 处隐藏线索，最多 3 处
  - 隐藏线索在当前卷的叙事中不起眼、不干扰阅读
  - 线索在未来某卷成为关键信息
  - 线索需在制作时记录坐标（卷号 + 页号 + 画面位置）

【示例】
  第3卷《蚂蚁的队列》中，操场角落一个不起眼的井盖图案
  → 第28卷发现地下旧管道走向时，这个图案成为解密关键

【线索管理表】（需插画师与故事策划共同维护）
  线索编号 | 首次出现卷/页 | 揭示卷 | 线索内容 | 画面位置

========================================================================
十、AI 生图辅助指南（如使用 AI 辅助创作）
========================================================================

☆ 重要前提：AI 生图仅作为概念探索/草图阶段的辅助工具。
   正式绘本插画应由插画师基于本规范手工绘制。

【如果使用 Midjourney / Stable Diffusion 进行概念探索】
  通用风格 prompt 模板：
  "Japanese elementary school classroom, watercolor and colored pencil style,
   warm morning light, sliding wooden door, vinyl tile floor, real classroom,
   children's book illustration, Ghibli background art inspiration, soft shadows,
   no manga style, no chibi, no exaggerated expressions --ar 4:3"

  角色 prompt 模板：
  "[角色名], [关键特征1], [关键特征2], Japanese elementary school student,
   wearing [本季穿搭], natural pose, children's book illustration style,
   watercolor texture, soft lighting, neutral expression, looking at [观察对象]"

【注意】
  AI 生图无法保证角色一致性。跨图保持人物不变只能靠：
    a) 角色三视图作为 ControlNet 输入
    b) 训练专属 LoRA 模型
    c) 或——用真人插画师（推荐）
