# Vol1 封面三方向 · 可执行计划与验收标准

> **版本**：V1.0  
> **日期**：2026-06-07  
> **适用对象**：《学堂奇事録》Vol1  
> **目标读者**：日本小学五、六年级为核心，宣传覆盖 9–12 岁  
> **项目目标**：以三种不同的销售命题完成封面方向验证，选出一个可进入商业精修与 32p 实体样书的主方向。

---

## 0. 核心裁决

本轮不是“设计三张漂亮封面”，而是验证三种不同的商业假设：

| 方向 | 商业假设 | 第一卖点 |
|---|---|---|
| **A · 异常事件型** | 儿童首先被“日常中出现了一件奇怪的事”吸引 | 翘边海报、风与可观察线索 |
| **B · 角色行动型** | 儿童首先被“这群孩子正在一起调查”吸引 | 四人观察社的行动、关系与角色魅力 |
| **C · 系列徽章型** | 儿童和家长首先被“这是一个可持续收集的系列品牌”吸引 | 社团徽章、校园地图、事件符号、卷号系统 |

### 最终出口

只有完成以下五个 Gate，才能锁定封面方向：

`COV-G0 Brief LOCK` → `COV-G1 Concept PASS` → `COV-G2 Three Covers Ready` → `COV-G3 Market Test PASS` → `COV-G4 Direction LOCK`

最终状态码：

`VOL1_COVER_DIRECTION_LOCKED`

---

# 1. 三个方向共同不变的正典条件

三方向只能改变“第一视觉卖点”，不得改变以下内容：

1. **系列名**：使用仓库 LOCK 的日文系列名。
2. **卷号**：Vol.01 / 01 必须清晰可见。
3. **卷名**：使用仓库最终 LOCK 的日文卷名；案件名不得替代卷名成为封面主标题。
4. **目标年龄**：日本 10–12 岁，不做低龄启蒙书视觉。
5. **品类识别**：校园、观察、趣事、推理；不得被误判为恐怖、犯罪、灵异或教辅。
6. **内容气质**：日常异常、可验证科学、零恐怖、善意和童心。
7. **角色原则**：第一主角最小化；四人行动核心要有群体感，但每人功能不可混同。
8. **风格禁区**：
   - 禁止 Q 版大头幼儿化；
   - 禁止侦探帽、放大镜、警戒线等陈旧犯罪推理套件成为主视觉；
   - 禁止暗黑、惊悚、鬼影、阴森走廊；
   - 禁止只靠文字解释卖点；
   - 禁止三个方向最后被折中成同一张“全都要”封面。

# 2. 项目阶段与时间建议

标准生产周期建议为 **20–24 个工作日**，不含外部试读招募等待时间。

| 阶段 | 建议工期 | 阶段目标 | 核心交付 | Gate |
|---|---:|---|---|---|
| **P0 基线冻结** | 2 天 | 锁定文字、规格、禁区与测试方法 | Cover Brief、信息层级图、测试表 | COV-G0 |
| **P1 黑白概念** | 4 天 | 证明三个方向真正不同 | 18 张缩略草图 | COV-G1a |
| **P2 构图与风格帧** | 4–5 天 | 每方向选出可商业化的一条路径 | 6 张中精度方案 | COV-G1b |
| **P3 三稿完整封面** | 5 天 | 形成可测试的三套候选商品 | 三套前封+书脊+封底+缩略图 | COV-G2 |
| **P4 市场验证** | 5–7 天 | 用儿童、家长和专业者数据判定 | 测试数据、访谈记录、评分表 | COV-G3 |
| **P5 方向锁定** | 2 天 | 选出主方向并形成精修命令 | Direction LOCK、整改清单 | COV-G4 |
| **P6 印前适配** | 后续 3–5 天 | 为 32p 实体样准备正式外装 | 封面展开、印前检查 | PRE-PROOF |

# 4. P1 · 黑白概念草图

## 4.1 产量要求

每个方向先出 **6 张黑白缩略草图**，共 18 张。

要求：

- 不上色；
- 不画细节；
- 只验证主视觉、视线动线、标题位置和人物比例；
- 每张草图 5–15 分钟量级，避免过早沉没成本。

## 4.2 每方向内部必须包含的构图变化

### A · 异常事件型

至少尝试：

1. 海报翘角近景，人物在背景观察；
2. 风的路径贯穿封面，线索分布在路径上；
3. 校园墙面/走廊构图，异常点成为视觉中心；
4. 海报与观察笔记形成双层画面；
5. 证据物特写，人物只以局部进入；
6. 异常现象与校园空间形成强透视。

### B · 角色行动型

至少尝试：

1. 四人围绕同一证据，动作分工明确；
2. 陸珣观察、光发现、慧美记录、志郎验证的连续动线；
3. 从读者视角看四人正在调查；
4. 走廊移动调查，而非站桩合影；
5. 团队围成半圆，中央留出谜题；
6. 四人不同高度、姿势和视线形成团队节奏。

### C · 系列徽章型

至少尝试：

1. 社团徽章为中心，事件符号环绕；
2. 校园地图为底，案件节点与路径叠加；
3. 档案封套/观察记录视觉，但避免教辅和公文感；
4. 大卷号与徽章组合；
5. 校园标志物、海报角、风线、笔记符号组成系列图腾；
6. 小比例角色剪影或头像作为品牌辅助。

## 4.3 P1 检验标准

- 三个方向去掉标题后仍可区分；
- A 的第一眼中心必须是“异常”；
- B 的第一眼中心必须是“孩子们正在做事”；
- C 的第一眼中心必须是“系列品牌符号”；
- 任一草图不得依靠长宣传语解释画面；
- 每个方向选出 2 张进入 P2，共 6 张。

**淘汰条件**：三个方向只是人物大小或背景颜色不同，直接判定 `REWORK`。

# 5. P2 · 构图与风格帧

每个方向保留 2 张，共 6 张中精度方案。

## 5.1 交付规格

每张必须包含：

- 主要色块；
- 人物/事件/徽章的基本造型；
- 日文书名真实排版；
- 卷号位置；
- 书架缩略图；
- 灰阶版本；
- 视觉焦点说明；
- 制作复杂度评估。

## 5.2 共同检验标准

| 维度 | 检验标准 |
|---|---|
| 3 秒识别 | 无口头说明时，测试者能判断为儿童校园故事 |
| 品类 | 不低于 70% 测试者提到“谜、调查、观察、奇怪事件”之一 |
| 年龄 | 不被多数人判断为 6–8 岁低幼读物 |
| 安全 | 恐怖/犯罪/灵异误判合计 ≤10% |
| 标题 | 标题不压住人物脸、证据点或徽章核心结构 |
| 缩略图 | 120–180px 下主视觉仍成立 |
| 灰阶 | 去色后层级不坍塌，主体与背景可区分 |

每方向选出 1 张进入 P3，共 3 张完整候选封面。

# 6. 三个方向的具体执行与专属验收标准

# A · 异常事件型

## A1. 销售命题

> “学校里发生了一件不吓人、但很奇怪，而且可以被观察和解释的事。”

## A2. 主视觉构成

建议视觉权重：

- 异常事件/翘边海报：35–45%
- 风与线索路径：15–25%
- 校园环境：15–20%
- 人物观察反应：20–30%

## A3. 必须呈现

- 海报“翘边”是明确异常，不是普通破损；
- 风应体现方向和因果线索，不是装饰流线；
- 人物处于观察状态，不是惊恐逃跑；
- 画面让读者自然产生“为什么会这样？”的问题。

## A4. 禁止表现

- 海报像被人为破坏或恶意撕毁；
- 风像超自然力量；
- 走廊昏暗阴森；
- 角色大幅惊恐表情；
- 线索全部揭底，失去阅读悬念。

## A5. 专属测试指标

| 指标 | PASS |
|---|---:|
| 3 秒后能指出异常对象 | ≥75% |
| 自发提出“为什么海报会翘/风从哪里来” | ≥60% |
| 判断为恐怖或破坏事件 | ≤10% |
| 表达想知道答案/想翻开书 | 儿童 ≥65% |
| 能看出人物是在观察而非害怕 | ≥70% |

## A6. 方向失败定义

- 海报异常不够清楚；
- 画面只有事件，没有系列角色记忆；
- 看起来像单本事件绘本，无法支持后续卷；
- 科学观察感弱于惊险感。

---

# B · 角色行动型

## B1. 销售命题

> “我想跟这四个性格不同的孩子一起调查学校里的奇怪事情。”

## B2. 主视觉构成

建议视觉权重：

- 四人行动核心：55–65%
- 谜题/证据中心：15–25%
- 校园空间：10–20%
- 品牌与卷号：固定清晰区

## B3. 四人动作必须体现功能

| 角色 | 封面动作语言 |
|---|---|
| 陸珣 | 安静观察、比较、注意微小异常 |
| 伊藤光 | 最先发现、指向现场、连接大家 |
| 加藤慧美 | 记录、整理、确认别人忽略的信息 |
| 松本志郎 | 动手验证、测量、尝试复现 |

人物不能只是“四张脸”或“四人站成一排”。

## B4. 必须呈现

- 四人围绕同一问题行动；
- 动作和视线形成闭环；
- 四人脸型、发型、服装、姿势轮廓可区分；
- 陸珣是视角中心但不成为传统超级主角；
- 角色之间有同伴感和日常幽默感。

## B5. 禁止表现

- 全员摆拍；
- 主角占画面 60% 以上，其他人成陪衬；
- 侦探帽、放大镜、推理制服化；
- 四人表情雷同；
- 过于热血战斗或少年漫画式冲锋。

## B6. 专属测试指标

| 指标 | PASS |
|---|---:|
| 3 秒后能判断为“几个孩子一起调查” | ≥75% |
| 能说出至少 3 人动作不同 | ≥60% |
| 至少 2 名角色被儿童主动记住 | ≥70% |
| 被误判为普通校园友情故事 | ≤20% |
| 被误判为战斗/犯罪侦探故事 | ≤10% |
| 儿童表示“想加入他们/跟他们一起查” | ≥60% |

## B7. 方向失败定义

- 人物漂亮但谜题不明确；
- 四人功能差异无法从动作中看出；
- 封面只适合 Vol1，不能长期复用角色结构；
- 角色数量导致手机缩略图变成一团。

---

# C · 系列徽章型

## C1. 销售命题

> “这是一个有自己徽章、地图、规则和事件档案的校园观察系列，我想继续收集下一卷。”

## C2. 主视觉构成

建议视觉权重：

- 社团徽章/系列图腾：30–40%
- 校园地图或结构线：20–30%
- 本卷事件符号：15–20%
- 角色辅助：10–20%
- 卷号与卷色：强识别

## C3. 系列系统必须证明可扩展

至少模拟：

- Vol.01 当前卷；
- Vol.02 假想下一卷；
- Vol.03 假想第三卷；
- 三本并排书脊；
- 电商系列页九宫格；
- 单色徽章和小尺寸图标版本。

## C4. 必须呈现

- 徽章不是普通学校校徽，而是“校园趣事观察社”的独特识别；
- 地图和事件符号要有探索感，不做教材知识图谱；
- 卷号、卷色和本卷异常符号可快速替换；
- 系列名在缩略图与书脊上稳定识别。

## C5. 禁止表现

- 过度像学校手册、教辅、文具或社团申请表；
- 徽章复杂到小尺寸不可辨；
- 缺少儿童人物温度；
- 地图信息过多；
- 依赖金属、烫金、特殊工艺才能成立。

## C6. 专属测试指标

| 指标 | PASS |
|---|---:|
| 120px 下仍能识别徽章或系列名 | ≥80% |
| 三卷并排时能识别同系列 | ≥85% |
| 儿童产生“收集/想看下一卷”表达 | ≥60% |
| 被误判为教辅、校刊或工具书 | ≤15% |
| 只看书脊能辨识系列和卷号 | ≥80% |
| 单色版本仍保持识别 | 专业审查 PASS |

## C7. 方向失败定义

- 品牌性强，但故事和人物温度不足；
- 像文具包装而不像儿童读物；
- 需要大量解释才能知道这是推理故事；
- 徽章无法在 20 卷中持续变化和保持统一。

# 7. P3 · 三套完整候选封面

三方向各产出一整套，不只是一张前封图。

## 7.1 每套必交文件

1. 前封面；
2. 书脊；
3. 封底；
4. 封面展开模拟；
5. 120px、180px、360px 电商缩略图；
6. 书架并排模拟；
7. 只露书脊模拟；
8. 灰阶版；
9. 单色打印版；
10. 无宣传语版；
11. 人物与关键物件分层文件；
12. 设计说明一页纸。

## 7.2 公平比较原则

- 三稿使用相同书名文字、作者信息和尺寸；
- 三稿精细程度一致；
- 不得让偏好的方向完成度更高；
- 测试顺序随机，避免 A 永远先出现；
- 测试者不知道设计团队偏好。

**COV-G2 PASS**：三套文件齐全，具备同等商品完成度，可以盲测。

# 8. P4 · 市场验证计划

## 8.1 建议样本

本轮为封面方向验证，不替代全卷读者测试。

| 群体 | 最低样本 | 建议样本 |
|---|---:|---:|
| 日本 10–12 岁儿童 | 18 | 24–30 |
| 日本家长 | 8 | 12–15 |
| 儿童书编辑/设计师/司书/教师 | 5 | 6–8 |

儿童、家长、专业者的分数必须分开报告，不得直接混成“平均满意度”。

## 8.2 测试模块

### T1 · 3 秒闪现

每张显示 3 秒后隐藏，询问：

- 这是什么类型的书？
- 你第一眼看到什么？
- 你觉得会不会可怕？
- 你觉得书里的人在做什么？

### T2 · 电商缩略图选择

将三稿与 8–12 本同类书缩到相同尺寸，询问：

- 第一想点哪一本？
- 哪一本最像系列第一卷？
- 哪一本最容易记住？

### T3 · 实体书架模拟

模拟正面陈列、侧放、只露书脊三种状态。

### T4 · 五分钟记忆

看完一组封面五分钟后询问：

- 记得哪个画面？
- 记得哪个人物/符号？
- 记得系列名或卷号吗？

### T5 · 购买与续卷

分别问儿童“想不想读”、家长“是否愿意购买”、专业者“是否适合产品线”。

## 8.3 共同量化指标

| 指标 | 门槛 |
|---|---:|
| 校园属性识别 | ≥75% |
| 趣事/观察/谜题属性识别 | ≥70% |
| 恐怖/灵异/犯罪误判 | ≤10% |
| 目标年龄判断正确 | ≥65% |
| 儿童翻开意愿 | ≥60% |
| 家长购买或考虑购买 | ≥60% |
| 专业者认为品类清晰 | ≥70% |
| 标题或系列名记忆 | ≥60% |

## 8.4 权重

| 评价来源 | 权重 |
|---|---:|
| 儿童吸引力与理解 | 40% |
| 家长购买与安全感 | 20% |
| 出版/设计专业判断 | 25% |
| 系列扩展与生产可行性 | 15% |

每个方向计算总分，但必须同时满足“恐怖/犯罪误判 ≤10%”这一硬门槛。硬门槛失败时，总分再高也不得直接胜出。

# 9. P5 · 方向锁定规则

## 9.1 选择规则

主方向必须同时满足：

1. 所有共同硬门槛通过；
2. 对应方向的专属指标通过；
3. 加权总分排名第一；
4. 与第二名至少拉开 5 分，或在关键儿童指标上明显领先；
5. 艺术总监确认可执行，出版社/印刷生产无明显障碍。

## 9.2 不允许的决策方式

- 不以 IP Owner 单人审美代替数据；
- 不以“大家都喜欢一点”做平均化折中；
- 不把三个方向所有元素全部塞入最终封面；
- 不因某稿完成度较高而误判方向更优；
- 不在结果出来后修改测试标准。

## 9.3 混合规则

允许胜出方向吸收第二方向的辅助资产，但必须遵守：

- 主方向销售命题保留 **≥80%**；
- 外来元素只承担辅助作用 **≤20%**；
- 吸收后需重新做一次 3 秒识别测试；
- 不能造成第一视觉卖点不清。

示例：

- A 胜出时，可以使用 C 的小徽章，但不能让徽章抢走异常事件中心；
- B 胜出时，可以使用 A 的翘边海报作为四人调查对象；
- C 胜出时，可以使用 B 的小比例角色群像增加温度。

## 9.4 最终交付

- `Vol1_Cover_Direction_LOCK.md`
- 胜出方向原图与分层文件
- 未胜出方向归档说明
- 测试数据和原始记录
- P0/P1/P2 整改清单
- 32p 实体样书封面生产 brief
