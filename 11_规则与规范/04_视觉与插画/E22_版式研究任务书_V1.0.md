## 一、任务目标

为《学堂趣事录》建立一套适用于 **中国首发**、故事背景位于 **日本名古屋**、目标年龄 **9–12 岁** 的 **儿童图文校园推理章节书** 出版版式系统。

最终成果须同时满足：

1. 儿童能够 **独立阅读**（单次 15–20 分钟 · 每案可读毕）；
2. **正文仍是主要叙事载体**（面积比 65–70%）；
3. 插图能够表现 **人物、空间、线索和情绪**（15–18% · 非复述正文）；
4. **不像**低龄绘本、纯文字小说、知识教辅、连续漫画格主体；
5. 能够形成 **20 卷以上** 的稳定系列视觉系统（书脊 · 卷色 · 模板复用）；
6. **32p 样章优先于 160p 整卷** — 字号与混排须经 **屏幕 PDF @100% A5** 验收（[`屏幕PDF打样清单`](../07_设计原档/05_读者版排版/E22_屏幕PDF打样验收清单_V1.0.md)）；印前实印延至出版社清样阶段。

**产品分类（对外 LOCK）**：

> **儿童图文校园推理章节书**  
> 9–12 岁自主阅读型校园趣事推理系列  
> （设计层补充：桥梁式低阻力阅读 · 短章 · 高留白 · 插图辅助）

## 二、竞品研究要求

请寻找 **8–12 套纸质书** 作为版式研究对象（纸质优先；PDF/电子预览仅作补充）。须覆盖 A/B/C/D 四类，**每类至少 2 套**。

### A 类 · 儿童推理图文小说

重点研究：

- 谜题如何进入章节；
- 插图怎样展示 **公平线索**；
- 推理前后怎样制造翻页节奏；
- 人物介绍怎样安排；
- 章节/案首怎样设计（对照我们 **半页标题条**）。

**建议书目（可替换同等品类）**：《放課後ミステリクラブ》《ぼくらは迷宮探偵団》《かいけつゾロリ》（节奏参考）

### B 类 · 小学中高年级章节书

重点研究：

- 正文字号、行距、每行字数、每页行数；
- 章节/案平均页数；
- 插图频率与留白；
- 读者自主阅读门槛（ルビ 密度策略）。

### C 类 · 系列化儿童文学

重点研究：

- 系列 Logo、卷号、卷名、书脊；
- 每卷代表色、封面模板、套装陈列；
- 下卷预告、角色资产复用。

### D 类 · 知识融入型故事

重点研究：

- 知识图解放在哪里（**须在故事揭晓后** · 对标我们 B 轨）；
- 如何避免教辅感；
- 实验页、地图页、笔记页怎样与正文区分；
- 图解是否打断故事节奏。

**研究原则**：拆解 **设计逻辑**，不得直接复制任何竞品的字体、构图、角色造型或版面模板。  
**借《放課後ミステリクラブ》**：短章 · 人名前置 · 地图/线索工具化 · 160p 量级 — **不借**：Q 版主视觉 · 竖排辞典卷首页 · 全书全注音。

### E 类 · 全球 IP 排版能力拆分（L1–L5 · 2026-06-06 增补）

> 详审 [`专家组审阅_五品牌排版组合系统_20260606.md`](../00_项目总览/专家组审阅_五品牌排版组合系统_20260606.md) · 一页纸 [`E22_五层排版组合_一页纸_V1.0.md`](./E22_五层排版组合_一页纸_V1.0.md)

## 五、书名四级层级

封面和扉页必须明确区分四级，**不得把五个案件全部堆到封面**。

| 级 | 内容 | 封面是否出现 |
|----|------|:------------:|
| **L1 系列** | 学堂奇事録（日）/ 学堂趣事录（中） | ✅ |
| **L2 卷号** | 01、02… | ✅ |
| **L3 卷名** | Vol1：**觉得奇怪，就先观察**（おかしいと思ったら、まず見てみる） | ✅ |
| **L4 案名** | 例：翘边的海报 / めくれたポスター | ❌ 不进封面主标题 |

**标准格式**：

> 《学堂趣事录 01：觉得奇怪，就先观察》

**卷内案名（目录级）**：

| 案 | 中文 | 日文（正典） |
|----|------|-------------|
| ① | 翘边的海报 | めくれたポスター |
| ② | 错位的泥印 | ずれた泥のあと |
| ③ | 空着的那一栏 | 空いているマス |
| ④ | 粉笔灰的圆圈 | チョークの粉の輪 |
| ⑤ | 橡皮屑的方向 | 消しゴムの屑の向き |

**废止**：旧卷名「总是湿的椅子」「いつも濡れる椅子」及封面「椅子与窗框」主视觉。

## 六、五级字体系统 T1–T5

| 层级 | 用途 | 变化方式 | Vol1 LOCK |
|------|------|----------|-----------|
| **T1** | 系列 Logo | 独立品牌字形或原创标 | 全系列固定 |
| **T2** | 卷名 | 有个性但清晰 | 封面/扉 |
| **T3** | 案件标题 / 案首条 | 每案可配图标+本卷色 | **半页标题条** |
| **T4** | 正文 | 全书统一 · 阅读优先 | **11–12pt** · 行距 **1.5–1.65** |
| **T5** | 日记、壁报、便签、图注 | 手写感 · 仅特殊栏目 | 瑆 **9–10pt** 横排 |

**要求**：

- 正文优先考虑长期阅读舒适度；
- 字体必须具备 **合法商业授权**（交付授权清单）；
- 全书主体字体 **≤3 种**（T4 一种 + 强调一种 + T5 栏目一种）；
- 手写字体 **仅** 用于 T5 栏目；
- 画面内日文环境文字（壁报/海报/门牌）与正文 T4 **明确分工**；
- 陸瑆日记手写体系建议 **原创设计**（非直接套用开源手写字体）。

### ルビ（Ruby）排版

| 项 | 规则 |
|----|------|
| **术语** | **ルビ** / **振り仮名** · **禁止「振假名」「全振假名」** · 仅 **ひらがな/カタカナ/漢字** · 标日规范 · 见 V1.1 |
| **策略** | 超四年级配当汉字 **首次** + **案首复现** · 不全书全注音 |
| **位置** | 横排：汉字**上方** · 5–6pt |
| **A 类必标** | 人名 · 地名设施 · 科学术语 · 表外汉字 |

### L3 表演字（老鼠记者层 · Gate E22-B）

| 项 | LOCK |
|----|------|
| 适用范围 | 拟声 · 案名半页条 · 物理词（风/湿/摩擦）· 光的高能台词 · 瑆夸张（单页可至 15%） |
| 全书上限 | **5–8%** 版心文字面积（T4 正文 **禁止**整段变色） |
| 日版 | 听出版社；优先片假名拉长/字号 · 慎用彩色变形 |
| 中文版 | 可彩色/倾斜/变形 · 须 E22 样张标注 |

**禁止**把正文设计成多色教辅排版。

## 八、插图与文字关系规则（P0）

插图须承担至少一种功能：**空间 / 人物关系 / 公平线索 / 科学现象 / 情绪 / 翻页钩子 / 第二层信息**。

### 8.1 混排 LOCK

- **插图与文字混排** · 图在相关 **段落后**；
- **禁止**：整页单图占码（**除** approved 跨页 spread）；
- **禁止**：图被分页切断 — 实现 `break-inside: avoid`（`.scene-block` / `.figure-wrap`）；
- 每 **4–6p** 至少一次视觉变化；
- 连续纯文字 **≤6p**。

### 8.2 B 轨（§4 收束之后）

以下模块 **禁止** 插入 §1–3 正文流：

- DB1 机制示意
- CL 线索卡
- 家庭实验
- V-S02 陸瑆日记

### 8.3 禁止复述

- 禁止仅将正文内容重复画一遍；
- 关键证据须在谜底揭晓前出现，可位于画面第二视觉层；
- 图注指场景 · **不写答案句**。

## 九、三套版式方向 A / B / C

三套方案须使用 **同一段正文**（A001 序+案①节选）与 **同一组插图** 对照，不得用不同内容干扰判断。

| 方向 | 特征 | 优先级 |
|------|------|:------:|
| **A · 文学感较强** | 正文比例高 · 插图克制 · 留白充分 · 偏 10–12 岁 | 卷 2+ 选项 |
| **B · 图文平衡** | 正文:图 ≈ **7:3**（对齐 65–70% / 15–18%）· 每 4–6p 视觉变化 | **★ 主推荐 · 首轮交付** |
| **C · 视觉参与较强** | 线索卡/路线图/日记页更多 · 互动感强 · 须防教辅/游戏书化 | 卷 2+ 选项 |

**专家组裁定**：**先交付方向 B** 的 20–32p **屏幕 PDF 样章**；A/C 作为后续卷选项备案。

## 十、32p 样章优先于 160p

**禁止**首轮直接排 128–160p 整卷。

### 10.1 首轮原型：≤32p 真实排版样章（A001 试读）

建议页序（对齐 [`V1.3 锁页`](../03_故事内容/第1卷_觉得奇怪就先观察/样章包/03_案01_分镜头与插页地图_V1.3_锁页版.md) + [`V1.4 五层标注`](../03_故事内容/第1卷_觉得奇怪就先观察/样章包/03_案01_分镜头与插页地图_V1.4_五层标注.md)）：

| 模块 | 验证点 |
|------|--------|
| 封面 / 扉 / 版权 | 四级书名层级 |
| 人名表 / 目录 | ルビ · 目录粒度 |
| 序 + 案①开篇（半页标题条） | 案首不占独立码 |
| 混排正文 + 段后插图 | **不分页切断** |
| 线索特写 / 收束 | B 轨位置 |
| 陸瑆日记 / 家庭实验 | T5 辨识度 |
| 下案尾钩 / 封底 | 翻页节奏 |

### 10.2 32p 须验证（屏幕 PDF @100% A5）

- 字号/行距是否舒服（**100% 缩放** · 禁「适合页面」）；
- 图文比例 · L1–L5 五层各可见；
- 角色是否过幼；
- 线索是否看得清；
- 瑆 T5 辨识度；
- **RD-06** 无图顶细条切页；
- 试读 **≥20 分钟** 连续翻页（E04/IP）。

**数字基准稿**：[`build_reader_edition.py`](../03_故事内容/第1卷_觉得奇怪就先观察/正式版/05_出版成果/tools/build_reader_edition.py) 产出 Reader PDF · 设计师样章 **页序须一致** · 质量 **≥ Reader**。

### 10.3 160p 整卷（P2 · 样章 PASS 后）

- 五案锁页地图 ×5；
- InDesign 母版（HTML 为过渡，非终态）；
- PRODUCT-GATE 出厂表。
