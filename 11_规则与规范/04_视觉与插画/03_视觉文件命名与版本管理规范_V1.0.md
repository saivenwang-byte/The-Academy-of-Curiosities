# 视觉文件命名与版本管理规范 · V1.0

> 链 [`04_视觉生产体系` §二](../04_视觉生产体系_分镜补充2_V1.0.md)

## 命名

```
角色名_资产类型_视角_季节_版本号_日期[_APPROVED].扩展名
```

## 禁止

`最终版` · `最终版2` · `真的最终版` · 无版本号覆盖

## 流程

修改 → 版本号 +1 → 旧版 `_archive/` → PRODUCT 仅 `APPROVED`

## 追溯

每 PNG 元数据或 sidecar 记录：`canon_role_version` · `canon_scene_version` · `shot_map_version`
