## 1. 双线验收（本轮）

| 管线 | 角色 | 通过标准 |
|------|------|----------|
| **Reader Edition PDF** | IP / E03 / E04 **节奏与混排** | RD-01～RD-07 · ≤32p |
| **设计师方向 B PDF/INDD** | E22 **正式版式** | E22-01～E22-12 · 与 Reader **页序一致** |

**原则**：Reader PDF = **数字打样基准**；设计师样章须 **≥ Reader 混排质量**，不得劣于过渡稿。

---

## 2. 屏幕打样环境（强制）

| 项 | 设置 |
|----|------|
| 查看器 | Adobe Acrobat / Chrome 打印预览 / 系统 PDF @ **100%** |
| 页面尺寸 | **A5 148×210mm**（禁「适合页面」缩放审版） |
| 背景 | 浅色 · 模拟内文纸 **#F7F4EF** 参考（非强制） |
| 连续阅读 | 一次翻页 **≥20 分钟** 试读（IP 或 E04） |

---

## 3. RD 验收（Reader · build 产出）

| # | 检查 | 方法 |
|---|------|------|
| RD-01 | 无门禁/四栏/Shot 全表 | 目视 |
| RD-02 | 序+A001 **≤32p** | `pypdf` 或 Acrobat 页数 |
| RD-03 | 图锚段落可找 | 每 DA 对应 §+首句 |
| RD-04 | B 轨在 §4 后 | DB1→CL→实验→瑆 |
| RD-05 | 连续纯文字 ≤6p | 人工翻页 |
| RD-06 | 无图顶细条切页 | **100% 逐页** |
| RD-07 | L1–L5 各 ≥1 次 | 对照 [`V1.4_五层标注`](../03_故事内容/第1卷_觉得奇怪就先观察/样章包/03_案01_分镜头与插页地图_V1.4_五层标注.md) |

## 4. 屏幕字号/heuristic（替代实印）

| 项 | 屏幕检查 |
|----|----------|
| 正文 11–12pt | A5 @100% · 每行 **24–32 全角** · 每页 **15–20 行** |
| ルビ 5–6pt | 放大 **150%** 抽检可读 |
| 半页图 48mm | 版心高约 **23%** · 不与下页文字分离 |
| L3 表演字 | 全书 **≤8%** 页 · 单页目测不过载 |
