## 三、为什么他们会「同时出现」？——没有合理理由

| 误区 | 事实 |
|------|------|
| 「观察社四个人 + 同学」 | 同学必须是 **5年2組 匿名群众** · 不是再画 4 个 **全脸全服** 的「准主角」 |
| 「L0 十人都要露面」 | L0 是 **造型排面** · 不是每帧全员打卡 |
| 「异班可以都在」 | **仅** 当帧有采访/查证任务的观察社会员（慧美·志郎）· **不是** 理纱·中谷·瑆 |
| 「GenerateImage 填背景」 | Prompt 写 `classmates gesturing` 时模型会 **克隆 L0 多样性** → 必须写 **max 2 anonymous silhouettes, no distinct faces** |

**结论**：8 人不是编辑 intentional · 是 **G-TEXT 缺人数上限 + 出图后未做 G-CAST 人头清点**。

## 五、强制门禁 · G-CAST（与 G-TEXT 并列）

**在 GenerateImage 之前**，每帧 Shot Map / 分镜文字稿 **必须** 填：

```markdown
### G-CAST（必填）
| 字段 | 内容 |
|------|------|
| 场景 | 例：5年2組 合班教室 · 四月上午 |
| 具名 P0 | 名单 · 每人 1 句在场理由（链正文锚句） |
| 具名 P1 | 可选 · 如水野仅 DA5b |
| 匿名群众 | 有/无 · **上限 N** · 仅背影/无 L0 脸 |
| 禁止具名 | 瑆·理纱·中谷·葛西·父母·… |
| **MAX_BODIES** | 整数 · DA2=6 · DA4=2 · … |
| **COUNT_PASS** | 成图后人工/Agent：**可见全设计人物 ≤ MAX** |
```

**PASS 条件**：
1. MAX_BODIES 与上表算术一致  
2. 成图可数人头 ≤ MAX  
3. 无禁止具名 · 无 L0 槽位脸冒充「同学」  
4. 异班角色仅在有 **采访/查证/邀请** 的正文锚时出现  

**FAIL → REJECT · 不得绑定 PDF · 不得标 ✅**
