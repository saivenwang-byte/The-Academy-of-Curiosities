## 三、Style B · 马克笔 LOCK（修正 AI 过重水彩）

**IP 原话**：「马克笔更重一点 · 水彩感太重」

| 层 | Style B（采用） | 禁（当前 AI 稿） |
|----|-----------------|------------------|
| **线** | 清晰马克线稿感 · 外轮廓略粗 · `#2A1810` | 糊边 · 线被水洗吃掉 |
| **色** | **马克笔平涂 + 边缘轻晕染** · 色块分明 · 饱和度中 | 大面积 wet wash · 灰蒙颗粒 |
| **肌理** | 轻纸纹 **≤15%** · 局部水彩点缀 | 全图水彩纹理 dominant |
| **光** | 四月侧光 · 电影空间 · **色阶清楚** | 整体发灰发脏 |

**Prompt 关键词（正篇 Global）**：
```
alcohol marker illustration style, Copic-like flat color blocks with soft marker blending at edges,
clean warm brown ink outline #2A1810, light paper grain only NOT heavy watercolor wash,
vibrant but not neon, children's mystery ages 10-12, same character designs as reference lineup sheet
NEGATIVE: heavy watercolor, wet-on-wet bleeding, muddy desaturated, grainy wash dominating, chibi
```

**垫图顺序**：① IP 十人排面 → ② 本案四人 crop → ③ 场景构图 ref（仅空间 · 禁抄 ref 人物）

**举牌（2026-06-10）**：画内 **仅日语**（假名 + 日文汉字）· **禁中文** · 见 [`CHAR_lineup_L0_StyleB_日文举牌_V0.2_brief.md`](../../../07_设计原档/04_样章视觉/CHAR_lineup_L0_StyleB_日文举牌_V0.2_brief.md)
