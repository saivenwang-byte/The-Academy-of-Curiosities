## 1. 议题与 IP 方提案（摘要）

IP 方提交：

1. 年龄：小五小六 10–12 岁为主；四高年级可读。  
2. 日版每卷正文 35k–45k 字符；Vol1 宜 32k–38k。  
3. 144–176 页；4 短 + 1 中长；单案 5k–7k JP · 案⑤ 9k–12k。  
4. 版式：正文 65–70% · 图 15–18% · 瑆 7–10% · 线索 7–10%。  
5. 读音标注：年级适配 + 首次标音（非全书全注音）。  
6. 句段：普通句 20–35 字符 · 每 1000–1500 字符一次情节回报。  
7. 三难度层：Vol1–4 入门 · Vol5–12 稳定 · Vol13–20 进阶。

### 4.4 版式与阅读阻力

| 内容 | 占比 |
|------|------|
| 正文故事 | 65–70% |
| 场景/人物插图 | 15–18% |
| 陸瑆日记 | 7–10% |
| 线索/复盘/路线图 | 7–10% |

| 规则 | 值 |
|------|-----|
| 每 4–6 页 | 至少 1 次视觉变化 |
| 每 8–12 页 | 至少 1 张较完整插图 |
| 连续纯文字 | ≤ 6 页 |
| 每案章节 | 6–10 页 |
| 每 1,000–1,500 字符 | 至少 1 次明显推进（线索/笑/冲突/错推/图/情绪/小答案/钩） |
| 每 20 分钟 | 至少 **2 项** 回报 |

### 4.5 日文句段与读音标注（E04 LOCK）

> 专档：[`日文读音标注策略_V1.0.md`](./日文读音标注策略_V1.0.md)  
> **市场参照**：《放课后ミステリクラブ》全汉字附注音 → 低门槛儿童推理可接受高注音密度；本 IP 主读者 **10–12 岁**，采用 **年级适配 + 首次标音**，非全书全注音。

| 项 | 标准 |
|----|------|
| 普通句 | 20–35 日文字符 |
| 复杂句 | ≤ 45–50 字符 |
| 段落 | 1–4 句 |
| 对话占比 | 35–45% |
| 四年级前配当汉字 | 原则上不逐字标音 |
| 五年级在学汉字 | 首次可标音 · 后文省略 |
| 六年级+/地名/人名/术语 | 必须标音（首次 + **案/序首复现**） |
| 陸珣/陸瑆/中谷琦等 | 首次 + 案首复现标音 · 光/瑆同音汉字区分 |
| 注音密度 | 低于全注音竞品 · E04 终检「独立读不卡」 |
