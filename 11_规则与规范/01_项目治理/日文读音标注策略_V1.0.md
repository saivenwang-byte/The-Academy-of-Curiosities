## 0. 用词定案（IP 锁 · 2026-06-05）

| 正确说法 | 废止说法 | 说明 |
|----------|----------|------|
| **平假名（ひらがな）** · **片假名（カタカナ）** · **汉字（漢字）** | 「振假名」 | 日文只有假名与汉字；**不存在「振假名」这一书写体系** |
| **读音标注** · **注音** | 振假名、振り仮名（中文文档内） | 指在汉字旁/上方标出怎么读 |
| 工作稿：**括号读音** `漢字（ひらがな）` | — | 括号内一律用 **平假名**（外来语术语可用片假名） |
| 出版：**ルビ**（汉字上方标平假名读音） | — | InDesign / HTML 排版术语 |
| 竞品对标：**全书全注音** | 「全书逐字振假名」 | 放课后类：几乎每个汉字都标平假名读音 |

**一句话**：我们控制的是 **哪些汉字要标平假名读音、标几次** — 不是发明第四种「假名」。

## 1. 市场参照（为何不全书注音）

| 作品 | 注音密度 | 主读者 | 对本 IP 的启示 |
|------|----------|--------|----------------|
| **《放课后ミステリクラブ》** | **全部汉字附平假名读音**（高注音密度） | 更低门槛儿童推理 · 独立阅读起步更早 | 证明 **日本儿童推理市场接受高注音** · 低学年可卖 |
| **《学堂奇事録》** | **年级适配 + 首次标音**（selective ruby） | **小五小六 10–12 岁** · 四高年级兼容 | 目标略高龄 · **不必全书全注音** · 保留桥梁书可读性而不降龄 |

**结论**：不否定全注音路线；本系列 deliberately 取 **略高读者带** 的注音方案，与 Vol1 篇幅/句长/推理密度一致。

## 2. 总原则

**年级适配 + 首次标音**

- **年级适配**：按日本小学 **五·六年级配当汉字** 为默读基线；未配当、表外、专业词才强制注音。  
- **首次标音**：同一词在 **全卷内首次出现** 注音；叙事重复处省略，避免视觉噪音。  
- **章/案首复现**：每个 **序 · 案（A001–A005）** 开头，对人名与该案核心术语 **复标一次**（读者隔日续读用）。  
- **出版形态**：工作稿用括号 `漢字（よみ）`；InDesign/HTML **ルビ** 由 E04 定稿 · E07 抽检。

## 3. 三级规则（E04 执行表）

### A · 必须标音（首次 + 案首复现）

| 类别 | 例 |
|------|-----|
| **所有人名** | 陸珣（りく しゅん）· 伊藤光（いとう ひかる）· 加藤慧美（かとう けいみ） |
| **地名·设施专名** | 側廊（そくろう）· 下駄箱（げたばこ）· 活動準備室 |
| **科学·机制术语** | 結露（けつろ）· 露点（ろてん）· 熱伝導（ねつでんどう） |
| **同音 disambiguation（LOCK）** | **陸瑆（りく ひかる）** 妹妹 · 读音 **冻结** · **优先保证** · vs **伊藤光（いとう ひかる）** — 姓で区別 · 禁单独「ひかる」· 禁 瑆=けい · 禁 陸ひかる |
| **表外·难读汉字** | 小学五六年未配当、读法非直觉的汉字 |

### B · 首次标音，后文省略

| 类别 | 例 |
|------|-----|
| 五·六年级 **在学汉字** 内的叙事词 | 掲示板 · 異常 · 確認（案内首次后可省略） |
| 该案内 **重复 ≥3 次** 的非术语词 | 第二次起不标，除非新案开头复现 |
| 社团·固定专名 | 学校おもしろ観察クラブ（全卷首次 + 每卷序首） |

### C · 原则上不标音

| 类别 | 例 |
|------|-----|
| 四年级前配当 **高频汉字** | 学校 · 先生 · 教室 · 友だち 等（若 E04 抽检无卡顿） |
| **已标过且同案内** 的普通词 | 不重复括号 |
| **假名书写** | ひらがな · カタカナ · オノマトペ 本体 |
| **对话口语** | 自然口语假名段（勿为「统一」强行汉字化再注音） |

## 4. 密度与质检（非机械全标）

| 指标 | 目标 |
|------|------|
| 相对《放课后ミステリクラブ》 | **明显更低** 注音覆盖率 |
| 工作稿 heuristic | 约 **15–25%** 带注音汉字（随案调整 · 非硬比例） |
| E04 终检 | 以 **「小五生独立读是否卡读」** 为准，非逐字统计 |
| 田中 E07 | 人名表 · 制度词 · 名古屋校园词 **0 误读** |

**禁止**：

- 为凑密度给常用词重复标音  
- 为「显得儿童向」**全书全注音**（与主读者年龄不符）  
- 科学术语 **零注音**（推理机制词必须可读）

## 5. 工作稿格式（Vol1 现行）

```
陸珣（りく しゅん）が5年2組に来た日……   ← 哥哥 · しゅん LOCK
陸瑆（りく ひかる）のノート              ← 妹妹 · ひかる LOCK · 优先表记
伊藤光（いとう ひかる）が立ったまま       ← 与瑆同音 · 必须姓+名或汉字「光」
```

### 光/瑆 同音区分（IP 锁 · 全项目对齐）

| 角色 | 关系 | 读音 | 正文表记 | 读音初出标注 |
|------|------|------|----------|--------------|
| **陸 珣** | 哥哥 | **りく しゅん** | 陸珣 / 珣 | 陸珣（りく しゅん） |
| **陸 瑆** | 妹妹 | **りく ひかる** | 陸瑆 / 瑆 | **陸瑆（りく ひかる）** · 笔记 **陸瑆ノート** |
| **伊藤 光** | 社员 | **いとう ひかる** | 伊藤光 / 光 | **伊藤光（いとう ひかる）** · 禁与瑆混写 |

**优先**：瑆 **ひかる** 为 IP 真源读音 · 光保留自然 **ひかる** · 靠 **汉字+姓** 区分 · 废止 瑆=**けい**（见 [`01_角色设定/00_十人名称_LOCK_2026-06-05.md`](../01_角色设定/00_十人名称_LOCK_2026-06-05.md)）

- 全角括号 · 读音用 **ひらがな**（外来语术语可用カタカナ）  
- 序 + 每案标题下 **人名表** 可选（E04 汇编卷首）  
- 正本附 `读音索引_VolN.md`（首次出现页码 · 供印前）
