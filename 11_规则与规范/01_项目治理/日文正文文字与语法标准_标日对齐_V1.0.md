## 1. 总则

**日文正文只允许「标日体系」内出现的书写与语法。**

| 允许 | 禁止 |
|------|------|
| **平假名**（ひらがな） | 简体中文 · 繁体（非日本通用表记）· 韩文 · 其他文字 |
| **片假名**（カタカナ） | 罗马字裸写（除 LOCK 表记如 A001 等案号 **源文件 meta**） |
| **日本汉字**（常用漢字表 + 人名 LOCK 用字） | 中国简化独用字 · 中国未传入日本的 campus 词汉字组合 |
| **标日语法**（標準語 · 共通語） | 中文句法直译 · 中国校园叫法 · 非日本儿童读物口语 |

**「标日」定义（本项目）**：以《新版标准日本语》及日本小学 **国語・標準語** 为语法基准；児童文学叙事可用 **だ・である調**（业界惯例），对话可用 **です・ます／だ** 按角色，但 **禁中式书面腔**。

## 2. 允许字符集（PRODUCT 正文）

### 2.1 必用三系

```
ひらがな ＋ カタカナ ＋ 日本漢字
```

- 人名汉字：仅 [`00_十人名称_LOCK_2026-06-05.md`](../01_角色设定/00_十人名称_LOCK_2026-06-05.md)  
- 科学·校园词：优先 `03_LANGUAGE` · `JP_PROSE_LEXICON` · 田中词库  
- 外来语：**片假名**（例：テープ · フォルダ · チャイム）· 禁英文裸写

### 2.2 允许标点·符号（日本排版）

| 允许 | 用途 |
|------|------|
| 「」『』 | 对话 · 引用 |
| 。、？！… | 句读 |
| ・ | 并列（**禁** 半角 `·` 作日文中点） |
| （） | 括号读音标注 · 补充 |
| ー | 长音 |
| ～ | 约数（ sparingly） |
| 0–9 | 数字 · 时间 · 班级（5年2組） |

### 2.3 源文件 meta 与 PRODUCT 正文分界

| 区域 | 允许 |
|------|------|
| `---` 以上 · 【语感编辑】 | 中文说明 · 中文【】标签 · Agent 路径（不进书） |
| **PRODUCT 日文叙事正文** | **仅 §2.1–2.2** · 禁 Markdown `**` `` ` `` · 禁中文 |

## 3. 禁止项（一票否决）

### 3.1 文字·标点

| 禁 | 改用 |
|----|------|
| 半角 `·` 并列 | **・**（全角中黒） |
| 中文破折号 `——` 叙事 | **……** 或改句读；对话中断可用 **……** |
| Markdown `**強調**` | 削除或改 **傍点**（出版）· 源稿 **不写** |
| 反引号 `` ` `` | 削除 |
| 中文《》 | 日本『』或不加书名号 |
| 简体/中国独用词 | 见 J1 · `03_LANGUAGE` §6 |

### 3.2 语法·叫法（标日外 / 中式）

| 禁（中式直译） | 标日/日本小学 |
|----------------|---------------|
| 报告老师 | 先生、これ見てください |
| 食堂 · 打饭 | 給食 · 配膳 |
| 班长 | 学級委員 · 係 |
| 推开教室门 | 引き戸を開ける |
| 你太棒了 | 頑張ったね |
| 首先…其次…最后… | 禁作文连接词堆砌 |
| 的 · 地 · 得 结构直译 | 日语修饰结构 |
| 进行～ / ～化 | 动词自然形 |
| 放課後＝放学后（误） | **放課**＝课间 · 放学后＝**放課後** |

完整表：`docs/world_reference/03_LANGUAGE_SOCIAL_FOOD_STANDARD.md` §6 · `JP_PROSE_LEXICON.md`

### 3.3 叙事文体

| 项 | 标准 |
|----|------|
| 语体 | **標準語** · 名古屋弁 ≤2 句/卷 · 陸珣 **不说** 方言 |
| 叙事 | 第三人称 · 过去形（…した / …だった） |
| 对话 | 口语自然 · 敬语分级见 03_LANGUAGE §1 |
| 儿童科学 | 小学理科用语（結露 · 熱伝導 等） |
