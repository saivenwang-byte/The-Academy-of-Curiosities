## 0. 三层验收架构


## 3. 卷内必检模板

### 3.1 三原理交叉段（正文必有一处）

Case Card 填写后，正文中须出现 **类似结构**（可改措辞，不可缺原理名）：

```text
慧美把三个词写在纸上，画三个圈，让它们交叠：
  · 结露 — 暖湿空气碰到更冷的表面，水蒸气变水珠
  · 热传导 — 窗框的冷沿椅背传到椅面
  · 湿度/露点 — 夜里空气够湿，椅面够冷，才会出水
三圈交叠的地方，就是光的椅子。
```

验收：grep 或 scorecard 字段 `principles_named_in_body: [结露, 热传导, 露点]`

### 3.2 误导段（正文必有一处）

```text
错误假设（谁提出）→ 验证动作 → 失败证据 → 转向下一假设
```

Vol1 范例：志郎「恶作剧泼水」→ 搜查/地面/邻座 → 不成立 → 陸珣/慧美转向结露链

## 4. 加权计分公式

```
原始分 = Σ(各项得分)                    # 最高 240
加权分 = Σ(得分 × 该项权重系数)         # 最高约 276（含加权）
归一化 = 加权分 / 276 × 240           # 映射回 240 尺度便于对比
```

**定稿门槛（L2）**：

| 等级 | 归一化分 | 含义 |
|------|----------|------|
| A | ≥200 | 可 READY_FOR_SAMPLE |
| A- | ≥180 | 可 READY，建议小修 |
| B+ | 160–179 | 定向二轮 |
| B | <160 | 结构或文化大修 |

> 实践简化：**原始分 ≥180 且 3.3≥6、3.6≥7、无一票否决** → 可标 READY_FOR_SAMPLE

## 7. 文学语感维度（Hybrid Voice · L2 高权重）

> Vol1 决策：`03_故事内容/第1卷_总是湿的椅子/语感决策记录_Vol1.md`

| ID | 二级维度 | 10 分标准 | 权重 |
|----|----------|-----------|------|
| 7.1 | 留白率 | 哲思/情绪不解释；金句型内心 ≥3/卷 | ×1.5 |
| 7.2 | 句长控制 | 叙事段均句 ≤22 字；无连续 3 句说明文 | ×1.0 |
| 7.3 | 陸珣声线 | 对外台词 ≤5 句；内心短、冷、准 | ×1.5 |
| 7.4 | 说明文比例 | 制度/科普解释句 ≤全文 8% | ×1.0 |
| 7.5 | 回味收束 | 结尾 1 意象 + 1 哲思；无说教 | ×1.5 |
| 7.6 | 词汇/动作去重 L12 | 同卷 habit/标志动作 ≤2 次/案；无英编辑残留 | ×1.0 |
| 7.7 | 感官入口变奏 L13 | 相邻案发现段主感官不同；禁同路径开场 | ×1.0 |

| 7.9 | 情节模板变奏 L15 | 禁连续案同一调查顺序（志郎误判→三人看→校工→鞋柜） | ×1.2 |
