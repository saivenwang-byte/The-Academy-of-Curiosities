## 2. 情感分寸（日式）

| 场景 | ✅ | ❌ |
|------|-----|-----|
| 鼓励 | **頑張ったね** | 你太棒了 / 太厉害了 |
| 友谊 | 同走一段 → **じゃあね** | 你是我最好的朋友 |
| 安慰 | 同坐沉默 | 别难过长篇 |
| 被拒绝 | **そっか**，转身离开 | 质问、追问 |
| 和解 | 第二天递纸条「昨日、ごめん」+ 默默坐回来 | 当面长篇道歉 |
| 请求 | **やらせてもらってもいい？** | 自我主张无许可 |

## 3. 名古屋弁

- 标准语为主 · 每卷 ≤2 句点缀  
- **放課（ほうか）= 课间**，≠ 放学后  
- **えらい**（名古屋）= 累/痛，≠ 伟大  
- 可使用：だわ / だわね、しゃーない  
- 陸珣 **不说** 名古屋弁/关西弁  
- 长辈专属：葛西泉蔵可使用上町ことば

## 5. 給食 · 饮食

- **教室配膳**（非食堂打饭）  
- 顺序：割烹着当番 → ワゴン → 主食→副菜→**牛乳** → いただきます  
- 名古屋：きしめん、赤だし、カレー日  
- 线索：气味/洒落/计数 → 见 maintenance/04

| 菜品 | 出现频次 | 卷目应用 |
|------|----------|----------|
| きしめん | 月1–2次 | 日常描写 |
| 味噌カツ | 月1次 | 特別日 |
| 手羽先風味唐揚げ | 偶尔 | 特別日 |
| カレーライス | 高人气 | 团队庆祝日可用 |
| 揚げパン | 怀旧之王 | 光的最爱 |
| 牛乳 | 每日1本200ml | 日常描写 |
