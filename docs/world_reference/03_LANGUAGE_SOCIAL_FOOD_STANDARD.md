## 1. 称呼规则速查

| 关系 | 正确称呼 | 禁止称呼 |
|---|---|---|
| 对学长 | ○○先輩 / ○○さん | 直呼其名 |
| 对同学 | 姓氏 + さん / くん | 未建立关系前无敬称呼び捨て |
| 对老师 | ○○先生 | 姓氏后加非学校语境后缀 |
| 对妹妹 | 下の名前 + ちゃん | 过度成人化称呼 |
| 转学生初来 | 姓氏 + さん，一周后可能改变 | 初见即昵称 |

## 2. 友谊与情感表达分寸

| 场景 | 日式表达 | 禁止表达 |
|---|---|---|
| 友谊宣告 | 放学后一起走一段路，自然分开 | “你是我最好的朋友！” |
| 鼓励 | 「頑張ったね」 | “你太厉害了！” |
| 安慰 | 安静坐在旁边，不说话 | “别难过，没什么大不了的” |
| 被拒绝 | 「そっか」，转身离开 | 质问、追问 |
| 和解 | 第二天递纸条「昨日、ごめん」+ 对方默默坐回来 | 当面长篇道歉 |

## 3. 名古屋弁使用规则

| 项目 | 标准 |
|---|---|
| 使用频率 | 每卷≤2句 |
| 可使用 | だわ / だわね、しゃーない、えらい、放課 = 休み時間 |
| 禁止 | 主角陸珣说名古屋弁，转学生语言必须保持一致性 |
| 长辈专属 | 葛西泉蔵可使用上町ことば |

## 4. オノマトペ分配

| 角色 | 拟声拟态倾向 |
|---|---|
| 伊藤光 | 多感叹、多重复：「えー、じゃあさー」 |
| 加藤慧美 | 逻辑句 + 偶尔拟态语，偏冷静 |
| 松本志郎 | 短句 + 动作音：「よし、やるぞ」「バチッ！」 |
| 陸珣 | 少话，名词 + 指示：「…あそこ。」 |
| 陸瑆笔记 | 可更书面，但保留童感 |
