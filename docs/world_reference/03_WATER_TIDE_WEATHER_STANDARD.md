## 1. 结露机制 `[VERIFIED_SOURCE]` — 卷1核心

```
暖湿室内空气 → 接触 T < 露点 的表面 → 水蒸气液化
```

| 因子 | 卷1应用 |
|------|---------|
| 4月夜冷日暖 | 窗框/铝框夜间冷却 |
| 靠窗单椅触窗框 | **传导冷却** → 椅面结露 |
| 邻椅不触窗框 | 温度&gt;露点 → 不湿 |
| 梅雨 | 湿度↑ → 结露更易（卷3背景） |

**陸珣句**：「その椅子、窓枠に当たってる。」  
**实验**：玻璃杯+冰+纸巾 `[CONSULTANT_CONFIRMED]`  
**进阶实验**：**セロハンテープ** 贴杯壁对比有无结露 `[VERIFIED_SOURCE]`（`09_/12_L类`）

## 3. 梅雨证据 `[SEASONAL_VARIATION]`

- 入梅 ~6/6 · 出梅 ~7/19 · 降水250–350mm  
- じめじめ、 mold 味、外遊び禁止 → 室内将棋/トランプ/読書激增  
- 教室 **除湿剤/除湿桶** refill；保健室防霉换气  
- 雨天 **プール** 若水温达标仍可实施 `[LOCAL_VARIATION]`  
- **替え靴下** 持参为常见校推荐  
- **可**：湿度致 **扉膨胀** 难开/易开 · **墨水滲** · 机缝 · 蚂蚁路径  
- **不可**：无科学闭环的「闹鬼湿痕」
