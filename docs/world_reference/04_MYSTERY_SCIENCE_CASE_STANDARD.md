## 1. 本格推理基本规则

《学堂趣事录》的每一卷必须满足：

- 核心异常来自日常生活，不来自超自然。
- 关键线索必须提前出现。
- 儿童读者有机会根据线索猜出真相。
- 科学机制必须可验证、可复现或可观察。
- 大人可以引导，但不能代替孩子解决谜题。
- 解释不能依赖作者临时补充的隐藏信息。
- 日本校园文化和名古屋环境必须支持该谜题成立。

## 2. 卷1核心机制：結露

### 科学链

室内暖湿空气 → 接触低于露点的表面 → 水蒸气凝结

铝合金窗框夜间冷却 → 早晨温度最低

靠窗单椅接触窗框 → 传导冷却 → 椅面结露

旁边椅子不接触窗框 → 温度高于露点 → 不结露

### 陸珣台词物理依据

「その椅子、窓枠に当たってる。」

中文含义：那把椅子，挨着窗框。

### 可复现实验

材料：玻璃杯、冰块、纸巾

步骤：

1. 室温杯外壁无明显水珠。
2. 加入冰块。
3. 杯壁变冷。
4. 空气中的水蒸气在杯壁外侧凝结。
5. 用纸巾擦拭，对比擦与不擦的痕迹。

原理：温度降至露点以下 → 水蒸气凝结。

## 4. 科学线索使用规则

### 4.1 可作为线索的类型

- 水迹
- 气味
- 光影
- 声音
- 温度
- 湿度
- 风向
- 物体位置
- 材质差异
- 植物或昆虫状态
- 语言误会或谐音
- 校园制度造成的行为轨迹

### 4.2 不能作为核心解释的内容

- 超自然现象
- 纯巧合
- 梦境
- 大人事后解释
- 主角突然知道读者不知道的信息
- 不可验证的心理猜测
- 违反日本校园制度的行为
- 违反名古屋气候与季节的现象
