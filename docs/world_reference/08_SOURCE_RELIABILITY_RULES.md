# 08 · Source Reliability Rules · 来源可靠性规则

> **Status**: ACTIVE  
> **Rule**: 无标签的事实 **不得** 进入定稿正文

---

## 1. 六级标签

| 标签 | 含义 | 定稿可用？ |
|------|------|------------|
| **VERIFIED_SOURCE** | 官方/気象庁/文科省/名古屋市等可核对来源 | ✅ |
| **CONSULTANT_CONFIRMED** | 田中みどり五维或人工顾问确认 | ✅ |
| **LOCAL_VARIATION** | 因学校/地区而异，正文须一句交代 | ✅ 带限定 |
| **SEASONAL_VARIATION** | 因月份/年变而异，须写月份 | ✅ 带月份 |
| **NEEDS_VERIFICATION** | 有用但未核实 | ❌ 仅策划 |
| **DO_NOT_USE** | 文化错误/伪科学/恐怖/中式校园 | ❌ |

---

## 2. 禁止来源

- 「我觉得日本学校大概是这样」  
- 影视剧/动漫/AI 生图视觉  
- 中国校园经验直译  
- 无出处的数值（气温、影长、潮汐时刻）

---

## 3. 数值类硬要求

| 类型 | 必须标注 |
|------|----------|
| 气温/湿度 | 地点（名古屋47636）+ 月份 |
| 日出日落 | 纬度 35°N + 日期 |
| 影长 | 物高 + 太阳高度角 + 公式 |
| 潮汐 | 港口 + 日期 + 大潮/小潮 |
| 物候 | 名古屋 + 年变幅度 |

## 5. 冲突处理

1. 两来源冲突 → 标 NEEDS_VERIFICATION，暂停用于定稿  
2. LOCAL vs 名古屋默认 → 以 **名古屋小学校** 为准  
3. 顾问 HTML 红色 → 等同 DO_NOT_USE 直至修改
