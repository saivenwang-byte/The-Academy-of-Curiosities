# 创作红线与禁止项清单 V0.1

Status: ACTIVE
Applies to: all story, dialogue, scene, visual, and market materials

---

## 1. 总红线

《学堂趣事录》面向日本 6–14 岁读者，核心是日常谜题、知识推理、伙伴喜剧与儿童可验证实验。

禁止把故事写成恐怖、暴力、政治、超自然、成人化校园冲突或中国校园直译。

---

## 2. 禁止项总表

| 类别 | 禁止内容 | 替代方案 |
|---|---|---|
| 政治 | 1949年后中国政治 | 1949年前历史可作为远背景，仍需谨慎 |
| 校园 | 霸凌、いじめ、暴力 | 误解后和解、社交尴尬、轻微误会 |
| 恐怖 | 鬼魂、血腥、生命威胁 | 科学解释所有“ふしぎ” |
| 超自然 | 穿越、超能力、魔法 | 纯本格推理与可验证科学 |
| 称呼 | 中式“报告老师” | 「先生、これ見てください」 |
| 友谊 | “你是我最好的朋友” | 放学后同走一段路，在路口说「じゃあね」 |
| 鼓励 | “太厉害了” | 「頑張ったね」 |
| 饮食 | 食堂打饭 | 給食当番在教室配膳 |
| 校舍 | 旋转门、普通推门 | 引き戸 |
| 天气 | 名古屋大雪 | 冬季干冷、偶尔小雪但不写常规积雪校园 |
| 台风 | 灾难性破坏 | 台风后的残留现象 |

## 3. 六项核对流程

每一卷、每一场景、每一句关键对话必须核对：

1. 气候核对：月份、气温、湿度、日照是否符合名古屋指标。
2. 空间核对：场景是否符合校舍位置与动线，如引き戸、片廊下、窓側、廊下側。
3. 行为核对：人物称呼、友谊表达、情绪反应是否符合日式分寸。
4. 语言核对：是否混入中国式校园口语，是否过度使用方言。
5. 科学核对：核心原理是否可验证，是否与已用原理重复。
6. 红线核对：是否触及政治、霸凌、恐怖、超自然四大禁区。
