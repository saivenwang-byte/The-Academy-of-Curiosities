## 2. 校庭 · 通学路自然 `[LOCAL_VARIATION]`

- 桜（1–2株）、つつじ、サツキ  
- **花壇轮换**：冬 pansy → 夏 marigold → 秋 chrysanthemum  
- 紫阳花：梅雨前后 **最可信** 自然线索  
- 银杏：行道/校内大树 → 11月果/叶 · **ジョロウグモ** 大蛛网（11月，非恐怖）  
- 通学路：**メダカ**（堀川浅段）、夏 **ザリガニ** · 空き地 **バッパ/カマキリ/カブト**  
- **ホタル**：市内罕见；瀬戸/尾張旭等郊外仍可  
- 公园 **砂場** 混猫迹 — 卫生张力，非 horror

## 5. 推理规则

写任何「花/虫/叶/气味」线索前：

1. 查上表 **月份** 是否成立  
2. 名古屋纬度下物候 `[SEASONAL_VARIATION]` 标注  
3. 儿童能否在校庭/通学路 **亲眼观察**  
4. 解释是否可复现（计数、对比、标本）  

**经典对**：
- 紫阳花 ↔ 梅雨 ✓  
- 蝉 ↔ 盛夏 ✓  
- 樱瓣 ↔ 4月 ✓  
- 银杏臭 ↔ 11月 ✓
