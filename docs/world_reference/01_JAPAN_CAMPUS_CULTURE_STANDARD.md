## 3. 制度与职务 `[VERIFIED_SOURCE]`

- **係体系**：黒板係 / 保健係 / 図書係 / 放送係 / 生き物係 等（非仅給食当番）  
- **給食当番**：白围裙+帽+口罩 → ワゴン → 配膳顺序 · **每周轮换** 每组约 4–6 人 `[LOCAL_VARIATION]`  
- **牛乳盖 campaign**：1000 caps → 疫苗捐赠（真实活动）`[VERIFIED_SOURCE]`  
- **掃除時間**：分区当番；椅子翻上桌、跪地推雑巾、**短柄箒** `[CONSULTANT_CONFIRMED]`  
  - ⚠ 时段因校而异：有的 **午饭后 15 分**，有的 **下课后** → 正文须写「今日の掃除当番」`[LOCAL_VARIATION]`  
- **日直 / 出欠**  
- **連絡帳**：每日签字  
- **学級会**：机下げ丸座；追求「みんなが納得」非对抗多数决 `[CONSULTANT_CONFIRMED]`

### 3.2 下校与托管 `[VERIFIED_SOURCE]`

- 1–2 年级 **集団下校**（东/西/南コース）；6 年级带 1 年级过马路  
- **児童クラブ** vs **学童**：托管约 14:00–18:00，规模上限约 70 人 `[LOCAL_VARIATION]`  
- 名古屋 326 校 · 給食 **自校式**（非中心厨房）`[VERIFIED_SOURCE]`  
- 普通教室 **2018 年起空调全覆盖**；体育馆仍常无 `[LOCAL_VARIATION]`

### 3.3 社交结构 `[CONSULTANT_CONFIRMED]`

- **スクールカースト**（一軍/二軍/三軍）自然形成；换班可重置  
- 3–4 年级起男女 **分群玩耍**；5–6 年级更明显  
- いじめ形态以 **无视** 为主（非肢体）— 本 IP 不写霸凌主线

## 6. 情感与冲突分寸 `[CONSULTANT_CONFIRMED]`

- 鼓励：**頑張ったね**（非「太棒了」）  
- 友谊：同走一段 → 路口 **じゃあね**（非「最好的朋友」）  
- 安慰：**同坐沉默**  
- 冲突：**无视/纸条道歉**（非打架霸凌）  
- 和解：「昨日、ごめん」纸条 + 默默坐回
