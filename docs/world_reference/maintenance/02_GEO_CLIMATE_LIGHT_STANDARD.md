## 4. 影长公式 `[VERIFIED_SOURCE]`

```
影长 = 物高 ÷ tan(太阳高度角)
```

| 高度角 | 5m 旗杆影 |
|--------|-----------|
| 78° (夏) | ~1.1m |
| 45° (秋) | ~5.0m |
| 32° (冬) | ~8.3m |

**写作铁律**：写「影子落到第X排」必须同时声明 **月份、时刻、杆高、窗/旗杆方位**。

## 5. 教室采光 `[LOCAL_VARIATION]`

- 典型：**南面教室 + 北廊下**（片廊下）  
- 4月早晨：斜射、**ひんやり**  
- 9–10月下午：低角光可投 **窗格影到黑板**（卷6）  
- 12–1月 15:00：已偏暗，须写天色  

**不能写**：「下午四点阳光正好照到第三排」而不查 **10月名古屋 + 南向窗 + 高度角**。

**逃げ水 vs 旗杆影** `[VERIFIED_SOURCE]`：路面虚像（夏）≠ 几何影（秋杆）— **机制不可混用**（卷7）
