## 2. 当番流程（顺序不可乱） `[VERIFIED_SOURCE]`

1. 白い割烹着 + 白帽子 + マスク  
2. 給食ワゴン从給食室推来  
3. 列队取餐：「お願いします」  
4. 顺序：主食 → 主菜 → 副菜 → **牛乳** → 水果/デザート  
5. 全班「いただきます」同时开始  
6. 食后：托盘分类、牛乳盒 **拆开压平**、歯磨き（若校实施）  
7. 当番擦桌、回收  

**线索用途**：计数少一份、汤咸、 spill 痕迹、当番围裙湿痕。
