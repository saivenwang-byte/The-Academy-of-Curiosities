## 3. 名古屋弁（点缀） `[LOCAL_VARIATION]`

- 全文 **标准语为主**；每卷 0–2 句方言即可  
- **放課（ほうか）= 课间**，≠ 放学后 — 中国读者易误 `[LOCAL_VARIATION]`  
- 例：しゃーない · **えらい**（名古屋=「累/痛」，≠标准语「伟大」）· そうやね  
- 儿童词：**ときんときん**（铅笔尖）、**机をつる**（搬桌）、**しゃびしゃび**（蔫）  
- 通学：**けった**=自転車 · **ドベ**=最下位 · **まわし**=仕度  
- ✗ 陸珣关西弁（除非设定改）  
- ✗ 方言当笑话本体  

### 3.1 许可与礼仪 `[CONSULTANT_CONFIRMED]`

- 请求：**やらせてもらってもいい？** 优于自我主张  
- 横断：**手を挙げて** 左右看再过  
- 信号：**ピヨピヨ/カッコー** 鸟叫提示音（视障设计）`[LOCAL_VARIATION]`

## 4. ダジャレ（谐音梗）规则 `[VERIFIED_SOURCE]`

### 硬规则

1. **中文谐音不能直接搬日文** `[DO_NOT_USE]`  
2. 须 **日语语音成立** + **儿童可理解**  
3. 每卷 **1–2 处**；勿连续堆砌  
4. 若谜题 **依赖谐音** → Case Card 须写清 **原文+读音+为什么孩子能想到**

### 可用例（课堂/操场）

| 梗 | 说明 |
|----|------|
| いか + が + どうしょう | 乌贼双关 |
| カエル飛び込んだ → 帰る | 操场游戏 |
| 低质量サッカー梗 | 光讲、慧美吐槽 ✓ |

### 课堂同音（慧美型谜题） `[VERIFIED_SOURCE]`

- **虹/二時、橋/箸、紙/神** — 须国語课语境  
- 3 年级起 **ことわざ**；4 年级 **ことわざ辞典** 作业

### 不可用

- 成人冷笑话 · 政治双关 · 需汉字学养的梗  
- 中文「谐音推理」直译 `[DO_NOT_USE]`
