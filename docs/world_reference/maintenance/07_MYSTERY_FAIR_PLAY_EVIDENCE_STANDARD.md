## 2. 线索可见性规则

### 必须

- 公平线索 **≥3 条**，正文可见  
- 误导线索 **可存在**，但须可解释为何误导  
- 核心机制依赖 **01–06 标准** 中可查事实  
- 结尾 **儿童可复现实验**（日常材料）

### 禁止 `[DO_NOT_USE]`

- 作者突然解释、读者无法回溯  
- 巧合一次性破案  
- **大人代解**（老师/家长/警察破核心谜）  
- 不可查文化信息（「日本人都这样」）  
- 伪科学、恐怖、玄学  
- 未在 Scene Card 登记的环境参数（光/水/物候）

## 6. 红鲱鱼规则

- 允许：怪谈谣言、同学误判、感官错觉  
- 须：中期可被 **同一证据体系** 排除  
- 禁止：红鲱鱼 **唯一** 指向超自然且未推翻

## 6.1 怪谈外壳规则 `[CONSULTANT_CONFIRMED]`

| 项 | 规则 |
|----|------|
| 传播场 | 放学后教室、学童、补习班；「6 年生说的」增权威 |
| 同伴压力 | 「不试=胆小」— 光 vs 慧美分歧 ✓；非霸凌 |
| 结构 | **条件化安全**（「不去第 X 间就没事」）→ 公平可控 |
| 昭和七不思議 | 人体模型=热胀/线轴；13 阶=数错/光学 — 须科学替换 |
| 令和新怪谈 | Wi-Fi 定时重启、监控反光/延迟 — `[LOCAL_VARIATION]` |
| 葛西框架 | 「怪谈=解释できないの仮名」→ 科学解谜 |
| **禁止外壳** | 口裂女/テケテケ/八尺様 → `[DO_NOT_USE]` |

### 6.2 天然 Case 嵌入点 `[CONSULTANT_CONFIRMED]`

自由研究 · 写生大会 · 遠足手册 · 理科借器材（须先生许可）
