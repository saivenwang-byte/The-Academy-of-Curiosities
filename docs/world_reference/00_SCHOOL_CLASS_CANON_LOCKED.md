## 锁定结论

```
主人公团队教室 = 4年2組 · 窗=校庭
```

| 项 | 锁定值 | 禁止 |
|----|--------|------|
| **学年** | **4年** | ❌ 5年2組 · ❌ 五年级 |
| **班级** | **2組** | ❌ 1組 / 3組（除非写别班路人） |

| **座席** | 1–2 周一换；Vol1 光=窓際三列目 | — |

**本锁定取代**：旧 docx / 早期对话稿中一切「5年2組」「五年级」表述。

## 为何是 4年2組（非 5年2組）

| 维度 | 4年2組 ✅ | 5年2組 ❌ |
|------|-----------|-----------|
| 现行正文 | v1.2 日/中文稿、Case Card、Scene Card 已统一 | 仅 legacy docx |
| 陸瑆 | **4年1組别班**，与珣同届异班，通学路/路口汇合 | 若5年则整库迁移 |
| 50卷时间线 | Vol1 起4年 → Vol10 仍可在6年内成长 | Vol1 起5年 → Vol11 理紗（5年）加入时主角已6年面临毕业 |
| 团队弧 | 「组建期」Vol1–10 从10岁到12岁 | 5年起跳压缩探索团童年窗口 |
| 视觉 brief | `14_角色设计稿创作简报_v2.md` 锁 4年 · 私服 | v1 docx 5年已作废 |
| 读者层 | 6–14 岁；4年≈10岁为系列基准年龄 | 5年≈11岁仍可读，但与全库迁移成本不成比例 |

**主编决策**：维持 **4年2組**，不采用 5年2組。
