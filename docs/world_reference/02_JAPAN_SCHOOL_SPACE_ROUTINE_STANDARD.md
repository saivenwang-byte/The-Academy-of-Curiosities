# 日本校园空间·建筑·通学路·行为标准 V0.1

Status: ACTIVE  
Applies to: all school scenes

---

## 1. 校舍基准参数

| 项目 | 标准指标 | 写作规则 |
|---|---|---|
| 建筑结构 | RC造，钢筋混凝土，3–4层 | 现代日本公立学校基准，不写成中国式书院 |
| 布局类型 | 片廊下式，单侧走廊 | 教室沿走廊排列，窗侧通常面向校庭或中庭 |
| 教室面积 | 约9m × 7m = 63㎡ | 影响桌椅排列、声音传播、光影距离 |
| 走廊宽度 | 约2m | 影响追逐、擦肩、观察视线 |
| 黑板尺寸 | 约3.6m × 1.2m，深绿色 | 适合窗格投影、粉笔印、板擦线索 |
| 课桌尺寸 | 约60cm × 40cm，钢管 + 合板 | 影响水迹、摩擦、滚动、遮挡 |
| 教室门 | 引き戸 | 禁止默认写成推门或旋转门 |

### 教室写作铁律

- 教室门必须默认为 `引き戸`，除非明确设定特殊建筑。
- 不写“推开教室门”，优先写“拉开教室的引き戸”。
- 窗户为推拉式，上半可开；4月仍凉。
- 靠窗列为 `窓側`，常是理想位置。
- 靠走廊列为 `廊下側`，相对嘈杂、视野差。

| 项目 | 指标 | 写作应用 |
|---|---|---|
| 通学距离 | 徒步15–25分钟 | 放学后活动时间计算 |
| スクールゾーン | 学校周边约500m | 车辆限行、安全描写 |
| 集団登校 | 低年级为主，高年级可担任班長 | 角色责任与照顾低年级 |

## 5. 行为规范硬规则

- 进入学校必须考虑：昇降口、下駄箱、上履き。
- 普通小学校午餐优先写教室給食，不写中国式食堂打饭。
- 小学生放学后不能无理由留校。
- 若三人留下来，必须指定理由：係活動、掃除当番、図書当番、先生许可、放課後子ども教室等。
- 中学校可使用部活、先輩 / 後輩、生徒会等机制；小学校不得随意套用中学校部活氛围。
- 任何学校空间线索必须考虑学生是否被允许进入该地点。
