# 日文版翻译 · 田中みどり全文督查门

> **Status**: ACTIVE · **Priority**: HIGHEST（与 L1 并列，**仅日文版**）  
> **原则**：中文定稿 ≠ 可出版日文。翻译完成后 **必须** 走本门禁，不可跳过。

---

## 0. 核心规则

| 阶段 | 语言 | 督查 |
|------|------|------|
| 中文正文定稿 | 中文 | L1 三关 + L2 评分卡（已有） |
| **日文版翻译** | **日文** | **本门禁 · 田中みどり全文督查** |
| 对外发布 / 送画 / 送印 | 日文为主 | 两语版本状态均为 `READY_FOR_*` |

**禁止**：仅校对语法、仅母语者通读、跳过顾问工具，即标 `READY_FOR_TRANSLATION`。

---

## 1. 文件规范

每卷日文版与中文版 **同目录并存**：

```

  完整文字稿.txt              ← 中文定稿（L4）
  完整文字稿_日本語.txt        ← 日文翻译稿（本门禁对象）
  文化校准报告_初稿.txt        ← 中文 L1 报告
  文化校准报告_日本語.txt      ← 日文全文督查报告（必产出）
  卷任务包_总策划.txt
```

## 2. 强制流程（每卷 · 按序）

### 步骤 1 · 翻译前

- [ ] 中文版已 `READY_FOR_SAMPLE` 或已定稿入库  
- [ ] Case Card / Scene Card / 人名定稿无变更  
- [ ] 阅读 `02_/日本校园文化顾问_田中みどり.txt` §校园词汇库  

### 步骤 2 · 翻译

- 人名：沿用 `00_项目总览/人物名称定稿.txt` 表记（漢字 + 読み）  
- 制度词：优先 `03_LANGUAGE_SOCIAL_FOOD_STANDARD.md` / maintenance/06 表记  
- 对话：日本小学口语自然度 > 中文直译对称  
- **保留**正文关键日语台词（如「その椅子、窓枠に当たってる。」）— 与日文版一致或微调注释  

### 步骤 3 · 田中みどり **全文**督查（必做）

**工具**：`japan_campus_consultant_agent.html`

| 项 | 要求 |
|----|------|
| 范围 | **全文**（正文 + **陸瑆ノート** + 实验说明），不得只抽检段落 |
| 维度 | **五维全选**：空间与物 · 制度与流程 · 人际隐规则 · 儿童亚文化 · 情感分寸 |
| 分批 | 单次 textarea 有长度限制时，**按章拆分**，每章一次校准，汇总为一份报告 |
| 红色 ⚠ | **0 实质红项** 方可过门（误触「进教室」等须人工标注） |
| 产出 | `文化校准报告_日本語.txt` |

### 步骤 4 · 日文专属加检（翻译层）

| # | 检查 | 说明 |
|---|------|------|
| J1 | 直译腔 | 无中式校园日语（报告老师、食堂、班长） |
| J2 | 敬语分级 | 先生 / さん·くん / 先輩 与年级关系一致 |
| J3 | オノマトペ | 自然、不过量；角色分配见 03 标准 |
| J4 | 名古屋弁 | ≤2 句/卷；陸珣 **不说** 方言 |
| J5 | 科学术语 | 結露・露点・熱伝導 与理科小学用语一致 |
| J6 | 儿童可读 | 7–12 岁日语读者可独立阅读（LLM 或母语 beta 读） |
| J7 | 双层叙事 | **陸瑆ノート**语气：克制、第一人称、非正文复述 |
| J8 | v1.2 深修 | 见 `docs/JP_PROSE_LEXICON.md` · `vol01_jp_deep_revision_v1.2.md`（时间线/载玻片/实验安全/窓側金属/同音人名） |

### 步骤 5 · 状态升级

全部通过后，Case Card / 卷任务包更新：

```text
READY_FOR_TRANSLATION  →  日文全文督查 PASSED
```

并写入 `docs/world_reference/maintenance/09_CONSULTANT_REVIEW_LOG.md` §日文版记录。
