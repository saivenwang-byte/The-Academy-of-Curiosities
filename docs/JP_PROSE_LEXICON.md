## 禁词 / 必换

| 禁（中文直译/错） | 用 |
|-------------------|-----|
| 載玻片 | **スライドガラス**（初出：スライドガラス（理科の小さなガラス板）） |
| ガラス片（实验页裸写） | 金属スプーン・プラスチック板；ガラスは**大人と一緒** |
| 一点五、 | 正文章节号禁用；用 二、三、… |
| 三時限目 | **3時間目** |
| 予備鈴 | **朝のチャイム** / チャイムが鳴った |
| イス面 | **座面** |
| イス背もたれ | **イスの背もたれ**（机制改为窓側金属时优先 **窓側の金属パイプ**） |
| 何個か（名札） | **いくつか** |
| 校ズボン | **ズボン** / 学校に履いていったズボン |
| 長い放課後の十分間 | **長い休み時間** |
| 陸ひかる（旁白） | **陸瑆**；笔记标题 **陸瑆ノート** |
