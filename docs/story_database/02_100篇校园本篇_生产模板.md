## 0. 单篇内容机器（每篇必填）

```
校园趣事（可采访/可投稿）
    → 观察社为写稿去采访·观察
    → 日常异常（零恐怖）
    → ≥3 公平线索（正文可见）
    → 1 误判
    → 跨学科还原（≥2 原理，本篇期；≥3 原理，Vol11+）
    → 温柔人际真相（情感锤）
    → 陸瑆笔记（800–1200 字 · 全卷 1 篇或按卷 1 篇 · 见卷任务包）
    → 可选：家庭实验 1 页
```

| `lead_char` | 主导角色 | 慧美·志郎·光 |
| `science_core` | 核心原理 | 湿度+胶带+风侧 |
| `science_cross` | 交叉原理 | 黏着；空调风道 |
| `fair_clues` | ≥3 · 分号分隔 | 翘边侧=风侧;… |
| `misread` | 第一误判 | 有人恶作剧 |
| `human_truth` | 人际真相 | 投稿人怕稿被吹走 |
| `kei_note` | 瑆笔记角度 | 海报在打招呼 |

| `status` | 资产状态 | OUTLINE → … → LOCKED |

**情感锤自检（P1）**：`human_truth` 是否 **读者想转述**？`lead_char` 是否有 **可记忆决定瞬间**？
