《学堂趣事录》七种核心推理范式
================================

概述
----
50卷故事中，每一卷的谜题将通过以下七种推理范式中的一种或多种来破解。
七种范式覆盖了全部30+知识领域，确保每卷的解法在方法层面上也有新鲜感。


范式一：实验验证
----------------
涉及学科：物理、化学、生物

核心逻辑：提出假设 → 设计对照实验 → 观察结果 → 得出结论

示例（第1卷）：椅子上的水 → 假设是冷凝 → 铁片 vs 玻璃片对照实验 → 证实导热差异导致凝结

适用场景：谜题的可验证性高，存在可量化的变量


范式二：结构还原
----------------
涉及学科：力学、工程学、建筑学

核心逻辑：观察物体结构 → 还原设计意图 → 找出"为什么要这样造" → 发现异常

示例：为什么走廊这个窗户打不开？→ 拆解窗锁机构 → 发现是通风系统的平衡设计

适用场景：谜题与器械、建筑、装置的设计逻辑有关


范式三：感知分析
----------------
涉及学科：光学、声学、色彩理论

核心逻辑：发现视觉/听觉/色彩异常 → 分析物理原理 → 排除心理错觉 → 定位真因

示例：为什么这面墙的颜色看着"不对劲"？→ 比较色卡 → 发现是灯光色温造成的视觉偏差

适用场景：谜题最初表现为"感觉不对"而非"事实不对"


范式四：历史溯源
----------------
涉及学科：历史学、档案学、考古学

核心逻辑：当下谜题 → 追溯历史档案/旧物 → 找到三十年前的关联事件 → 解开因果链

示例：为什么旧校舍地下有一截废弃的水管？→ 查阅校史 → 发现三十年前的改建记录

适用场景：谜题的答案不在今天，在过去——葛西爷爷的记忆和旧物成为关键


范式五：微观领域专家
--------------------
涉及学科：昆虫学、动物行为学、植物学

核心逻辑：异常出现在特定领域 → 团队卡住 → 陆珣调用专业知识 → 给出唯一答案

示例：蚂蚁为什么排成奇怪的线？→ 陆珣认出是举尾蚁，解释信息素路径 → 谜题破解

适用场景：谜题的钥匙藏在需要深度专业知识的领域


范式六：人际逻辑
----------------
涉及学科：心理学、行为经济学、沟通学

核心逻辑：分析人的行为模式、心理动机、社交习惯 → 找到"为什么这个人会这样做"

示例：为什么每到周三，某人的行为就变了？→ 分析其家庭/社交作息 → 发现与外部事件相关

适用场景：谜题表面上是一个物理现象，但根源是人的行为


范式七：系统权衡
----------------
涉及学科：社会学、经济学、法律

核心逻辑：多因素无法同时最优 → 寻找最佳平衡点 → 发现矛盾是"取舍"的结果

示例：为什么学校食堂的菜单每年都一样？→ 分析预算、供应商、营养标准的三角关系

适用场景：谜题没有"唯一正确答案"，是所有约束条件下的最优解


范式与角色能力映射
------------------
范式一（实验验证）  → 加藤慧美（记录整理·证据校准·对照实验）主导
范式二（结构还原）  → 松本志郎（行动派）主导
范式三（感知分析）  → 山本理紗（感知者，第11卷起）主导
范式四（历史溯源）  → 中谷琦（现场照片）或葛西爷爷（口述记忆）提供线索
范式五（微观领域）  → 陆珣（微观猎人）主导
范式六（人际逻辑）  → 伊藤光（发现异常·串联）+ 陸瑆（笔记层·人际逆推·灵感）※正文不抢戏
范式七（系统权衡）  → 加藤慧美（整理可验证问题）+ 伊藤光（串联多方）协作


范式使用分布规划
----------------
第一阶段（1-10卷）：范式一二五为主（建立基础推理习惯）
第二阶段（11-30卷）：加入范式三六（增加感知和人际维度）
第三阶段（31-50卷）：加入范式四七（引入时间维度和系统思维）
