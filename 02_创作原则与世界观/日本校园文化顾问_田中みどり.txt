《学堂趣事录》日本校园文化顾问 · 田中みどり
================================================

角色定位
--------
田中みどり（たなか みどり）是本项目的**首要文化顾问**。
所有正文、故事卡、实验说明、角色台词——凡涉及日本小学场景的文本，
在定稿前**必须**经过她的五层感知校准。

知识库（写作前必读 · 与红线同级）：
  02_创作原则与世界观/日本背景知识库_推理写作必读.txt
  含：一日时间表、校舍动线、教室结构、給食流程、名古屋气候、
      中国作家高频错误对照表、推理场景自检清单

交互工具（可视化校准面板）：
  项目根目录 / japan_campus_consultant_agent.html
  用法：粘贴段落 → 选择校准维度 →「开始文化校准」→ 按报告修改


强制工作流（写入创作流程）
--------------------------
【中文正文】
1. 初稿完成
2. 粘贴至 japan_campus_consultant_agent.html 做文化校准（五维全选）
3. 红色警告（⚠）必须修改；黄色/绿色建议择优采纳
4. AI 或人工改写后，再次校准，直至无红色警告
5. 方可进入正典文件夹（03_故事内容/）中文定稿

【日文版 · 翻译后必做 · 不可省略】
6. 生成 `完整文字稿_日本語.txt`
7. 对日文全文再次运行 japan_campus_consultant_agent.html（五维全选，按章分批直至覆盖 100%）
8. 完成 J1–J7 日文加检（见 docs/00_JP_TRANSLATION_REVIEW_GATE.md）
9. 产出 `文化校准报告_日本語.txt`，归档至 09_CONSULTANT_REVIEW_LOG
10. 方可标 READY_FOR_TRANSLATION 或对外送日文样章

※ 中文 L1 已过，仍须日文全文督查。翻译不是终点，顾问督查才是日文发布前最后一关。

本流程与「创作红线与原则.txt」第四条文化红线并列，不可跳过。


五层感知系统
------------
写任何日本校园场景前，对照以下五层自检：

【第1层 · 制度与流程的体感】
  开学式 · 毕业式 · 给食当番 · 部活/放課後 · 学級会 · 係活動
  → 写错了，骨架就断了。

【第2层 · 空间与物的记忆】
  上履き的气味与触感 · 引き戸（推拉门）的漏风方向 · 教壇位置
  · トイレスリッパ（厕所专用拖鞋）· 黒板消し的放置位置
  → 这些细节查资料查不出来，靠「在场感」。

【第3层 · 人际关系与隐性规则】
  先輩/後輩 · 空気を読む · いじめ的真实形态（集体无视，非肢体暴力）
  · スクールカースト（社交等级，与成绩无关）
  → 与本 IP「无霸凌红线」配合：可写传言/误会，不可写系统性恶意欺凌。

【第4层 · 儿童文化与亚文化】
  ランドセル的分量与磨损 · 夏休みの自由研究 · 学校の怪談经典类型
  · 女孩书包钥匙扣 · 男孩侧袋藏玩具
  → 怪谈素材可做「被科学揭穿」的谜面包装，不得真恐怖。

【第5层 · 情感表达与分寸感】
  克制型友谊（走一段路，路口じゃあね）· 頑張ったね（非「你真棒」）
  · 安慰是默默坐在旁边 · 道歉与原谅的仪式感
  → 与本 IP「结尾干净利落」原则一致。


校园词汇库（12 核心词）
-----------------------
① 上履き（うわばき）Uwabaki — 第2层
   室内鞋。玄关换上，编号鞋柜存放。忘换走进教室是「错误」，同学会提醒。

② 給食当番（きゅうしょくとうばん）— 第1层
   给食值日。白大褂+帽子+口罩打饭，全班昼歯磨き。

③ 係活動（かかりかつどう）— 第1层
   全员班级职责（黒板係、保健係、配膳係等），比中国班干部更平等。

④ 部活（ぶかつ）— 第1层
   社团活动。放学后进行，秋冬结束时天已黑。先輩/後輩关系严格。
   ※ 本项目主角为小学四—六年级；小学部活较轻，勿写成中学部活强度。

⑤ ランドセル — 第4层
   小学生双肩皮书包。传统男黑女红，现代多样。一年级新，六年级旧。

⑥ 空気を読む — 第3层
   读空气。何时说话、沉默、附和——决定孩子在班里的位置。

⑦ 先輩/後輩 — 第3层
   纵向等级。敬语、收器材。主要在部活/高年级互动中体现。

⑧ いじめ — 第3层
   欺凌最常见形态：集体「无视」。本 IP 红线：不做霸凌剧情，可做传言误会。

⑨ 自由研究（じゆうけんきゅう）— 第4层
   暑假自主研究，开学展示。与本 IP 精神高度契合，可作卷外延伸。

⑩ 学校の怪談 — 第4层
   钢琴自弹、人体模型、生物教室夜声——可做谜面，必须科学揭穿。

⑪ じゃあね — 第5层
   「那么明天见」。路口克制告别，衡量友谊的分量。

⑫ 頑張ったね — 第5层
   「你努力了」。强调过程，非结果导向的「你真棒」。


文化校准规则库（高频触发项）
----------------------------

【红色警告 · 必须修改】

| 触发词/场景 | 问题 | 修改方向 |
|-------------|------|----------|
| 推开/推开教室门 | 中国铰链门语感 | 改为「拉开引き戸」，写轨道滑动声「カタッ」 |
| 直接走进教室（无换鞋） | 缺上履き制度 | 先写玄关（げんかん）换室内鞋，鞋柜编号 |
| 厕所场景无拖鞋 | 缺トイレスリッパ | 进厕换拖鞋，出来忘换是常见尴尬 |
| 你真棒/太厉害了 | 中式结果鼓励 | 改为「頑張ったね」「よく見てたね」 |
| 系统性霸凌/殴打 | 违反 IP 红线 | 改为传言、无视误会、或自然现象误读 |

【黄色提示 · 建议加强】

| 触发词/场景 | 建议 |
|-------------|------|
| 黑板/值日 | 加入黒板消し放置、学級目標 |
| 放学后留教室 | 明确时间：放課後直后（15:00前后）或係活动后；勿与中学部活混淆 |
| 开学/入学 | 4月入学式：正式、紧张、家长参与、ランドセル |
| 怪谈/鬼 | 作为同学传言出现，结尾科学揭穿 |
| 告别 | 改为走一段路，路口「じゃあね」 |

【绿色确认 · 已到位】

| 元素 | 说明 |
|------|------|
| 室内鞋/上履き | 日本小学核心制度，应保留并细化 |
| 给食/值日 | 标志性日常，可完整写出流程 |
| 係活動 | 可通过角色係身份展示性格 |
| 自由研究 | 天然契合本 IP，可作番外或暑假卷 |


名古屋地域补充（本项目专属）
----------------------------
舞台：名古屋市，爱知县。富裕住宅区与老街区交界。
  · 上下学：斜坡 → 神社鸟居 → 老街区 → 葛西杂货铺 → 口袋公园 → 学校
  · 气候：四季分明；4月开学（樱花）、6月梅雨、1月干冷（静电卷素材）
  · 建筑：小学多为单层或低层校舍，教室引き戸，走廊换鞋
  · 转学生：4月春假后转学（陸珣/陸瑆入社时间线）符合日本学年节奏


与创作红线的关系
----------------
· 文化顾问不替代科学准确性——两者并行
· 顾问负责「像不像在日本小学」，红线负责「能不能给孩子看」
· いじめ：顾问描述真实形态；红线禁止将其作为剧情引擎
· 怪谈：顾问提供素材；红线要求零恐怖、科学揭穿
