## 加藤慧美 · 实验者

```
│ 五年级 │ 外向0.40 │ 话少0.35 │
│ 人格Solo：爱做实验的认真派——不急着下结论
│
│ 锚点：银框圆眼镜 + 低马尾 + 银色细发夹
│       思考时→食指推眼镜→手指停半秒
│       做实验时手极稳：倒水不洒 / 夹试纸不歪
│       书包外侧口袋：一包纸巾（给志郎的）
│
│ 绝不说：「你错了」→ 替代「まだ確かめてない」
│         「太厉害了！」→ 替代 推眼镜+「……うん」
│         「我来总结一下」（禁止班长腔）
│        长串术语（让陸瑆笔记或附录解释）
│ 绝不做：证据不足时下结论 / 当面让人难堪 / 替全队做决定
│
│ 声线：偏慢 / 平稳 / 6-15字 / 一句就是一句
│ 口头禅：「ちょっと待って——まだ確かめてない。」
│
│ 对志郎：纠正+关心——从不说「你错了」，说「还没验证」
│ 对陸珣：记关键词——他讲太快只问「…それ、今の謎と関係ある？」
│ 对理紗：把她的捕捉放进实验框架验证
│
│ 例外：实验和预测一致→嘴角动一下（不笑出声）
│       被志郎逗到→低头轻轻笑一下→恢复
│       被夸→推眼镜/偏头/耳尖红
```

## 陸珣 · 微观猎人 ★ 男主

```
│ 五年级 │ 外向0.05 │ 专注1.00 │ 话极少0.05 │
│ 人格Solo：钝感的专注者——日常迟钝，领域内骤然精准
│ 一周前从大阪搬到名古屋
│
│ 锚点：椭圆脸 + 偏白肤色（树荫下的斑驳白）
│       深棕短发/刘海遮眉/后脑勺常翘
│       深褐单眼皮——日常平静疏离 / 观察时骤然聚焦
│       蹲下观察：蜷起+左手撑膝+右手靠近+手指极稳
│       紧张时→左手摸书包侧袋（收集管=安全锚点）
│       深蓝尼龙包 + 迷你温度计 + 放大镜挂坠
│
│ 绝不说：「我觉得」→ 沉默或直接说事实
│         「谢谢」→ 替代「……そうかな」
│         连续三句以上解释（被盯就停）
│ 绝不做：第一个发现日常异常 / 主动和陌生人说话
│         非专业领域发表意见 / 情绪外露（陸瑆除外）
│
│ 兴趣：① 收集昆虫+研究习性 ② 花园果树与鸟 ③ 看爸修车
│       ④ 弹钢琴与声音 ⑤ 科学杂志（机器人/航天/汽车）
│
│ 声线：日常1-3字（うん/そう/違う）；领域8-15字+术语
│ 口头禅：「そこにある。」
│
│ 对陸瑆：唯一的例外——温和+多说话（妹妹能听懂）
│ 对伊藤光：被邀请——光看到了他笔记本里的东西
│ 对加藤：让她验证自己的直觉
│ 对中谷琦：无声默契——他蹲下，琦已举起相机
│
│ 陸瑆语录：「哥哥对世界转过身去——但从来没对我转过身」
│
│ 例外：话题进入昆虫/机械/声学→语速加快+眼神发亮
│       被盯→戛然而止+耳尖红+低头
│       陸瑆叫→温和度+50%
```

## 山本理紗 · 感知者

```
│ 五年级 │ Vol11加入 │ 话极少0.05 │ 听觉/视觉担当 │
│ 人格Solo：沉默的捕捉者——能看到别人忽略的一切
│
│ 锚点：鹅蛋脸 + 茶褐丹凤眼（眼尾微挑）+ 眼神迷离
│       肤色白皙/半透明——能看到隐约血管
│       动作极轻→走路几乎无声
│       听声音→偏头+眼神凝固+耳朵转向声源（像调谐收音机）
│       用手触摸材质→用指尖确认，不是用眼睛
│       A5布面速写本+自制色卡本（每页一色+标注色相/明度/饱和度）
│
│ 绝不说：「我觉得是这个原因」→ 替代「色が、違う」
│         长篇解释（她的贡献=一句话+一个指向）
│ 绝不做：大声说话/成为中心/没被问到就主动解释
│
│ 声线：极慢/轻柔/3-8字/陈述句多疑问句少
│ 口头禅：「色が、違う。」
│
│ 敏锐 ≠ 聪明：她能捕捉但不能分析
│ 需要加藤：把她的捕捉放进逻辑框架
│ 加藤需要她：没有她的感知，逻辑链会漏掉最细微一环
│
│ 例外：完美色彩过渡/和弦共鸣→嘴角微笑（难得）
│       被夸→低头/刘海遮眼/耳尖粉
│       需要走进黑暗→毫不犹豫上前——那是她的"领域时刻"
```

```
│ 三年级 │ 9岁 │ 女主人公 │
│ 人格Solo：安静的洞察者——不喜欢昆虫，喜欢看人
│
│ 锚点：圆润脸+婴儿肥 / 齐肩黑短发 / 深褐圆眼（扫视型）
│       听人说话→头微偏+快速扫视脸→手→物品（1-2秒）
│       抱笔记本→左手托底/右手拿笔/笔尾轻点下巴
│       牛皮纸封面线圈本 + 银色自动铅笔（哥哥送的）
│
│ 兴趣：① 不喜欢昆虫/不碰不收集 ② 看人（视线/表情/停顿）
│       ③ 魔术逆推 ④ 写笔记 ⑤ 看云/天色/远处（抬头型）
│
│ 绝不说：「哥哥好厉害！」→ 替代「兄はああ見えて、ちゃんと見てる」
│         公开纠正哥哥（私下→早餐桌上淡淡说）
│         说虫子可爱（保持距离+递纸巾）
│ 绝不做：参与现场破案 / 碰虫子 / 替哥哥谦虚
│
│ 声线：平缓/轻柔/清亮/每字清晰（和哥哥完全相反）
│ 口头禅：「ちょっと待って——順番が、たぶん逆。」
│ 人际一刀：「あの人、さっき目が右に行った。」
│
│ 与哥哥：信任不盲从——笔记里写满钦佩，早餐桌上轻轻纠正
│ 成长轨迹：从「今日、兄は言った…」到「私は思うんだけど…」
│          再到「兄と同じ結論になった。」
```
