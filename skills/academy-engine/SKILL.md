## 一、创作红线（触线即废）

以下四条为不可触碰的硬约束，违反任何一条即需要重写：

1. **零恐怖**：无超自然、鬼怪、血腥、暴力。所有"怪事"均可被科学原理完美解释。
2. **大人永不破案**：成人可提供场景/道具/有限信息，但核心推理和验证必须由孩子独立完成。
3. **无霸凌无政治**：角色冲突限于认知差异或性格火花，不涉及任何现实政治。
4. **每卷原理不重复**：每卷核心原理组合唯一，不与任何已出版卷重复。

---

## 二、八条创作原则

1. **知识驱动而非说教**：原理通过角色对话和实验过程自然带出，不设"知识小贴士"或旁白框。
2. **每卷至少3个学科原理交叉**：单一原理太简单，三个以上原理交叉才构成真正的"谜题"。
3. **实验可复现**：每卷末尾必须包含家庭/教室可复现的验证实验，材料日常易得。
4. **人物先行**：谜题服务角色，每个谜题与团队当前的人物关系、能力水平匹配。
5. **日常异常**：所有谜题发生在普通日本小学校园日常场景中，无"惊天大案"。
6. **第三人称限知 + 妹妹笔记**：主线跟随陆珣视角，每卷结尾附陆瑆第一人称笔记本记录。
7. **结尾干净利落**：谜题解开→记录→一句意象留白→结束，不过度抒情。
8. **三阶段难度递进**：1-10卷初级→11-30卷中级→31-50卷高级。

## 四、七种推理范式与角色映射

每种范式对应特定角色的主导能力：

| 范式 | 学科领域 | 主导角色 | 核心逻辑 |
|------|---------|---------|---------|
| 实验验证 | 物理/化学/生物 | 加藤慧美 | 假设→对照实验→结论 |
| 结构还原 | 力学/工程/建筑 | 松本志郎 | 拆解结构→还原设计意图 |
| 感知分析 | 光学/声学/色彩 | 山本理紗 | 发现感官异常→定位物理原因 |
| 历史溯源 | 历史/档案/考古 | 葛西/中谷琦 | 当下谜题→追溯旧物/记忆 |
| 微观领域 | 昆虫/动物/植物 | 陸珣 | 调用专业深度知识→唯一答案 |
| 人际逻辑 | 心理/行为经济 | 伊藤光 | 分析行为模式与动机 |
| 系统权衡 | 社会/经济/法律 | 加藤+伊藤 | 多因素无法同时最优→找平衡点 |

每卷需至少使用一种范式，并确保与该卷原理组合匹配。

## 五、叙事规范

### 视角规则
- **主线**：第三人称限知视角，跟随陸珣（哥哥）
- **结尾**：陸瑆（妹妹）第一人称笔记本记录
- 陸珣不是第一个发现异常的人（那是伊藤光的角色），但他是故事的主视角摄像机

### 每卷标准结构
```
1. 发现异常（日常中的"不对劲"） → 伊藤光主导
2. 提出假设（群策群力，各有贡献） → 加藤慧美主导
3. 设计验证（实验/观察/溯源） → 松本志郎或团队协作
4. 关键突破（陆珣的领域知识或山本的感知发现）
5. 得出结论（科学解释，干净利落）
6. 陸瑆笔记（妹妹视角收束，情绪留白，不直白抒情）
```

### 语言风格
- **不抒情不说教**：禁止"原来科学这么奇妙啊"等感叹式收尾
- **对话生活化**：日本小学生日常口语，翻译腔要轻
- **知识自然嵌入**：角色对话带出，如"你家冬天窗户上会结水珠吗？那个就是凝结。"
- **每卷结尾只留一个意象**：如"窗框依然是冰凉的，但椅子是干的。"

### 陸瑆笔记写作规范
- 第一人称，"我是陸瑆，三年级"
- 语气温柔但克制，不煽情
- 结尾通常是关于哥哥的一个观察
- 好例子："今日、兄は新しい学校で初めて友達ができた。帰り道、少しだけ背が伸びたみたいに見えた。"
- 坏例子："哥哥太厉害了！我好崇拜他！"（太直白 + 感叹号泛滥）

## 六、故事卷设计模板

每卷需包含以下维度：

```
卷编号：第X卷
卷名：【日文暂定名 / 中文名】
季节：月份 + 学期阶段
谜面包装：孩子视角看到的"怪事"（一句话）
核心谜题：真正需要解开的科学问题
核心原理：≥3个，标注学科
辅助原理：配合核心原理的知识点
推理范式：七选一或组合
关键线索：3-5个，含误导项
验证方法：实验/观察/追溯/对比
陸珣入社时刻（仅第1卷）：陆珣靠什么贡献被邀请入社
陸瑆笔记方向：这卷结尾她观察到什么
实验设计：家庭可复现，材料清单 + 步骤
```

---

## 七、实验设计规范

每卷末尾实验页必须满足：

- **材料**：全部为日常家庭用品（冰箱、铁片、玻璃片、纸巾等）
- **步骤**：≤5步，每步一句话
- **安全**：无火、无电、无化学危险品
- **课标对应**：标注中日小学科学课标对应年级
- **实验与故事关联**：实验直接验证故事中的原理

## Gotchas

**名字绝对不能错**：山本理紗不是山本琦理。陸珣（りく しゅん）不是陸琦音。以角色数据库为准。

**陸珣的"钝感"是关键**：他在日常中沉默寡言、存在感低，但一旦谜题进入他的领域（昆虫/机械/声学）就骤然精准。他的入社事件（第1卷）必须是他靠一次领域观察解开了三人团卡住的谜题，而非偶然。

**伊藤光才是第一发现者**：虽然主视角跟随陸珣，但"第一个发现异常"永远是伊藤光。陸珣的主视角功能是"观察伊藤光在观察什么"。

**每卷扣一个季节/节日**：日本小学三学期制（4-7月 / 9-12月 / 1-3月），迷题应嵌入当季的自然现象或校园活动。

**陆瑆笔记不要太长**：3-5句话，一个观察，一句留白。禁止超过8句。

**实验材料必须是家里就有的**：铁片（五金店）、玻璃杯、纸巾、温度计、冰箱。禁止要求"购买专业实验器材"。

**日本校园细节要准确**：室内鞋换鞋、给食值班、扫除时间、部活动放学后、鞋箱（下駄箱）、生徒会——这些场景元素要自然嵌入，不能错。
