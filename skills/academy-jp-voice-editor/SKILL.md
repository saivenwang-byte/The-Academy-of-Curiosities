## 工作流（四遍 + 一门禁）

```
① 读中文定稿 + 人名定稿 + 03_LANGUAGE 标准
② 日译稿对照：漏译/增译/科学术语表记
③ MoA -lite 四视角自评（见下表）→ 改稿
④ 可选：npx textlint 完整文字稿_日本語.txt（若已配置 tools/textlint/）
⑤ 产出 语感编辑摘要_日本語_VolN.md → 交田中全文督查 · **J10 标日字符自检**
```

### MoA-lite 四视角（单 agent 内分四轮读稿）

| # | 视角 | 本 IP 检查 |
|---|------|-----------|
| 1 | 論理 | 公平线索是否在日文版仍可见；揭晓前无剧透 |
| 2 | 文体 | 均句 ≤40 假名（叙事）；短句留白；比喩 ≤2/场景 |
| 3 | 語彙 | J1–J5 + **J10 标日字符**；结露・露点・熱伝導 小学理科一致 |
| 4 | 読者 | 10–12 岁可独立读；注音见读音标注策略；**仅假名+日汉字** · 禁中式语法 |

## 日文铁律（Vol1+）

### 文体

- **基本**：第三人称过去形（だ・である / である調の叙事，对话用口语）
- **陸珣（りく しゅん）**：哥哥 · 全卷对外台词 ≤5；沉默观察；不用名古屋弁
- **陸瑆（りく ひかる）**：妹妹 · **读音锁定** · 笔记层克制
- **伊藤光（いとう ひかる）**：与瑆同音 **ひかる** · **姓で区別** · 正文必写汉字（光 vs 瑆）
- **保留金句意象**（可日文化，勿扩写）：
  - 「怪事可以被重复…」→ 「不思議は、繰り返せる。繰り返せるなら、もう不思議じゃない。」
  - 「词多了，反而像掩饰。」→ 「言葉が多いほど、隠しているみたいに聞こえる。」
  - 「窗框还是凉的，四月还是四月。」→ 「窓枠はまだ冷たい。四月は、まだ四月。」

### 直译腔禁止（J1 速查）

| NG | OK |
|----|-----|
| 報告する先生 | 先生に言う / 先生のところへ |
| 食堂 | 給食 / 配膳 |
| 班长 | 学級委員 / 当番 |
| 推开门进教室 | 引き戸を開けて |
| 你太棒了 | 頑張ったね |

### 科学发声（正文 3 处，与中文版对齐）

1. 误导推翻后：陸珣观察句（保留或微调「……その椅子、窓枠に当たってる。」）
2. 验证成功：加藤慧美 **一句** 三原理（熱伝導・湿度・露点）
3. 硬壳本表 / 实验页：表格化

### オノマトペ

- 适度保留：ガチャン、キュッキュッ、ひんやり
- 禁止堆砌；角色分配见 `09_日本参考资料库/04_儿童文化与亚文化/`
