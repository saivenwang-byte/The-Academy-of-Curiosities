第一章：先輩／後輩 — 纵向等级关系
========================================================================

一、制度基础
------------------------------------------------------------------------
先輩（せんぱい）和後輩（こうはい）是日本社会最重要的纵向等级关系。
这一制度从小学就开始训练，一个人一生都在这个框架内感知自己的位置。

在小学阶段：
- 同年级之间没有先輩/後輩关系
- 高一个年级就是明確的先輩
- 六年級对一年級拥有絕對的前輩地位
- 教师和学生的关系是在この基础上叠加的另一个纵向层级

二、称呼规则
------------------------------------------------------------------------
称呼先輩时：
  ✅ 姓氏 + さん（田中さん）
  ✅ 姓氏 + 先輩（田中先輩）
  ✅ 名前 + さん（相手との関係性による）
  ❌ 直呼其名（タブー）
  ❌ あだ名で呼ぶ（親しくても不可）
  ❌ 呼び捨て（絶対に不可）

称呼後輩时：
  ✅ 姓氏 + さん/くん/ちゃん
  ✅ 下の名前 + くん/ちゃん
  ✅ あだ名（OKだが、相手が嫌がるものは不可）

三、行为规则
------------------------------------------------------------------------
1. 部活中的先輩/後輩
   - 一年生要负责收器材、打扫场地
   - 先輩分配任务，後輩执行
   - 先輩的指导必须安静听
   - 反对先輩的意见需要极其谨慎的表达

2. 日常生活中的先輩/後輩
   - 走廊里遇到先輩要让路
   - 先輩说话时不能插嘴
   - 先輩面前不能太随便
   - 但小学生之间的先辈关系比中学宽松很多

3. 上下学路上的先輩/後輩
   - 通学班（集体上下学）中，高年级照顾低年级
   - 六年级牵着一年级的手过马路
   - 高年级有责任保护低年级

四、故事中的应用
------------------------------------------------------------------------
《学堂趣事录》Vol1 观察社活动核心为 **4 人**（光·慧美·志郎·珣 · 跨 3 班）；
陸瑆（**4年2組**）为笔记层、非社员。Vol11 起对外「核心五人团」才含山本理紗。
先輩/後輩关系主要存在于：
  - 与六年级学生的互动（偶遇场景）
  - 部活场景（如有社团活动描写）
  - 陆珣（5年2組）与陆瑆（4年2組）的兄妹关系（差 1 学年）
  - 一年级小朋友对高年级观察者的仰望

第二章：空気を読む — 读空气
========================================================================

一、核心概念
------------------------------------------------------------------------
「空気を読む」（くうきをよむ / kūki o yomu）直译为"读空气"，
是在没有明确言语指令的情况下，感知并顺应集体氛围的能力。

这是日本社会最重要的社交技能。从小学一年级就开始被期待掌握。
"空気が読めない"（读不懂空气 = KY）是一种严重的社交缺陷。

二、小学里的"读空气"场景
------------------------------------------------------------------------

1. 课堂场景
   - 老师说"誰か答えてくれる人？"（谁来回答？）
     → 全班低头，没有人举手
     → 但空气中有一种默契：不是没人知道答案，是等待老师点名
     → 如果一个人举手，其他人开始陆续举手——这就是"读空气"

   - 老师说"ここ、大事なところだよ"（这里是重点哦）
     → 这不是提醒，是暗示"你们该记笔记了"
     → 中国孩子可能只听，日本孩子会立刻拿笔
     → 不记笔记=没读懂空气

2. 给食场景
   - 有讨厌的食物时，不能直接说"嫌い"
     → 大多数孩子默默吃
     → "吃完了"是给做饭的人和全班的面子
     → 如果一个人大声说"これ嫌い！"——KY

   - おかわり（加菜）时
     → じゃんけん（猜拳）决定谁获得
     → 如果一个人直接说"俺がもらう"（我要了）——KY

3. 掃除场景
   - 分到的任务不喜欢（比如扫厕所），不能说"嫌だ"
     → 默默接受是基本社交规范
     → 抱怨的人会被认为"わがまま"（任性）

4. クラス替え（换班）场景
   - 公布新班级名单时
     → 和朋友分到同班——可以高兴，但不能太夸张
     → 和朋友分到不同班——不能哭，不能抱怨
     → 表现得太兴奋或太沮丧都是"读不懂空气"

三、"读空气"的三层含义
------------------------------------------------------------------------

1. 感知层：感觉到"现在的气氛不对"
   例：走进教室，发现同学们都低着头安静写字
   → 知道老师可能刚发过火

2. 适应层：调整自己的行为以符合气氛
   例：本来想大声和朋友打招呼，但看到教室很安静
   → 改为小声说"おはよう"

3. 创造层：用自己的行为影响空气走向
   例：全班都在犹豫要不要帮某个同学
   → 第一个站出来帮忙，其他人会跟着

第三章：いじめ — 日本形态的特殊性
========================================================================

一、日本いじめ与中国霸凌的本质区别
------------------------------------------------------------------------
中国的校园霸凌通常以肢体冲突、公开辱骂、财物侵占为主。
日本校园的いじめ（欺凌）最常见形态是"无视"（無視 / むし）。

| 特征       | 日本いじめ                    | 中国校园霸凌              |
|-----------|-----------------------------|-------------------------|
| 主要形式   | 集体无视、孤立、冷暴力         | 肢体冲突、公开辱骂         |
| 可见度     | 隐蔽，外人难以发现             | 相对明显                   |
| 施害者     | 群体，至少3人以上              | 个人或小群体               |
| 受害者的反应 | 沉默、不报告（报告=更大的孤立） | 可能反抗或报告老师         |
| 社会态度   | "无法根除"的宿命感            | "必须严肃处理"的道德意识    |

二、いじめ的六种日本形态
------------------------------------------------------------------------

1. 無視（むし）—— 集体无视
   - 班上所有人（或主流群体）约定不和某个人说话
   - 不看他、不回应他、当他"不存在"
   - 走廊里遇到时故意转身
   - 这是最常见的日本学校いじめ

2. 悪口（わるぐち）—— 在背后说坏话
   - 不是在面前骂，是背后传播
   - LINE群组里建"除某人外"的群
   - 在课桌上写坏话然后擦掉（让受害者看到残留痕迹）

3. 仲間外れ（なかまはずれ）—— 排除在群体外
   - 分组时不让他加入任何组
   - 午饭时不让他坐在一起
   - 放学后不和他一起走

4. 隠す・壊す（かくす・こわす）—— 藏东西/破坏
   - 把室内鞋藏起来
   - 在教科书上乱画
   - 把文具盒扔进垃圾桶
   - 但这些行为通常不是主要形式，而是辅助手段

5. ネットいじめ —— 网络欺凌
   - LINE群组的恶意信息
   - 在SNS上发嘲笑帖子
   - 小学阶段比中学少，但呈上升趋势

6. からかい —— 以"开玩笑"为名的持续性骚扰
   - 最难界定的一种
   - 施害者说"冗談だよ"（开玩笑的啦）
   - 受害者不敢反驳（反驳=开不起玩笑）
   - 但每天这样就是いじめ

三、いじめ的班级生态
------------------------------------------------------------------------
- いじめ通常不是全班参与，而是核心3-5人带动
- 其余人是"旁观者"——沉默即共犯
- 反对いじめ的人自己可能成为下一个目标
- 老师通常无能为力（いじめ太隐蔽）

四、故事创作中的注意事项
------------------------------------------------------------------------
《学堂趣事录》遵循"无霸凌"红线。
但这不意味着不能写社交冲突。
可以使用的安全替代方式：
  ✓ 新来的转学生一时难以融入（陸珣的自然状态）
  ✓ 小组作业中的意见不合（认知差异，非恶意）
  ✓ 误解后被冷淡（后续和解）
  ✓ "读不懂空气"导致的尴尬（非恶意的）
  ✗ 任何人被全班孤立
  ✗ 故意藏东西/破坏
  ✗ "无视"作为情节


========================================================================
第四章：スクールカースト — 校园隐形等级
========================================================================

一、什么是スクールカースト
------------------------------------------------------------------------
スクールカースト（School Caste / 校园种姓）是日本中小学中
自然形成的社交等级制度。与成绩无关，由以下因素决定：

- 社交能力（朋友多、说话有趣）
- 运动能力（体育好、运动会主力）
- 外貌（可愛い・かっこいい）
- 流行敏感度（知道最新的梗、最新的游戏）
- 表达能力（能给全班带来欢乐的人地位高）

二、カースト层级
------------------------------------------------------------------------

1. 一軍（いちぐん）—— 班级中心人物，约20%
   - 朋友多，午餐时周围围着人
   - 决定班级的"空气"
   - 提出的意见容易被全班接受
   - 被他们排斥=班级末日

2. 二軍（にぐん）—— 普通学生，约60%
   - 有自己固定的小团体（グループ）
   - 和一軍有交集但不主导
   - 大多数孩子的状态
   - 五人团的主要社交位置

3. 三軍（さんぐん）—— 边缘学生，约20%
   - 安静、不太说话
   - 喜欢一个人待着
   - 不太关心流行
   - 被いじめ的高风险群体
   - 陸珣最初的社交位置

三、カースト的流动性
------------------------------------------------------------------------
小学阶段的カースト不如中学固定：
  ✦ 换班（クラス替え）可以重置カースト
  ✦ 运动会上的出色表现可以提升
  ✦ 一个新来的有趣转学生可以改变カースト地图
  ✦ 一学期内カースト会发生微妙变化


========================================================================
第五章：グループ（小团体）
========================================================================

一、自然形成过程
------------------------------------------------------------------------
班级里会自然形成3-5个稳定的小团体（グループ）：

1. グループ的形成时间
   - 通常在开学后2-3周内形成
   - 换班是形成新グループ的关键时刻
   - 转学生在グループ已经形成后才加入——天然的不利位置

2. グループ的类型
   - 基于座位的团体（隣の席がきっかけ）
   - 基于爱好（ゲーム/スポーツ/漫画/アニメ）
   - 基于通学路（同じ方向に帰る）
   - 基于習い事（同じスイミングスクール等）

3. グループ的日常
   - 午餐一起吃
   - 远足时自由组队（2-4人一组）
   - 运动会時自然聚在一起
   - 休息时间一起在走廊聊天

二、跨グループ的社交规则
------------------------------------------------------------------------
- 不同グループ可以在课堂上融洽对话
- 但课余时间各回各グループ
- "誰と誰が仲がいい"是每个孩子都知道的情报地图
- 加入新グループ需要微妙的社交操作

第六章：転校生 — 转学生的社交困境
========================================================================

一、転校生的特殊位置
------------------------------------------------------------------------
转学生（転校生 / てんこうせい）在日本小学是一个特殊的存在：

优势：
  ✦ 自动获得全班关注（新鲜感）
  ✦ 可以重新定义自己的形象
  ✦ 没有过去的"黑历史"

劣势：
  ✦ グループ已经形成，需要硬挤进去
  ✦ 不熟悉班级内部的隐形规则
  ✦ 不知道谁和谁是朋友/谁和谁关系不好
  ✦ 方言或口音可能被注意
  ✦ 不熟悉学校的习惯（如换鞋的具体流程）

二、転校生の典型的な初日
------------------------------------------------------------------------
1. 早上被班主任带到全班前
   "今日から新しい友達が増えました。○○です。"
2. 在黑板上写下自己的名字
3. 简短的自我介绍（名前・前の学校・好きなこと）
4. 被安排到空座位
5. 课间休息时，几个好奇的孩子围过来问问题
6. 一周内，要么融入、要么停留在"照顾对象"阶段

三、陸珣的社交定位（创作参考）
------------------------------------------------------------------------
陸珣作为転校生，他的社交位置符合以下特征：
  ✦ 三軍カースト（安静、不起眼）
  ✦ 还没有グループ归属
  ✦ 被伊藤光（一軍）主动接触——这是極めて幸運
  ✦ 他的笔记本（昆虫素描）成为一个社交"卖点"
  ✦ 通过专业知识获得团队认可——这是最稳固的入社方式

第七章：道歉与原谅 — 日本の儀式感
========================================================================

一、道歉的层级
------------------------------------------------------------------------

1. 轻微道歉（教室日常）
   情景：不小心碰掉了同学的笔
   用语：「あ、ごめん」（Ah, gomen）
   姿势：可能不鞠躬，只是语气

2. 中等道歉（造成麻烦）
   情景：忘记做值日、借的书逾期
   用语：「すみませんでした」（Sumimasen deshita）
   姿势：微微低头

3. 正式道歉（产生后果）
   情景：弄坏别人的东西、伤害了别人
   用语：「本当にごめんなさい」（Hontō ni gomen nasai）
   姿势：正式鞠躬（お辞儀 / おじぎ）

4. 全班面前道歉
   情景：影响了全班的重大错误
   流程：
     a) 站在全班前面
     b) 「みんなの前で謝ります」
     c) 详细说明自己做错了什么
     d) 承诺不会再犯
     e) 鞠躬
     f) 全班或沉默或接受

二、原谅的方式
------------------------------------------------------------------------
- 不是口头说"没关系"（大丈夫だよ）
- 而是轻轻地点头，用表情表示"嗯我知道了"
- 肢体动作比语言更重要
- 被原谅后通常会说「ありがとうございます」

三、故事应用
------------------------------------------------------------------------
道歉场景的描写要点：
  ✦ 克制，不要过度渲染
  ✦ 动作比语言重
  ✦ "没关系"不是必须说的台词
  ✦ 沉默+眼神接触=已经原谅了

第八章：情感表达 — 日本の「間」と「分寸」
========================================================================

一、表と裏（おもてとうら）
------------------------------------------------------------------------
日本人有一个著名的概念：「本音」和「建前」。

本音（ほんね）—— 真实想法，内心深处的声音
建前（たてまえ）—— 公开表达，符合社会期待的台词

小学阶段的日本孩子已经开始掌握这套系统：
  - 老师说"給食は残さず食べよう"（午餐不要剩）
  - 学生嘴上说"はい"（好的）
  - 但内心可能在说"これ嫌いなんだよな"（这个我讨厌啊）

这不是"虚伪"，这是社会润滑剂。

二、友達表达の「分寸」
------------------------------------------------------------------------
日本孩子表达友谊的方式极其克制：

❌「あなたは私の一番の友達だよ！」（你是我最好的朋友！）
   太直接、太重、太异常。日本小学生不会这样说。

✅ 放学后一起走一段路，在某路口自然分开
   "じゃあね"（那么明天见）
   没有更多言语，但这段同行的距离就是友谊的证明。

✅ 课间休息时自然坐到旁边
   不说话，只是做自己的事
   但选择坐在谁旁边是经过了选择的

✅ 给食时，把不爱吃的菜自然放到对方碗里
   "これ食べる？"（这个你要吗？）
   不是施舍，是"我记得你喜欢这个"

三、鼓励与称赞
------------------------------------------------------------------------
日式鼓励：「頑張ったね」（你努力了）
  → 重心在过程，不在结果

中式鼓励：「太厉害了！」「真棒！」
  → 重心在结果

在《学堂趣事录》中：
  ✦ 所有鼓励台词必须使用日式表达
  ✦ 避免"你太厉害了！"
  ✦ 使用"頑張ったね"、"すごい"（但克制使用）
  ✦ 最真心的赞美往往是无声的 —— 一个点头、一个注视

四、沉默的力量
------------------------------------------------------------------------
在日本文化中：
- 沉默 ≠ 冷漠
- 沉默 = 尊重对方的空间
- 沉默 = 给双方思考的时间
- 沉默 = 最深的理解不需要语言

安慰场景的正确写法：
❌「別に大したことじゃないよ。気にしないで！」
   （没什么大不了的。别放在心上！）
   → 否定对方感受，让人更难受

✅ 走到旁边，坐在地上，什么都不说。
   沉默了几分钟。然后对方自己开口了。
   → 尊重对方的情绪节奏


========================================================================
第九章：学級会と話し合い文化
========================================================================

一、学級会の基本構造
------------------------------------------------------------------------
- 学級委員（男女各1人）主持
- 全班围坐成圓（机を下げて、椅子だけを丸く並べる）
- 讨论的主题通常在黑板上写下
- 每个人的发言不能说太长
- 多数决定制（多数決）

二、討論のルール
------------------------------------------------------------------------
1. 发言前先举手，被点到后才说
2. 反对他人意见时用"でも…"（但是…）开头
3. 说"はい"表示听到了，不等于同意
4. 少数派意见也会被记录在黑板上

三、合意形成
------------------------------------------------------------------------
日本学級会追求的不是"胜利"，而是"みんなが納得できる"（大家都能接受）。
这是一种极其日本的决策哲学：
  - 不是A方案打败B方案
  - 是找到一个A+所有人都能接受的修改版
  - 时间比效率重要


========================================================================
第十章：性别与社交
========================================================================

一、小学的性别社交特征
------------------------------------------------------------------------
1. 低学年（1-2年级）
   - 男女混合玩，差异不明显
   - 一起玩おにごっこ（捉迷藏）、ドッジボール（躲避球）

2. 中学年（3-4年级）
   - 开始出现男女分群
   - 男生群体和女生群体自然分离
   - 但男女同桌依然正常

3. 高学年（5-6年级）
   - 性别意识明显增强
   - 男女分开玩的趋势更强
   - 异性之间的互动会增加
   - 但从"男女朋友"的角度看，小学生极少公开谈论恋爱

二、观察社核心与 Vol11 五人团（创作参考）
------------------------------------------------------------------------
**Vol1 观察社活动核心 = 4 人**（伊藤光 · 加藤慧美 · 松本志郎 · 陸珣 · 跨 3 班）。
陸瑆（4年2組）= 笔记层 · 非社员 · 不出场主帧。

========================================================================
第十一章：家族関係
========================================================================

一、きょうだい（兄弟姐妹）
------------------------------------------------------------------------
- 年上の子は年下の子を守る責任がある
- 在学校里，哥哥姐姐会在意弟弟妹妹的感受
- "お兄ちゃん"（哥哥）和"お姉ちゃん"（姐姐）的称呼在学校里使用

二、親と教師の関係
------------------------------------------------------------------------
- 家庭訪問（5月）：班主任来家访15-20分钟
- 保護者会（每学期1-2次）：家长到校开会
- 連絡帳（れんらくちょう）：家校之间的日常沟通工具
  - 家长每天签名
  - 特殊事项通过連絡帳沟通
  - 連絡帳是"半公开"的——孩子可以看到内容

三、陸家の家族力学（创作参考）
------------------------------------------------------------------------
- 父亲陸直人：温和的创造者，不直接教但"做给孩子看"
- 母亲陸美咲：厨房实验室的操作者，生活中渗透科学
- 陸珣（哥哥）：沉默，被崇拜但不自知
- 陸瑆（妹妹）：崇拜哥哥但保持独立思考
- 没有中国式"你要好好学习"的说教
- 教育是"做"的，不是"说"的
