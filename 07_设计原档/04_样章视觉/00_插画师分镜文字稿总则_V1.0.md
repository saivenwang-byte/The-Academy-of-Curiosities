## 一、两条铁律（清查工单 · 不可协商）

| # | 铁律 | 执行 |
|---|------|------|
| **T1** | **锁一种画风** | 全卷 **Style B**（= **桥梁书动画线稿+数字厚涂** · 口语「马克笔平涂」）· 垫图 [`CHAR_lineup_L0_StyleB_马克笔_V0.2.png`](./CHAR_lineup_L0_StyleB_马克笔_V0.2.png) · **禁** G1DRAFT 水彩混用 · 命名对照见 [补充规范 §0.1](./00_A001-A005_插画师分镜补充规范_V1.0.md#01-画风锁定只许一种) |

| **T3** | **出场人数算术** | 每帧 **G-CAST** 必填：`MAX_BODIES` · 具名名单 · 匿名上限 · 禁止具名 · 成图后 **数人头 ≤ MAX** · 见 [`106_G-CAST_出场人数算术关_V1.0.md`](../../03_故事内容/第1卷_觉得奇怪就先观察/V2迁移/106_G-CAST_出场人数算术关_V1.0.md) · **未 PASS 禁止出图/进 PDF** |
| **T4** | **JP 正文锚句 + M0 PASS** | 分镜 `原文范围` / `画面瞬间` 须链 **JP V3.8 定稿段落**（非 CN-only）· **翻译部 M0-A/B 全绿前禁止定稿分镜与 GenerateImage** · 见 [`00_翻译部与编辑部_分工_V1.0.md`](../../03_故事内容/第1卷_觉得奇怪就先观察/单元1_第一单元_五案/00_翻译部与编辑部_分工_V1.0.md) |
| **T5** | **G-BRIEF 双签** | 编辑部确认「要画什么」· 翻译部确认画中日文 · **设计部** 接活见 [`00_设计部_插画画工流程_V1.0.md`](../../03_故事内容/第1卷_觉得奇怪就先观察/单元1_第一单元_五案/00_设计部_插画画工流程_V1.0.md) · 未双签禁止出图 |

## 二、每镜头必填字段

| 字段 | 说明 |
|------|------|
| **Shot** | DA1 / DB1 / TAIL … |
| **画面瞬间** | 这一帧「冻结」哪一秒的故事 |
| **视角与情绪** | 景别 · 机位 · 读者第一眼情绪 |
| **出场** | 具名角色 + 背景同学 · **禁止名单**见各案 |
| **画中文字** | 默认 **なし** · 有则写完整日文文案 |
| **重绘** | 🔴 必须重绘（PDF 已烤坏）/ 🟡 修订 / ⚪ 可沿用构图 |
