## 用户红字修订（Lock · 画师必遵）

### ② 陆珣

| 要 | 不要 |
|----|------|
| **专注的眼神** | 涣散、纯背景人 |
| **沉着冷静、善于解决问题** 的气质 | 热血喊叫脸 |
| **精明、有主角聚焦感**（观察型主角） | 普通偶像剧帅气男主 · 强装酷 · 侦探风衣 |

> 正典平衡句：他是 **「蓝色路线少年」** — 安静但能看见空间关系；主角感来自 **凝视与路线**，不是运动番 C 位。

### ⑧ 葛西泉藏

| 要 | 不要 |
|----|------|
| **精神矍铄** · 亲切 · 能讲故事的长辈 | **明显佝偻** · 衰弱病容 · 黑化神秘老人 |
| 可略弯背（高龄）但 **整体挺实** | 魔法导师感 |

> **用户倾向覆盖** brief 原「佝偻」表述 → 定稿用 **「精神矍铄 · 略弯背可接受 · 禁佝偻缩肩」**。

## 进画师稿前 · 正典修正（探索图里有的错）

| 探索图问题 | 正典裁决 |
|------------|----------|
| 「葛西**全藏**」 | 必须 **葛西泉藏** |
| 葛西拿 **放大镜** | **禁** · 改 **旧地图 / 旧笔记本 / 木盒** |
| 「陆**瑈**」等异体字 | **陆瑆** |
| 身高排面用 Sakura/Yuto 等假名 | 举牌以 brief **§3** 假名+中文为准 |
| 理紗 **制服领结** 过重 | Vol1 私服日 · 蓝白秩序用 **私服** 表达（brief 春装） |
| 设定 Sheet 偏「经典动画」浓度 | 保留温暖水彩 · **降** 表演性动态 · 对齐 Style A |
