## 0. 坐标系与尺度（LOCK）

| 项 | 值 |
|----|-----|
| 坐标 | **东向为上**（活动准备室在北）· 西=走廊深处 · 东=下駄箱方向 |
| 走廊净宽 | **≈2.4 m**（小学侧廊标准 · 可画 2.2–2.6） |
| 走廊净高 | **≈2.8 m**（灯管吊顶） |
| 海报板宽 | **≈0.9 m**（A2 横贴感） |
| vent → 海报顶边 | **≈0.15 m** 间隙 |
| 海报板 → 壁报栏 | **步行 ≥3 m**（约 4–5 步 · 禁混为一板） |
| 地面 | 全程 **上履き** · 浅灰瓷砖 |
