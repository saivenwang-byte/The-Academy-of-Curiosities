## 0. 设计原则

1. **读者只见故事与复盘**，不见流程表。  
2. **图文同流**：插图紧跟对应段落后，禁止图库附录。  
3. **机制/实验/线索/瑆** 一律在 **§4 收束之后**（B 轨）。  
4. **每卷同一骨架**：卷首模块 + 单案模块 + 卷末模块；仅换文案与图。

---

## 1. 卷首模块（固定 · 序+A001 试读亦适用）

| 序 | 模块 | 页数 | 内容 |
|:--:|------|:----:|------|
| R1 | **外封** | 1 | 品牌 · 卷名 · 主视觉（DA1 或卷主图）· 无「审阅包」字样 |
| R2 | **版权** | 1 | © · 试读声明（若试读）· ISBN 位 · **无**门禁表 |
| R3 | **人名表** | 1 | 核心四人 + 瑆 · 读音 · 观察社一行 |
| R4 | **目录** | 1 | 序 + 第 N 起（案名）· **不进** §1–4 |

---

## 2. 单案模块（每案重复 · Q1=A 半页标题条）

```
┌─────────────────────────────┐
│ 【半页标题条 · 约 1/3 页高】   │
│  第 1 起 · めくれたポスター    │
│  （可选一行异常句 · 无剧透）    │
├─────────────────────────────┤
│  §1 正文 …                   │
│  [DA1 半页图 · 段后]          │
│  §2 … [DA2] [DA2b inset 可选] │
│  §3 … [DA3] [DA3b 条件]       │
│  §4 … [DA4] [DC1 尾钩]        │
├─────────────────────────────┤
│ ── 故事读完 · B 轨 ──         │
│  DB1 机制示意（半页）          │
│  CL-01 线索卡（1页）          │
│  家庭实验（1页）              │
│  V-S02 瑆日记（半页）          │
└─────────────────────────────┘
```

| 规则 | 说明 |
|------|------|
| 标题条 | **非**整页扉 · 与 §1 首段可同页或占上 1/3 |
| 连续纯文字 | **≤ 6 页**（E03） |
| 完整图间隔 | 每 **8–12 页** 至少 1 张 A/B 档 |
| inset（DA2b/3b/4b） | ¼ 页 · 嵌在对应 § 段后 · **不计独立页** |

## 5. 插图三档（嵌入正文）

| 档 | CSS class | 版心宽 | 用途 |
|----|-----------|--------|------|
| A | `.fig-half` | ~100% · **max-height 48mm** | DA1–DA4 主场景 |
| B | `.fig-quarter` | ~55% · **max-height 38mm** | DC1 · DB1 · V-S00 |
| C | `.fig-inset` | ~40% · **max-height 28mm** | DA2b/3b/4b |

**分页 LOCK（v1.1）**：`.scene-block` / `.figure-wrap` 设 `break-inside: avoid` · 图在**段落后混排** · 禁整页单图（封面与 approved 跨页 spread 除外）。
