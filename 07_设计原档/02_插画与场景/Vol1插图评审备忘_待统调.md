## 总评（主编 2026-06-02）

**画风方向对**：水彩、安全、温柔、日本儿童读物气质。  
**最大缺口**：部分偏「科普绘本」或「泛日本校园美术」，未完全锁住 **名古屋公立小学校 × 本格推理 IP**。

### 总体优点

- 水彩质感柔和干净，亲子安全
- 引き戸、木质课桌、钢管椅、黒板、上履き、掲示物、掃除场景准确度提升
- 结露 / 冷表面 / 湿度 / 露点 / 金属传热方向正确
- 儿童观察感强

### 总体问题（统调必改）

1. **制服感过重** → 改 **私服 + 上履き + ランドセル**；不要中学制服、领结、长裙（除非改设为中学校）
2. **水迹过重** → 结露 = 边缘微湿、细水珠、淡淡纸痕；非泼水 / 扫除积水
3. **科学机制** → 仅 **一把椅背贴窗框**；邻椅干燥、湿痕在靠窗框边缘；非整椅 / 非全列

### 角色固定（Vol1 四人 + 瑆笔记层）

| 角色 | 画面功能 |
|------|----------|
| 陸珣 | 少话；指窗框/细节；蹲姿观察 |
| 伊藤光 | 活跃；夸张误判 |
| 加藤慧美 | 记录；逻辑整理 |
| 松本志郎 | 搬椅；动手实验 |
| 陸瑆 | 笔记层 / 手账页（非正文场景必现） |

### 场景 prompt 必含

```
Nagoya public elementary school, April, 4th grade classroom (4年2組),
RC school building, single-side corridor, sliding door (引き戸),
aluminum window frame, wooden desks, steel-pipe chairs, uwabaki on floor only,
classroom notices, cleaning tools cabinet, 14-15°C spring humidity context
```

**禁止**：generic Japanese classroom / old abandoned school / anime classroom / 中学制服

### 湿痕科学（公平线索）

- 仅 **一把** 椅 **背** 贴アルミ窓枠
- 湿痕：边缘 > 中央；量少；纸巾淡痕
- 邻椅、地面、邻座 **干**
- 时间：早晨或放课后温差成立（4月名古屋）

### 服装

- 私服；上履き在 **地面/下駄箱区域**，不在实验台
- 无统一制服；无领结水手服
