## 场景01 · 教室靠窗座位区（4月 · 早晨）

### 性质

- 室内 · 早晨约 8:00–8:30 · **4月名古屋**
- 气温约 14–15℃ · 春季湿度 · 窗玻璃可有薄雾
- 全书约 60% 画面主场景

### 结构（L2 文化 + 环境指标）

| 元素 | 要求 |
|------|------|
| 门 | **引き戸**（右侧），非铰链门 |
| 窗 | 最右侧靠窗列 · **アルミサッシ** 窗框（冷色对比） |
| 黑板 | 左前方 · 上方 **学級目標** · 課表框 |
| 牌 | **4年2組** 教室牌（可选） |
| 掲示 | **今日のめあて** 等 |
| 外景 | 校庭 + **樱瓣**（4月，非落叶秋樱） |
| 鞋 | **上履き** 仅在地面/鞋柜区，不在课桌/窗台 |

### 谜题关键（公平线索 · 本格）

| 必须可见 | 说明 |
|----------|------|
| 椅背贴窗框 | **仅一把**（伊藤光位）· **椅背** 贴死，非椅腿 |
| 邻椅缝隙 | 其余靠窗椅与窗框 **留缝** |
| 湿痕 | **椅背/贴窗框边缘** 薄水膜 · 量少 · 非泼水 |
| 铁片+载玻片 | 第3章后在窗台 · 可辨材质 |
| 误导可选 | 掃除场景另图 · 本场景地面 **干** |

### 光线

- 4月 **斜射晨光**（非 autumn amber 10月）
- 窗边亮、黑板区略暗
- 金属窗框 `ひんやり` 视觉对比

### AI 提示词（修订版 · 英文）

```

sliding door (引き戸) on right, aluminum window frames on right wall,
```
