## 0. 视觉总原则（IP LOCK）

> **用电影导演的方法组织画面，用儿童桥梁书的方法控制信息，用原创 IP 的方法塑造角色，用科学审校的方法保证线索。**

| 是 | 不是 |
|----|------|
| **电影导演式构思**（景别、机位、节奏、呼吸） | 把桥梁书画成 **写实电影截图** |
| 每一帧回答：**为何必画 · 孩子看见什么 · 如何推动推理/关系** | 每一情节、每一转头都配图 |
| **轻漫画化** + 强 IP 符号 + 可控表情夸张 | 纯写实 · 弱表情 · 幼儿化 |
| 线索公平 · 印刷可读 · 翻页节奏 | 炫技透视 · 冲击压过叙事 |

**未通过 §8 分镜审的分镜规划书 → 不得正式绘制 → 不得 PRODUCT。**

**流程 LOCK**：按 **卷 → 案 → 场 → 镜头** 完成分镜规划书 → **审核** → 正式绘制。

## 1. 分镜的叙事功能（核心 · 强制）

插图 **不得仅重复文字**。每一帧须承担 **至少一项** 功能：

| # | 功能 |
|---|------|
| 1 | **建立空间**（教室/走廊/动线） |
| 2 | **呈现线索**（公平推理所需证据） |
| 3 | **展示人物关系**（分工/冲突/协作） |
| 4 | **表现情绪变化**（误会/理解/成长） |
| 5 | **解释科学现象**（可验证、非图鉴） |
| 6 | **制造翻页钩子** |
| 7 | **第二次阅读才可发现的信息** |

**策划必填问句**（写入分镜表「镜头功能」列）：

1. 为什么一定要画这一帧？  
2. 孩子从这一帧能看到什么？  
3. 它如何推动故事、推理或人物关系？

---

## 2. 六种分镜功能类型（S/C/E/L/P/H）

**景别是形式；叙事功能才是核心。** 每镜头须标注 **主功能**（可双功能，如 S+L）：

| 代码 | 名称 | 核心用途 |
|------|------|----------|
| **S** | 空间建立 | 教室、走廊、路线、人物位置 |
| **C** | 角色互动 | 关系、对话、团队分工 |
| **E** | 情绪反应 | 误会、犹豫、被理解、成长 |
| **L** | 线索 | 公平推理所需 **可见** 证据 |
| **P** | 原理 | 科学现象或验证过程 |
| **H** | 钩子 | 翻页、篇尾、卷尾轻悬念 |

**示例**：S+L 全景 — 既交代教室，又让孩子发现窗边椅背与窗框关系（湿椅子范式）。

## 3. 配图优先级（P0 / P1 / P2）

**原则**：每一个 **不可仅依赖文字完成理解** 的重要 **视觉节点**，必须配置分镜。

**不是**每一次说话、转头、查资料、走路、微表情变化都需要独立插图。

### P0 · 必须画

| 节点 |
|------|
| 谜题 **首次出现** |
| 公平推理所需的 **关键视觉线索** |
| **空间/路线** 关系（Grabenstein · 珣能力） |
| **核心科学现象** |
| **谜底还原** + **人的原因**（C04） |
| 人物关系 **重大转折**（如正式入社） |
| 每案 **陆瑆日记页**（B 轨 · 全卷配额见案⑤） |
| 每案 **≥1 机制 SUM**（P 轨 · Q2=A） |

### P1 · 建议画

| 节点 |
|------|
| 第一误判 |
| 社团协作 |
| 重要情绪反应（见 §5 三级表情） |
| 篇尾趣事 **L1/L3 尾钩** |

### P2 · 按版面决定

| 节点 |
|------|
| 普通过渡 |
| 重复讨论 |
| 文字已足够清楚的动作 |

**禁止**：为凑张数重复 P2；为省预算跳过 P0 线索或 C04。

## 4. 景别与机位（有意识 · 非炫技）

### 4.1 景别

| 代码 | 名称 | 典型用途 |
|------|------|----------|
| EWS/WS | 大全景/全景 | **S** · 教室与路线 · 制度空间 |
| MS | 中景 | **C** · 社团协作 |
| MCU | 中近景 | **E/C** · 人物反应 |
| CU | 特写 | **L** · 线索主体 |
| ECU | 微距 | **L/P** · 见 §7 限制 |

**要求**：同一案内 **≥2 种景别**；L 帧 **≥CU**；禁止全案单一 MS。

**陆珣专项**：全景交空间 · 略俯 MS 交动线 · CU 交线索 · 观察态重心降低、眼神聚焦（非阴郁退缩）。

### 4.2 机位与构图

平视默认 · 略俯（下駄箱/路线）· 略仰（壁报制度物 · 禁压迫）· 引导线 · 框中框 · 三分 · 深浅景 · **吉竹式留白**。

### 4.3 构图标准（取代「冲击力唯一」）

构图须 **同时** 满足：

1. **叙事清晰**  
2. **线索公平**（揭晓前已可见）  
3. **角色识别**（符号+轻漫画化）  
4. **情绪表达**（三级表情 · §5）  
5. **视觉美感**（温暖观察气质）  
6. **翻页节奏**（对开页有变化 · 高密度后 **空镜/呼吸**）

**电影镜头语言是实现以上目标的方法，不是独立目的。**

## 5. 表情系统（三级 · 强制）

**不是禁止夸张**，而是 **夸张程度符合人物性格与叙事层级**。

| 层级 | 适用 | 标准 |
|------|------|------|
| **A · 正常叙事** | 正文主镜 | 自然克制 · **轻漫画化可读** · 印缩小仍辨 |
| **B · 喜剧反应** | 光冲动/志郎查错/社团轻喜 | 眉眼、嘴形、姿态 **适度夸张** |
| **C · 瑆日记** | B 轨贴纸层 | 明显符号化、拟人化、可画错 |

**禁止**：纯写实弱表情（书页上看不出）· 恐怖变形 · 嘲笑式 caricature。

## 6. 角色动作与姿态（分镜服道化 · 对齐 P0）

> 脸型、主色、道具 **以** [`00_插画师视觉创作说明书_V1.0.md`](../../05_视觉设定/00_插画师视觉创作说明书_V1.0.md) **为准**。本节为 **分镜层** 动作禁令与推荐。

### 加藤慧美

| 用 | 禁 |
|----|-----|
| 身体微微前倾 · 开放倾听 | 推眼镜+双手交叉胸前（封闭/挑剔） |
| 一手采访本 · 笔尖重要处短暂停 | 论文腔冷脸 |
| 微侧身 · 先看对方再看周围反应 | 165 cm 高校体态 |

### 中谷琦

| 用 | 禁 |
|----|-----|
| 橄榄绿工装 · **挂颈耳机** · 斜挎包 · 交通卡/街区小物 | **相机** 作核心符号（除非 IP 另立项） |
| 半眯眼 · 靠墙/校界 · 一手插兜 | 不良少年符号 · 抢 C 位 |

### 陆瑆

| 用 | 禁 |
|----|-----|
| 好奇 · 眼亮 · 听懂/误解时 **表情明显** | 固化「安静微笑型」 |

### 陆珣

| 用 | 禁 |
|----|-----|
| 安静但 **非无表情** · 肩自然 | 微驼+视线向地=阴郁/社交退缩 |

### 山本理紗（Vol1 非具名/F1）

| 用 | 禁 |
|----|-----|

## 7. 科学微距（ECU）使用限制

微距 **仅在** 下列 **同时成立** 时使用：

1. 该细节是破解谜题 **不可替代** 的证据；  
2. 10–12 岁可通过图示理解；  
3. 不依赖专业设备才可能被角色发现；  
4. 不让故事变成科普图鉴；  
5. **经科学审校** 确认准确。

**否则改用**：简化 P 轨图示 · 志郎验证卡 · 瑆日记 · 放大镜观察 · 前后对比图。

**禁止默认**：真菌菌丝、水珠接触角等 **过度学术** 微距。

## 8. 场景连续性与科学准确性（验收硬标准）

| 项 | 要求 |
|----|------|
| **空间连续** | 同一教室/走廊的门窗、课桌、光源方向 **跨 Shot 一致** |
| **科学准确** | 现象可验证 · 不得「看着像」 · 须过科学审校单 |
| **季节** | 名古屋 4 月链 · 禁冬雪/盛夏漂移 |
| **制度** | 上履き/引き戸/壁报流程 · startup gate |

**E06-S Fail**：同案前后窗位矛盾 · 线索与正文机制不符。

## 9. 双层画风（正文 vs 瑆日记）

| 层 | 用途 | 要求 |
|----|------|------|
| **正文层** | A/C 轨场景 | 稳定比例 · 清晰空间 · 轻漫画化 · 真实材质 · 电影化构图 · **线索公平** · 可承担 L/P |
| **瑆日记层** | B 轨（案⑤卷级+它案可选） | 更圆线条 · 简单色块 · 手写圈注 · **可画错** · 物品拟人 · 幽默反差 · **不承担唯一关键证据** |

二者 **一眼可区分** · 角色 **核心识别符号不变**。

## 12. Vol1 预算区间（128–160p · 5 案 · 非固定张数）

**最终数量以各案分镜规划书 + V1.0 正文核定为准**，不得提前锁死「每案 N 张」。

| 类型 | 建议数量 |
|------|----------|
| 跨页/整页主插图 | 8–12 |
| 半页/局部叙事插图 | 12–20 |
| 线索特写与小图 | 15–25 |
| 陆瑆日记页 | 5 |
| 地图/关系/道具图 | 3–6 |

**合计概念**：约 **25–38 个主分镜单元** + **15–25 个线索/图标小单元**（含版式角标、inset，非 50 张独立大画）。

案① 当前 **6 帧深度锚点** = P0 节点 **工作样张**，全卷总数 **待五案规划书齐后汇总**。
