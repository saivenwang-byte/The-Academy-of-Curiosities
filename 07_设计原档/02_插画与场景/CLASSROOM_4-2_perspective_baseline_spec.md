## 三、Vol1 机制几何（v1.2 · 禁椅背叙事）

```
[校庭] ← ガラス ← アルミサッシ枠
                    ↑ 接触（隙間なし = 光の椅のみ）
              窓側金属パイプ ──→ 座面（结露传导）
```

| 验收 | 要求 |
|------|------|
| 湿痕 | 仅光座面 · 小 · 局部 |
| 邻席 | 山田（R5C6）及 C6 列他席 **金属有空隙** → 干 |
| 地面 | **干** · 排除泼水 |
| 时间 | 朝湿 · 昼干 |
| 修正 | 椅外移一拳 · 金属离枠 8–10 cm |
