## 0. 定义

**趣事尾钩**：每一篇校园趣事在 **真相解决、温柔收束之后**，用一件小物、一句话、一张投稿、一幅陆瑆误画或一个画面细节，**轻轻引出下一篇**。

| 是 | 不是 |
|----|------|
| 讲完一个趣事，发现 **下一个趣事正在冒头** | 主谜题没讲完，逼读下一篇 |
| **好奇心** | **焦虑感** |
| 观察社自然动作（投稿/壁报/采访栏） | 「真正的犯人还没出现」 |

**一句话**：结尾不是「未完待续」，而是 **「下一件趣事已经在旁边悄悄发生了」**。

---

## 1. 铁律

1. **主谜题必须完整解决**（科学谜底 + 人际真相）。  
2. **尾钩只开新疑问**，不得用尾钩承载本案核心答案。  
3. **公平线索**：尾钩细节须在本案正文或画中 **已可见或可回溯**（禁彩蛋独有线索）。  
4. **禁 B4 禁用语**：错过 · 不够聪明 · 必须读完 · 恐吓 cliffhanger。  
5. **Vol11 前**：理紗尾钩 **非具名、无正脸**（F1 §3）。  
6. **传播案**：允许 **无源收束** · 禁逼「找到造谣者」才结案 → 引擎 **§1b**。

---

## 2. 每案结尾 · 五步（LOCKED）

| 步 | 内容 |
|----|------|
| 1 | 真相解释清楚 |
| 2 | 被误会的人得到温柔处理 |
| 3 | 观察社完成趣事稿（壁报/投稿/观察栏） |
| 4 | 陆瑆写日记 / 画插图 |
| 5 | **趣事尾钩**（轻 · 开新疑问） |

---

## 3. 四级尾钩

| 等级 | 名称 | 用途 | 强度 |
|------|------|------|------|
| **L1** | 篇尾轻钩 | 卷内 A(n) → A(n+1) | 最轻 |
| **L2** | 周末复盘钩 | 周日纪念页末 · 下一周预告 | 轻 |
| **L3** | 卷尾彩蛋 | 全卷读后 · 引下一卷 | 中（禁恐怖） |
| **L4** | 季尾长线钩 | Vol4/8/12/16/20 等 · L01–L08 | 中强 · 仍须温柔 |

**频度建议**：L1 **每案必有** · L3 **每卷 1 次** · L4 **每季 1 次** · L2 与周末复盘页绑定。

---

## 4. 五类推荐形式（固定菜单）

| 类型 | 示例 |
|------|------|
| **投稿钩** | 投稿箱多一张无署名纸条 |
| **道具钩** | 本案小物出现在不该在的位置 |
| **瑆误画钩** | 妹妹画错一笔，却指向关键细节 |
| **中谷一句钩** | 邻居前辈路过，半句奇怪的话 |
| **理紗影子钩** | 不具名者扶正书架/色条/bookmark |

**观察社外壳**：空栏 · 待核实 · 下一期标题 · 采访回执 — 优先于「侦探预告」。

---

## 5. 与日节拍 · 尾钩位置

| 时间 | 推进 | 尾钩/悬念 |
|------|------|-----------|
| 周一 | 小怪事出现 | 当日末：「哪里不对？」 |
| 周二 | 采访·线索 | 错误方向（非本案结论） |
| 周三 | 第一次误判 | 关键物件露面 |
| 周四 | 珣还原 + 慧美情绪 | 真相前一秒 |
| **周五** | 真相 + 瑆日记 | **L1 篇尾轻钩 → 下一案** |
| **周末** | 复盘页 | **L2 下一周预告** |

卷末另设 **L3 卷尾彩蛋**（2–4 页 · 合订/壁报/可选短 epilogue）。

---

## 6. 禁用钩子

| 禁用 | 原因 |
|------|------|
| 有人失踪 / 危险 | 过重 |
| 「真正犯人还没出现」 | 犯罪感 |
| 「下篇揭开惊天秘密」 | 成人悬疑腔 |
| 每篇强行反转 | 疲劳 |
| 恐惧驱动续读 | 家庭/学校场景不适合 |
| **主谜题不解决** | 满足感不足 |

---

## 7. 卷间 · 卷末彩蛋（L3）公式

```
已闭合的情感（本卷人物弧）
+ 未闭合的「系列趣事线」（可跨卷，但须有本案内线索）
+ 观察社动作（壁报「第 N 起」/ 待续栏）
= 卷尾彩蛋 → 下卷冷启动 3–5 页内接住
```

**湿椅子级素材**：若跨卷，须 **本案机制已解释**；跨卷只追 **行为链/人际层**，禁「机制下卷再说」。
