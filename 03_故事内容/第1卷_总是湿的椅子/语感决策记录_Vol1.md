## 二、第一版更好的地方（必须保留进正典）

### 语言 · 克制高级的文学语感

- 短句、留白、不解释、不科普堆砌  
- 少年式沉默观察，词少意多  

**金句型（Vol1 必保留密度 ≥3 处/卷）**：

- 「怪事可以被重复。重复，就不叫怪。」  
- 「窗框还是凉的，四月还是四月。」  
- 「观察这件事，也许不需要很多词。词多了，反而像掩饰。」  

### 节奏 · 电影剪辑

- 开场直接入谜题，无长铺垫  
- 情绪收束快、稳  

### 气质 · 安静少年文学

- 清冷、克制、孤独、温柔、理性  
- 四月微凉、樱瓣、安静走廊、少年心事  

### 陸珣 · 高级文学男主

- 话极少、观察极准、沉默、神秘、有距离感  
- **全卷对外台词 ≤5 句**（含「窓枠に当たってる」）

### 立意 · 有回味

- 怪事不是鬼，是冷与湿的碰面  
- 观察即力量；孤独与融入

## 四、第二版问题（第三版必须修）

| 问题 | 表现 | 修法 |
|------|------|------|
| 说明文化 | 制度/科学「解释出来」 | **展示，不解说**（动作嵌入 1 个名词即可） |
| 节奏慢 | 晨会/仪式/科普段过密 | 再瘦 10%；删「后来想」类旁白 |
| 陸珣太乖 | 阳光转学生 | 压台词、加内心留白 |
| 原理重复讲 | 黑板+对话+硬壳本三遍 | **正文 1 句串联** + 硬壳本表 + 实验页 |
| 绘本体 | 热闹、直白、可爱 | 叙事层回归第一版句长与留白 |

## 五、Hybrid Voice 铁律（Vol2+ 强制执行）

### 三层分离

```
叙事层（80%）  → 第一版：短句、留白、陸珣内心
正典层（嵌入）  → 第二版：日式名词在动作里，不解说
产品层（附录）  → Case Card / 实验 / 硬壳本表 / scorecard
```

### 写作顺序

1. **先文学稿**（不管细节满不满）  
2. **正典注入**（只改动作层名词）  
3. **L1 门禁**  
4. **文学回切**（删说明句、加金句、压陸珣台词）  
5. **L2 + §7 文学维度评分**

### 陸珣声线卡

| 项 | 标准 |
|----|------|
| 对外台词 | ≤5 句/卷 |
| 内心 | 短句、不结论、可诗性 |
| 禁止 | 「后来想」「像…一样」连续 3 句以上 |

### 科学发声点（正文 + 双层 · v4）

1. 误导推翻后：陸珣观察句  
2. 验证成功：慧美 **一句** 三原理串联（无术语展开）  
3. **陸瑆笔记**：原理交叉表 + 妹妹口吻总结（主科普层）  
4. 光的硬壳本 / 实验页：现象·线索·验证·处理（无长原理段）
