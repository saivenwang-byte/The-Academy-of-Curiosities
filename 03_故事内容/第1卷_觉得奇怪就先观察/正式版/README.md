## 重建展示版

```powershell
cd "03_故事内容\第1卷_觉得奇怪就先观察\正式版"

# 1. 生成/更新 V-S02–V-S05（v0.9 程序化成图）
python 02_插画/tools/generate_vol1_illustrations.py

# 2. 日本語 PDF（默认）
python 04_展示版/tools/build_display_edition.py
```

### 出版社 / 专家完整成果（推荐对外）

python 05_出版成果/tools/build_expert_publication_pack.py
