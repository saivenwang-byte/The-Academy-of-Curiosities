## 一键生成

```powershell
cd "03_故事内容\第1卷_觉得奇怪就先观察\正式版"
python 05_出版成果/tools/build_expert_publication_pack.py
```

## 输出文件

| 文件 | 用途 |
|------|------|
| `学堂奇事録_Vol1_正式出版成果_日本語_YYYYMMDD.pdf` | **主交付** · 封面+门禁+定位+Case表+全文日+笔记+挿画+专家附录 |
| `学堂奇事録_Vol1_出版提案_YYYYMMDD.pptx` | 出版社提案 **16:9 宽屏** deck |
| `出版成果_门禁对照表_YYYYMMDD.md` | Agent/人审勾选对照 |
| 同名 `.html` | 浏览器重印 PDF |

---

## PDF 结构

1. 封面（品牌 · 謎が、解ける日 · V-S01）
2. 出版成果声明（非卖品）
3. 正典·专家门禁对照表
4. IP 定位摘要（B1/E1）
5. Case Card 五案一览
6. 登場人物
7. 目次
8. **全文日本語**（序+事件①–⑤ · 挿画嵌入）
9. 陸瑆ノート
10. 挿画一覧 V-S01–V-S05
11. 专家审稿指引
