## V0.1-A · 第一话审阅包（序 + A001）

**范围说明**：日文主版 + 中文附录；仅含**序 + 案01（翘边的海报 / A001）**，**不含**「总是湿的椅子」卷内容。

### 一键生成

```powershell
cd 03_故事内容/第1卷_觉得奇怪就先观察/正式版
python 05_出版成果/tools/build_expert_publication_pack.py --mode v01a
# 旧入口 build_v01a_episode1_review_pack.py 已并入主脚本
```

#### 输出文件

| 文件 | 用途 |
|------|------|
| `学堂奇事錄_Vol1_V0.1-A_第一話審閱包_日本語_YYYYMMDD.pdf` | **主交付** · 序 + A001 审阅 PDF |
| `学堂奇事錄_Vol1_V0.1-A_出版提案_YYYYMMDD.pptx` | 出版社提案 **16:9** deck（V0.1-A 范围） |
| `V01A_第一話審閱包_门禁对照表_YYYYMMDD.md` | Agent/人审勾选对照 |
| 同名 `.html` | 浏览器重印 PDF |
