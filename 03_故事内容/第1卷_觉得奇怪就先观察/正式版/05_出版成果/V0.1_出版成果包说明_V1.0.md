## 审稿工作流

1. **分发**：按受众选 A 或 B 包 PDF + 对应 PPT
2. **四栏填写**：文字(E04) / 科学(P0-04) / 插画(V13) / 日本文化(E07)
3. **门禁 MD**：Agent 预填 + 人审勾选 `V01A_*` 或 `V01B_*` 对照表
4. **汇总**：意见 → V0.2 迭代（L0 v1.0 精修 · 全文 E04 · E07 PASS）

### Gate A 不可变项

- Vol1 = **Plan B**（A001–A005）
- A001 = **めくれたポスター**（非湿椅子 C001）
- 首市场 = 日本 · 品牌 = 学堂奇事録
- 珣=しゅん / 瑆=ひかる
