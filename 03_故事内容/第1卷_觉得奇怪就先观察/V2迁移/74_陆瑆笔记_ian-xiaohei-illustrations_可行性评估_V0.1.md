### 1.2 项目自述（GitHub README · 2026-05-28 v1.0.0）

**定位**：Codex / Cursor **Agent Skill**（非 prompt 包、非 LoRA、非画风模型权重）。

**工作流**：

1. 读取中文文章 / Markdown / Notion / 截图
2. 提炼认知锚点（判断、流程、状态、隐喻）
3. 输出 **4–8 张 shot list**（主题 · 结构类型 · 小黑动作 · 中文标注建议）
4. 结构类型：Workflow · 系统局部 · 前后对比 · 角色状态 · 概念隐喻 · 方法分层 · 地图路线 · 小漫画分镜
5. 每张图单独调用图像模型 → PNG → `assets/<slug>-illustrations/`
6. QA：白底 · 留白 · 小黑参与核心动作 · 非 PPT 感 · 非旧案例复刻

**风格关键词**：

| 维度 | 关键词 |
|------|--------|
| 画幅 | **16:9 横版** 正文配图 |
| 背景 | **纯白** · 禁纸纹/米色/阴影/渐变 |
| 线稿 | 黑色细线 · 轻微抖动 |
| 构图 | 大量留白 · 主体占 40–60% |
| 批注 | 少量 **红/橙/蓝** 中文手写 |
| 角色 | **「小黑」** — 黑色实心体 · 白点眼 · 细腿 · 空表情 · 怪诞工作者 |
| 气质 | 怪诞 · 清爽 · **不幼稚 · 不卖萌** |

**官方「不适合」清单**（README 原文）：

- 商业插画 / 品牌 KV
- PPT 信息图 / 复杂架构图
- **儿童卡通 · 可爱 IP · 表情包风格** ← 与陆瑆 B 轨直接冲突
- 把大段正文塞进一张图
- 可编辑矢量源文件

**安装**：复制 `ian-xiaohei-illustrations/` 子目录至 `~/.codex/skills/` 或 `.cursor/skills/`；触发 `Use $ian-xiaohei-illustrations …`

## 2. ian-xiaohei-illustrations 是什么

| 问题 | 答案 |
|------|------|
| Cursor skill？ | ✅ 是 · 标准 `SKILL.md` + `references/` + QA checklist |
| Prompt pack？ | 部分 · 内含 `prompt-template.md` · 但捆绑完整 agent 工作流 |
| 插画风格？ | ✅ **Ian 个人「小黑怪诞正文配图」视觉 IP** · 非通用「手绘笔记」 |
| 与「手绘笔记」关系 | 视频标题泛称「手绘」· 项目实为 **成人向中文长文编辑配图** · 非儿童日记/手账 |

**本质**：把抽象方法论 **视觉化** 的生产 skill；主角固定为 **小黑**，不是可替换的 generic doodle。

## 3. 与项目陆瑆笔记设定对照

### 3.1 陆瑆笔记 · 正典功能（文字层）

来源：`01_角色设定/06陆瑆.txt` · `03_笔记/陆瑆笔记_Vol1.md` · academy-engine

| 维度 | 正典 |
|------|------|
| **叙事位** | 双层叙事 **原理层** · 每卷结尾第一人称笔记 |
| **篇幅** | 3–5 句 · 一观察 · 一句留白 · **禁超 8 句** |
| **年级** | **4年2組** · **非观察社** · 不出 A 轨主帧 |
| **内容** | 原理图解 · 箭头图表 · 对伙伴观察 · 问句收束（吉竹体） |
| **Vol1 配额** | 全卷 **1 篇总笔记**（案⑤后）+ 样章级案①页 |

### 3.2 陆瑆笔记 · B 轨视觉正典

来源：`05_视觉设定/00_插画师视觉创作说明书_V1.0.md` §1.2B · §10 · `02_插画创作规范手册_V1.0.md` §6.2

| 维度 | 正典 B 轨 |
|------|-----------|
| **用途** | 篇末日记 · 手账页 · 贴纸页 · 表情包延展 |
| **底纸** | 微黄横线/方格 **`#F5EDE0`** · 手账拼贴 · **非纯白** |
| **线条/色** | 暖棕 `#2A1810` · 粉橙主色 · 水彩/彩铅感 · 低–中饱和 |
| **比例** | 可 **二头身 Q 版**（**仅此轨**）· 更圆更简 |
| **元素** | 贴纸 · 胶带 · 便签 · 拍立得小插图 · 修正痕 |
| **拟人** | 椅子/海报/书本等 **圆圆小物** · 无恐怖 |
| **角色** | **陸瑆本人** · A5 牛皮纸线圈本 · 彩铅 · 银色自动铅笔 |
| **商品化** | 2–3 cm 贴纸 · 书签 · 小图标 |

### 3.3 契合点

| # | 契合 | 说明 |
|---|------|------|
| C1 | **一图一结构** | 小黑 skill「每张图只表达一个核心动作/结构」≈ 陆瑆笔记「一个观察 + 简笔图解」 |
| C2 | **shot list 前置** | 从笔记正文先列「画什么」再出图 · 可接入 B 轨分镜门禁 |
| C3 | **箭头/流程/对比** | 结构类型（前后对比、路径、系统局部）与案①「风→角翘」类机制图同型 |
| C4 | **手绘非矢量** | 与项目「禁完美对称/软件渐变」方向一致 |
| C5 | **Agent 可复用** | 与 Vol2 迁移「正文后 → B 轨插图」流水线形态兼容（方法论层） |

### 3.4 冲突点

| # | 冲突 | 严重度 | 说明 |
|---|------|--------|------|
| X1 | **视觉 IP 外来** | 🔴 P0 | 「小黑」≠ 陸瑆 · 引入第三方识别角色 · 破坏「每个孩子是一种观察方法」 |
| X2 | **背景/色温** | 🔴 P0 | 纯白 vs `#F5EDE0` 横线本 · 黑白 vs 粉橙+暖棕 · **不可直接套 style DNA** |
| X3 | **受众气质** | 🔴 P0 | 项目 README 明确排除「儿童卡通/可爱 IP」· 陆瑆 B 轨恰恰需要 **儿童手账+贴纸化** |
| X4 | **画幅/版式** | 🟡 P1 | 16:9 横版正文配图 vs 日记 spread / 竖版网格本 / E22 横排手写页 |
| X5 | **语言/正典** | 🔴 P0 | skill 默认 **简体中文批注入图** · A 轨 KF-LOCK-J 禁简中 · 日版笔记须假名/汉字表记正典 |
| X6 | **角色出镜** | 🟡 P1 | 小黑必须承担核心动作 · 陆瑆页应以 **瑆的手+本+简笔物件** 为主 · 非空心怪诞人形 |
| X7 | **名称漂移** | 🔴 P0 | 历史已多次 **陆瑆/陆珣/5年2组** 入图 · 外来 skill 易复制 CN 标注习惯 |
| X8 | **商品/IP 授权** | 🟡 P1 | 小黑 MIT 开源 · 但 **不可与《学堂趣事录》陸瑆 IP 混用** 作系列贴纸 |

### 3.5 对照矩阵（一句话）

> **ian-xiaohei = 成人中文长文的怪诞系统配图；陆瑆笔记 = 四年级女孩暖色手账里的科学小图解。**  
> 前者借「小黑」讲抽象判断；后者借「瑆的本子」讲案件原理。**工具链可借鉴，视觉语言不可移植。**

## 4. 建议

### 4.1 Verdict：**partial adopt**

| 层级 | 决策 |
|------|------|
| **整包安装为项目 B 轨 skill** | ❌ **reject** |
| **小黑风格 / 16:9 白底 / 中文批注模板** | ❌ **reject** |
| **shot list · 认知锚点 · 一图一结构 · QA 清单** | ✅ **adopt（方法论）** |
| **项目自有 B 轨 prompt 模块** | ✅ **建议新建**（见 §4.3） |

### 4.2 为何不 adopt 整包

1. **风格 DNA 与 B 轨 SSOT 相反**（白/黑/编辑 vs 黄/粉/手账）。
2. **固定 IP「小黑」** 与陸瑆角色资产冲突，且官方声明不做儿童/可爱 IP。
3. **Vol1 插画已第三次跑偏**（doc72）· 引入外来 visual IP 会加剧 drift。
4. **KF-LOCK-J + 十人名称 LOCK** · 外来 skill 的 CN 入图习惯与门禁相悖。

### 4.3 若 partial adopt · 具体集成路径

**原则**：只借 **流程**，不借 **画风**；B 轨 only；A 轨零接触。

| 步骤 | 动作 | 产出 |
|------|------|------|
| **S1** | **不** 将 `ian-xiaohei-illustrations/` 装入 `.cursor/skills/` 为默认 | — |
| **S2** | 在 `academy-illustration-pipeline` 或新建 **`academy-hikaru-diary-b-track`** 增加 §「B 轨 shot list」 | 从 `陆瑆笔记_VolN.md` 输出：锚点 · 结构类型 · 瑆简笔动作 · **无文字或日文标注建议** |
| **S3** | 写 **`references/hikaru-diary-style-dna.md`**（项目内） | 底纸 `#F5EDE0` · 暖棕线 · 粉橙点缀 · 拟人小物 · 禁小黑/禁纯白/禁16:9强裁 |
| **S4** | 复用 ian 的 QA 结构，改 checklist | 横线本？瑆道具？一机制一图？KF-LOCK-J？名称正典？ |
| **S5** | 首测 **案①样张页**（风→海报角翘）· 对比现有 ref `assets/…52914b88…` | IP Owner 签后再卷末总页 |
| **S6** | 可选：内部阅读 ian 的 `composition-patterns.md` 作 **结构灵感库** · 标注 `REFERENCE ONLY · NOT STYLE` |

**B 轨 scrapbook only？** ✅ 是。仅 `03_笔记/` · `.diary-page` · 卷末 spread / 角标 inset。**不得** 进入 DA 场景帧、L0、封面、PRODUCT batch。

**Prompt module 示例骨架**（非 ian 原文）：

```text
[B-TRACK · 陸瑆日记 · Vol1 案①]
底纸: #F5EDE0 横线本 · 暖棕线 #2A1810 · 粉橙点缀
主体: 简笔海报+风箭头 · 可二头身拟人风 · 陸瑆的手+线圈本边缘入画
禁: 小黑 · 纯白底 · 简中大字 · 5年2组 · 陆瑆/陆珣混名 · Q版入主帧
一图一机制: 风从上方吹 → 左角翘起
```

### 4.4 红线（执行时必须挂门禁）

| 红线 | 要求 |
|------|------|
| **A 轨隔离** | ian 工作流 **不得** 用于 SC/DA/PRODUCT 正篇帧 |
| **KF-LOCK-J** | B 轨若含文字 · CN 版控字数 · JP 版用 **`陸 瑆` / 4年2組** 表记 · 禁简体乱入 |
| **名称正典** | 禁 **陆瑆** 入 A 轨 tag · B 轨署名 **瑆** 或 **陸 瑆** 依语种 SSOT |
| **IP** | 禁 **小黑** 形象出现在任何《学堂趣事录》交付物 |
| **配额** | Vol1 全卷总笔记 **1 页** · 它案角标可 · 禁 shot list 扩成 4–8 全页 |
