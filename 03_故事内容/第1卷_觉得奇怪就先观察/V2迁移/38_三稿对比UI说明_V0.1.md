## 2. 打开方式

### 方式 A：双击启动器（推荐）

同目录 **`打开对比UI.bat`**（或 PowerShell 运行 **`打开对比UI.ps1`**）：

- 自动启动 `python -m http.server 8765`
- 自动打开浏览器 → **http://localhost:8765/**
- 规避 Windows `file://` + 中文/括号路径无法加载 `comparison_data.js` 的问题

### 方式 B：file:// 直接打开

数据已内嵌于 `index.html`（由 `build_comparison_ui_data.py` 写入 `<!-- COMPARISON_DATA_EMBED -->` 区块）。

在浏览器地址栏打开：

```
file:///D:/【AI Project】/【The Academy of Curiosities】/03_故事内容/第1卷_觉得奇怪就先观察/正式版/06_审读对比UI/index.html
```

> 若仍显示「未能加载对比数据」，请改用方式 A，或重跑数据脚本后再开。

### 可选：手动 HTTP

```powershell
cd "03_故事内容/第1卷_觉得奇怪就先观察/正式版/06_审读对比UI"
python -m http.server 8765
# 浏览器 http://localhost:8765
```

## 3. 重新生成数据

正文或打分更新后，在仓库根目录执行：

```powershell
python "03_故事内容/tools/build_comparison_ui_data.py"
```

输出：

| 文件 | 用途 |
|------|------|
| `06_审读对比UI/comparison_data.json` | 机器可读 SSOT · diff/段落/分数 |
| `06_审读对比UI/comparison_data.js` | HTTP 模式备用加载 |
| `06_审读对比UI/index.html` | UI 入口 · 含内嵌数据（file://） |
| `06_审读对比UI/打开对比UI.bat` | Windows 一键启动（推荐） |

数据来源：

- 三稿正文：`git show` @ `050f810` / `a374cb7` / `b0adc91`
- 打分：`V2迁移/scores_mvp_latest.json`
- 修改摘要：`*_R16_globalIP.md` + `37_A001_R16_原稿对比表_V0.1.md`（A001）

## 4. UI 功能

1. **案 Tab** A001–A005
2. **三列并排**：原稿 | 修改1稿 | 修改2稿 · 可同步滚动
3. **打分面板**（可折叠）：文学卷专家、机制/科学、H/C/S before→after、读者群、E20 探针占位
4. **修改内容面板**：R16 维度表 + M1–M4 标签
5. **段落高亮**：改1 vs 原稿、改2 vs 改1 差异段
