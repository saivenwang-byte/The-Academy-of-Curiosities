# 0. 单元定义

## 0.1 一句话故事

新学期社团介绍会前的一周，海报、泥脚印、壁报空栏、粉笔灰和活动名单接连发生五件怪事；每一件都像在证明活动委员水野直人故意破坏活动。全校都在等待他承认，观察社却必须在活动开始前，把五件真实的怪事从一个错误的故事中拆开。

## 0.2 对儿童的钩子

> **五件怪事都是真的。可它们真的都是同一个人做的吗？**

## 0.3 对家长的价值

> 孩子在故事中学习：痕迹是真的，不等于我们对痕迹的解释就是真的；洗清不属于自己的责任，也要承担真正属于自己的责任。

## 0.4 单元总宣言

> **怪事不会伤人，误会会。**

## 0.5 卷级元问题

> **痕迹是真实的，责任判断就一定正确吗？**

# 1. 时间、场景与大事件

## 1.1 时间

- 名古屋，4月，新学期第二周；
- 从周一早晨至周五下午；
- 学校正在准备「高学年クラブ活動紹介会」（工作称，须日本学校文化顾问确认正式制度名称）；
- 周五15:00活动正式开始。

## 1.2 主要空间

- 五年级侧廊与社团介绍海报墙；
- 鞋柜区与雨天侧门；
- 校内壁报准备室；
- 五年级教室黑板与讲台；
- 体育馆后台、活动名单板和物资准备室。

## 1.3 卷级压力场

全校正在争取活动顺利举行。老师、委员、班级和社团都有正当目标，但时间越紧，大家越倾向接受一个简单解释：

> 「一定有人故意不想让活动举行。」

## 1.4 卷级被指向者

**水野直人（みずの なおと，暂名，待日文母语与角色编辑确认）**

- 五年级活动执行委员；
- 做事认真，但害怕公开讲话；
- 曾因临时修改活动内容与同学争执；
- 经常出现在海报、名单、广播与物资附近；
- 五件怪事发生后，被同学逐渐编成「幕后破坏者」；
- 他确实隐瞒过几件事，但没有策划五案。

**错误责任**：故意制造五件怪事，破坏社团介绍会。

**真实责任**：为了掩盖自己的紧张和一次登记错误，他擅自用可擦笔修改正式名单，并隐瞒了自己接触过文件与物资。

# 3. 五案关系网

## 3.1 共同压力场 `pressure_field`

五案都发生在社团介绍会截止期限内。每个人都想保护活动，于是更愿意迅速找到一个责任人。

## 3.2 信息污染 `information_pollution`

A001 的「不要参加」→ A002 脚印 → A003 空栏 → A004 粉笔圈，四条传言逐渐拼成：

> 水野一直在秘密破坏活动，观察社在替他掩盖。

## 3.3 物件传递 `carrier`

同一个临时文具箱穿过五案：胶带、旧编号纸、塑膜姓名条、可擦笔、临时标签。

## 3.4 拼图关系 `puzzle_piece`

- A001：文字层与材料变化；
- A002：位置与真实路线；
- A003：事件、记录、版本；
- A004：自然图案与人为意图；
- A005：材料、路线、版本、图案、时间重组。

## 3.5 角色换位 `role_exchange`

水野 → 光 → 慧美 → 志郎 → 观察社整体；陸珣在 A005 公开站出。

# 4. 卷级学科编织

> 单案用一个学科解释怪事；整卷用学科之间的关系证明，五件真实痕迹可以被编成一个错误故事。

| 案 | 主机制 | 交叉机制 | 社会放大器 | 学科命题 |
|:--:|--------|----------|------------|----------|
| A001 | 空气流动、湿度、黏着 | 视觉遮挡 | 报名截止 | 看见一句话 ≠ 完整信息 |
| A002 | 摩擦、转移痕迹 | 几何路线、时间窗 | 为朋友保密 | 痕迹位置 ≠ 人的路线 |
| A003 | 信息版本、版面 | 记忆重构 | 群体转述 | 大家都记得 ≠ 同一事实 |
| A004 | 静电、粉尘 | 塑膜、清扫 | 班级投票 | 图案真实 ≠ 人为意图 |
| A005 | 热致变色、温度 | 文件版本、运输 | 活动取消 | 多条真痕迹 → 假故事 |

**知识接力**：A001 材料改变文字呈现 → A005 温度改变墨水；A002 路线 → 物资追踪；A003 版本 → 名单分叉；A004 塑膜与可擦笔 → A005 物证。

**错误迁移**（≥2）：A002 路线成功 → A005 过度相信「谁搬箱谁负责」；A004 静电 → A005 误推名字消失也是静电。

# 5. 角色弧线

## 陸珣

序章旁观 → A001 见观察社不 rush 定罪 → A002 路线图 → A003 理解记录误解 → A004 见实验改变现场 → A005 公开「这还不能证明是他」→ 入社。

## 伊藤光 / 加藤慧美 / 松本志郎

光：快问也快传 · A002 被疑；慧美：核实与沉默 · A003 学「谨慎≠不解释」；志郎：实验改现场 · A004 承担真实责任。

## 水野直人

非反派 · 怯懦+参与渴望 · 最终承认擅改与隐瞒，拒绝承认未做的蓄意破坏。

## 陸瑆

> 哥哥说，他只是把五件怪事一件一件拆开。  
> 可我觉得，他真正做的，是把一个人从五件怪事里拆了出来。
