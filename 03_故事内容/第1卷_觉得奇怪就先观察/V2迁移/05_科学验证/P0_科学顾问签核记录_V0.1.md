## A001 · 广播链 · 旧排练录音误播

### 机制陈述（可检验）

| 项 | 内容 |
|----|------|
| 现象 | 全校广播播放似「伊藤光」声线的一句「水野根本不该参加」 |
| 机制 | 数字音频文件 **误触播放** + **有损压缩** + **非连续跳播/硬切** → 完整句中间段被播出 |
| 完整句（工作稿） | 「……**迟到者**不该参加 **今天的彩排**。」 |
| 播出段 | 「水野根本不该参加。」（语境截断 + 跳切） |
| 公平线索 | FC-1 唇未动 · FC-2 文件时间旧 · FC-3 波形硬切 · FC-4 完整句还原 |

### 复现条件（课堂/家庭简化）

1. 两段录音：A 段含完整句；B 段仅中间 3–4 字。  
2. 用同一说话人（或同学代读）录制，导出 **低码率 MP3**（≤64 kbps 或 school PA 等效）。  
3. 在波形编辑器中 **删除前导 0.2–0.5 s** 或 **硬切** 至中间词起播。  
4. 听众仅听 B：多数会 **指认说话人** 但 **无法还原完整句意**。

### 日本校园边界（E07 待田中并行）

- 校内广播多为 **午後の校内ニュース** / クラブ紹介彩排录音 · 非刑侦取证级设备。  
- 教师 **不得在证据不足时** 当众撤销导播资格以外的羞辱性处分 · 本稿为「暂缓取消+书面说明」✅  
- 文件名 `排练_0328.wav` 为工作稿 · 日译须改为 plausible 学校命名。

## A005 · 全景拼接 · 消失的影子

### 机制陈述（可检验）

| 项 | 内容 |
|----|------|
| 现象 | 预展大屏幕上五人合照 **人在、影不在** |
| 机制 | 手机/平板 **全景模式** 逐段扫描 · 拍摄中 **人物移动/光线变化** · 算法 **保留人体、用相邻帧地面覆盖** → 阴影时间错位 |
| 非机制 | **非** 恶意 P 图 · **非** 纯静电致影消失（A005 初期可误推 · 须实验失败） |
| 变量上限 | 跨案回收 ≤4 · 本机制为主 · 辅以时间戳/路线 |

### 复现条件（简化）

1. 平板全景拍 5–8 人横排 · 拍摄中一人 **短暂侧让** 再站回。  
2. 午后侧光 · 地面阴影方向在 10–15 s 内 **可见变化**。  
3. 对比单次快门：全景图可出现 **人体连续、地面/阴影不连续**。  
4. **禁** 向儿童展示「灵异」解释 · 只展示 **拼接算法局限**。

### 前置依赖（卷级）

- A001 文件/时间意识 · A004 器材车路线 · 全景 **非单次快门** 须在 A005 前 **公平预埋**（预告页或慧美记录卡）。
