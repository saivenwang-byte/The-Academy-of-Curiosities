## 一、单案故事引擎（写作方法 · 步骤级）

| # | 规则名称 | V1.0 具体内容 | V2.0 新增/变更的具体内容 |
|---|----------|---------------|-------------------------|
| E-01 | 引擎总步数 | **9 步**：趣事→采访→误判→(可选传播)→跨学科→分工→温柔真相→瑆笔记→(可选实验)→L1尾钩 | 🆕 **13 步**：在 Step③ 与 Step④ 之间插入 **双倒计时④**；在机制后插入 **误指峰值⑧**、**true澄清⑨**；瑆升级为 **第二真相⑪**；尾钩扩展 **relation_hook⑬** |
| E-02 | Step④ 双倒计时 | 无此步骤；压力靠情节自然出现 | 🆕 **事件钟**（具名时间+可感知变化：雨/风/张贴/彩排）与 **社交钟**（委员/壁报/导播资格/展示取消）**必须同时启动** |
| E-03 | Step⑦ 机制闭合顺序 | Step⑥「温柔人际真相」与科学谜底 **同层收束** | 🆕 **铁律**：Step⑦ **机制真相必须先闭合** → 才允许 Step⑧ 误指峰值 → Step⑨ 澄清 **不得替换** 科学谜底 |
| E-04 | Step⑧ wrong 责任峰值 | 无专门步骤；群体误指可写可不写 | 🆕 必须有 **wrong_responsibility 峰值场**：全校/班级 **错误归咎** 到具体孩子（fair-play 可反驳） |
| E-05 | Step⑨ true 责任澄清 | `human_truth` 一条温柔收束 | 🆕 拆成 **机制层**（科学已闭合）+ **动机层**（怯懦/善意/习惯）；观察社 **不写审判词** |
| E-06 | Step⑪ 瑆笔记 | `kei_note_angle`：笔记层、不参与破案 | ⬆️ **第二真相**：`kei_second_truth` 必填 · **零新机制** · Card 上写 `kei_second_truth_test` 情感验收句 |
| E-07 | Step⑬ 关系尾钩 | L1 尾钩：本案闭合 + 下案轻疑问 | ⬆️ 尾钩须标注 **`relation_type`**（六类之一）+ `relation_hook_text` 指向下案 **关系边**（不只「下一怪事」） |
| E-08 | 分场脚本 | 无硬性场数 | 🆕 每案 **8–10 场** · Card 字段 `scene_count_target` |
| E-09 | 单案知识讲解占比 | 未量化 | 🆕 **≤5%/案**（卷末实验页另计）；情绪配比：钩子20%+舆论25%+验证25%+幽默15%+科学10%+修复5% |
| E-10 | 禁止项（继承） | 禁超自然/大人代解/说教体/抓造谣者/rival社 | ➡️ **全部保留** · V2 追加：信息污染案须 **可反驳节点** |
