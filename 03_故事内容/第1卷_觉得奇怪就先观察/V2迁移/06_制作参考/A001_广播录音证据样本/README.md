# A001 · 广播录音证据样本 · 制作参考

> **用途**：科学顾问 P0 · 插画/音效 · E20 公平线索 FC-2/FC-3  
> **禁**：真实羞辱语句 · 真实学生录音未经同意

---

## 1. 文件清单

| 文件 | 说明 |
|------|------|
| `generate_sample.py` | 生成演示 WAV（完整句 + 硬切版） |
| `rehearsal_full.wav` | 运行脚本后生成 · 完整排练句 |
| `broadcast_cut.wav` | 运行脚本后生成 · 模拟误播硬切 |
| `waveform_spec.md` | FC-3 波形断点标注说明 |

---

## 2. 音频规格（与正文一致）

| 项 | 值 |
|----|-----|
| 采样率 | 32000 Hz |
| 位深 | 16-bit PCM |
| 完整句文案 | 「迟到的人不该参加今天的彩排。」（中文演示 · 日稿另译） |
| 误播播出 | 「不该参加。」或「水野根本不该参加。」（截断版） |
| 硬切位置 | 「不该参加」前 **0.3 s** 静音后硬起 |
| 压缩 | 可选二次导出 MP3 64kbps 再切（更贴近 PA） |

---

## 3. 生成命令

```bash
python "03_故事内容/第1卷_觉得奇怪就先观察/V2迁移/06_制作参考/A001_广播录音证据样本/generate_sample.py"
```

依赖：Python 3 · 标准库 `wave` · 无第三方包。
