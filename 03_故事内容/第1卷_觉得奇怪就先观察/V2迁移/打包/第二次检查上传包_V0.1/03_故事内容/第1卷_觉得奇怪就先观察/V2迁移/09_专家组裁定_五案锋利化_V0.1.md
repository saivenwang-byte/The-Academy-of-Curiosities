## 裁定 A001 ·「听见了 vs 不可能说过」

**问题**: 仅「唇未动 + 旧录音」只证非现场，未兑现标题「不可能说过」。

**专家组选择（LOCK）**: **双重不可能**

| 层 | 内容 |
|----|------|
| **生理不可能** | 彩排日上午，光因 **咽头发炎** 在 **保健室登记**，医嘱 **当日禁止发声**；全班可见他张口只有气声/手语 |
| **录音不可能** | 播出为 **三周前** `rehearsal_0328.wav` + **波形硬切**；完整句≠传播句 |
| **传播句** | 「水野根本不该参加。」← 由「迟到者不该参加今天的彩排」截断误听 |

**hook 定稿**: 广播里是他的声音——可 **他今天根本说不出话**。

**可复制话题句**: 「全班都听见他说了，可医生说他今天一个字都不能讲。」

## 裁定 A005 · 卷终洗冤焦点

**问题**: 五人全无影 → 稀释「救一个孩子」的刺痛。

**专家组选择（LOCK）**: **主洗冤者 = 水野真帆**

| 层 | 内容 |
|----|------|
| **画面** | 预展合照五人都在；**仅水野脚下无影**，其余四人及器材车影均在 |
| **误会** | 「只有她有问题 / 她带来诅咒」→ 观察社被疑 **帮她遮丑** |
| **机制** | 全景拼接 · 她拍摄中 **短暂侧让** + 侧光变化 → 她的地面阴影被相邻帧覆盖 |
| **情绪** | 承接 A004「抽屉里的失物」· 卷级误传链 **落到具体一个孩子** |
| **修复** | 重拍后 **她的影子回来**；壁报写回水野名字 |

**八秒传播句**: 「合照里只有一个人的影子不见了——大家都说是她干的。」
