## 3. 建筑/空间 · 专家组（插画前置）

> 对应 IP 记忆：「建筑相关专家 · 早期插图阶段」→ 正典化为 **固定席位 + 文献锚点** · 每案 **仅在新空间出现时开会**

### 3.1 专家组构成

| 席位 | 职责 | 正典锚 |
|------|------|--------|
| **空间结构** | 片廊下/引き戸/动线是否可读 | 09 `02_校园空间与建筑` · Macaulay 式结构图解原则 |
| **日本校园文化** | 上履き/下駄箱/给食动线 | `docs/world_reference/02_JAPAN_SCHOOL_*` · E04 文化审 |
| **机制可见性** | 公平线索能否「看出来」 | Case Card · `academy-visual-auditor` P0 |
| **分镜导演** | S/C/E/L/P/H · 景别机位 | `Vol1_分镜头电影化构图规范_V1.0` |
| **插画美工编辑** | prompt §11 · 发包 · 连续性 | `01_画师发包清单` · `00_单案分镜头交付模板` |

### 3.2 输出物（每新空间一次）

1. **空间 brief 1 页**：平面引用（链已有 SVG/HTML）· 新道具尺寸 · 光向 · 四月时间  
2. **P0 可读清单**：机制在画里必须可见的 3–5 项  
3. **Shot Map 增补行**：仅新场/新 DA · 不重写已有教室/侧廊行  
4. **画师发包段**：英文 prompt 块 + Negative · 链 Global Style  

**门禁**：brief 未签字 → **禁止** G1 探索稿标 PASS

## 4. 工作流（与 V1.0 一致 · 不增 Phase）

```
G-CN / G-JP 定稿段落
    ↓
Shot Map（链 SC · 优先复用已有 DA 编号与背景）
    ↓
【若新空间】建筑专家组 brief（§3.2）
    ↓
深度锚点 / prompt §11（academy-illustration-pipeline Phase 6–7）
    ↓
G1 探索 → E06-S → visual-auditor → 精修 PNG
```

**与 V1 唯一差**：V2 在 Phase 6 前多一步 **「查 §1 资产表 → 能复用则禁止重画建筑」**。
