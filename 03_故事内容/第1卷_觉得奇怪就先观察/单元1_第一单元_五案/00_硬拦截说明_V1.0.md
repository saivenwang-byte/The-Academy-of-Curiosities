# 硬拦截说明 · V1.1

> **原则（2026-06-11）**：审查 **只过 / 退回** · **禁止 pending 卡全线**  
> **脚本**：`tools/workflow_preflight.py` · `tools/guard_protected_path.py`

---

## 两种模式

| 模式 | 命令 | 用途 |
|------|------|------|
| **produce** | `--mode produce`（默认） | 设计部出图 · 写 prompt · 内部样张 |
| **deliver** | `--mode deliver --phase review-pack` | 试读 PDF · 审阅包 · PRODUCT 路径 |

---

## produce · 顺序（exit 1 = RETURN，不 exit 2）

| 序 | 检查 | RETURN 代码 | 退回工位 |
|:--:|------|-------------|----------|
| 1 | 编+导 brief `editorial_verdict: PASS` | `EDITORIAL_REVIEW_REQUIRED` / `EDITORIAL_RETURN` | 编+导 |
| 2 | 译部分镜审核 `translation_verdict: PASS` | `TRANSLATION_REVIEW_REQUIRED` / `TRANSLATION_REVIEW_RETURN` | 译部 / 编+导 |
| 3 | prompt `g_cast_prompt_gate`（generate-image） | `GCAST_PROMPT_FAIL` | 设计部改 prompt |
| 4 | **成图后** `illustration_post_gate`（promote-image） | `ILLUSTRATION_POST_FAIL` | 设计部重出+填 manifest PASS |
| 5 | manifest 仅 **PASS+canonical** 可绑 PDF/审阅包 | `ILLUSTRATION_CANON_BLOCK` | 见 `illustration_canon_manifest.json` |

**不检查（produce 默认）**：M0-B · G-AB-JP · 导演表 `[ ]` 双签 · COUNT_PASS · **成图像素内容（须 post-image 人工+manifest）**

---

## deliver · 顺序（试读交付前 · exit 1 = 退回）

| 序 | 检查 | RETURN 代码 | 退回工位 |
|:--:|------|-------------|----------|
| 1–2 | 同 produce 编+导 + 译部 PASS | 同上 | 同上 |
| 3 | M0-B 田中签 | `DELIVER_RETURN_M0B` | 翻译部 |
| 4 | G-AB-JP PASS | `DELIVER_RETURN_GAB` | 翻译部 |
| 5 | COUNT_PASS | `GCAST_COUNT_RETURN` | 设计部/导演组 |

---

## 裁决格式（替代空 `[ ]` 网关）

**分案** `A00X/02_分镜头/00_G-BRIEF_双签_A00X_V1.0.md` frontmatter：

```yaml
editorial_verdict: PASS    # PASS | RETURN
translation_verdict: PASS  # PASS | RETURN
return_to: editorial
return_notes: ""
```

**单元台账**：[`00_译部分镜审核_单元1_V1.0.md`](./00_译部分镜审核_单元1_V1.0.md)

COUNT_PASS：`→ [x] PASS 2026-06-10` 或 frontmatter `count_pass: true`
