## 0. 核心机器（必须按序）

```
① 校园小趣事（可采访 · 可投稿 · 零恐怖）
    ↓
② 观察社为写稿而采访 / 观察（校刊外壳 · 非侦探社）
    ↓
③ 日常异常 · 第一误判
    ↓
③b 【可选】校内传播 / 谣言变形（うわさ · 传话 · 无署名投稿）
    ↓ 观察社澄清 = 观察+验证+核实后写稿 · **非**抓造谣者
    ↓
④ 跨学科验证（≥2 原理 · 公平线索 ≥3）
    ↓
⑤ 角色分工补足
    · 陸珣 — 空间 / 路线 / 结构
    · 慧美 — 人的情绪 / 证据校准
    · 志郎 — 动手验证（可能想复杂）
    · 光 — 推动行动（可能先错）
    ↓
⑥ 温柔人际真相（情感锤 · 不可替换科学谜底）
    ↓
⑦ 陸瑆日记 / 插画再创作（笔记层 · 不参与破案）
    ↓
⑧ 可选：家庭实验 1 页（日/中 · 禁 EN-only）
    ↓
⑨ 趣事尾钩（L1）· 本案闭合 + 下案轻疑问 → [`00_趣事尾钩规范_V1.0.md`](./00_趣事尾钩规范_V1.0.md)
```

**禁止**：从「本周学物理」反推谜面 · 超自然 · 大人代解核心谜 · 说教讲义体 · **主谜题未完待续** · **每案锁定造谣者/捣蛋主谋**（见 §1b）。

## 1b. 传播对立层（うわさ · 无源收束 · LOCKED）

> **主编裁定** 2026-06-04：**对立面优先是传播本身，不是固定反派社团或个人。**  
> **关联**: 题材 **F 域** · Vol8 传播误会 · [`11_常驻对照角色_草案卡_校报标题担当.md`](../01_角色设定/11_常驻对照角色_草案卡_校报标题担当.md)

### 典型流程（与 IP 一致）

```
校园怪事/趣事
    → うわさ / 夸张标题 / 伝言ゲーム式变形
    → 观察社：采访 · 观察 · 验证 · **核实后再写**
    → 科学/机制澄清 + 温柔人际真相
    → 谣言源头：**可以找不到**（无源收束合法）
```

### 是 / 不是

| 是 | 不是 |
|----|------|
| **澄清误会**（机制+感受） | 刑侦式 **「辟谣抓人」** |
| **无署名投稿** · 传话链 · 查无此人 | 每案必有 **造谣者/捣蛋主谋** |
| **学級新聞 / 放送 / 掲示板** 放大（制度层） | 第二个固定 **对立社团** + 3–5 人 cast |
| Vol8+ **可选** 具象「标题担当」一次 | 观察社 vs 捣蛋社 **善恶对抗** |

### 叙事用语（现象/栏目 · **非 rival 社名**）

| 中文 | 日文工作稿 | 用途 |
|------|------------|------|
| 校内谣言/误传 | **うわさ** · **伝言**（误解 · 非恶意デマ） | 无源 · 链式变形 |
| 传话游戏 | **伝言ゲーム** | 一句变十句 |
| 午后广播 | **午後の校内ニュース** | 制度放大 |
| 班级报纸 | **学級新聞** | 比观察社草稿更「正式」 |
| 掲示板纸条 | **掲示板のメモ** | 无署名投稿 |
| 观察社壁报 | **今週の校内おもしろ** | 本方 · **核实后刊** |
| 旧栏目（长线 L06） | **おもしろ欄** | Vol17 旧校报 |

**社团正典名不变**：**学校おもしろ観察クラブ** / 校园趣事观察社 · 废止 **怪事研究社**（素材库旧称）。

### 写稿自检

- [ ] 本案若含传播线：科学谜底是否 **独立于**「谁传的」？  
- [ ] 收束是否 **温柔澄清**，而非羞辱/指控传播者？  
- [ ] 若源头未现：是否仍 **公平闭合** 机制+人际层？  
- [ ] 是否误设 **rival 社团** 或固定捣蛋组织？

## 1. 科学谜底 × 人际真相（湿椅子级裁定）

| 层 | 作用 | 规则 |
|----|------|------|
| **科学谜底** | 骨架 · 公平线索 · 可复现实验 | **不可**被人际故事替换 |
| **人际真相** | 情感锤 · 温柔收束 | **叠加**在机制之上 |

示例（B-3 湿椅子素材）：

- 科学：结露 · 湿度 · 传导 ·  airflow  
- 人际：为何被误会 / 谁不好意思说 / 善意被藏住  
- **禁**：「纯湿毛巾、无结露」版

| `month_nagoya` | 季节锚 |
| `science_core` + `science_cross` | 原理 · 去重表校验 |
| `fair_clues` | ≥3 |
| `misread` | 第一误判 |
| `propagation_layer` | 传播对立：**none** / **rumor** / **broadcast** / **submission** / **title_distortion**（见 §1b） |
| `source_resolved` | 谣言源头是否锁定：**yes** / **no** / **partial** · **no 合法** |
| `human_truth` | 温柔真相（情感锤自检） |
| `char_lead` | 主导角色 |
| `kei_note_angle` | 瑆误读/再创作角度 |
| `foreshadow_id` | Lxx 或 — |
| `gate_pass` | Remap Gate 七项 Y/N |
| `tail_hook_grade` | L1–L4 · 见 [`趣事尾钩规范`](./00_趣事尾钩规范_V1.0.md) |
| `tail_hook_type` | 投稿/道具/瑆误画/中谷/理紗影子/壁报 |
| `tail_hook_text` | 篇尾 1–2 句 |
| `tail_hook_fair_in` | 公平线索在本案何处已出现 |
| `next_case_id` | 承接 Axxx |
| `day_mon` … `day_fri` | 周节拍各日推进要点（可选细纲） |

## 4. 十域场域地图（A 线 100 案 · 排期辅助）

**域 = 场域 × 机制**，不是学科目录。

| 域 | 名称 | 主场景 | 知识底层 |
|----|------|--------|----------|
| A | 校园空间与物理 | 教室·走廊·窗 | 热·光·力·湿度 |
| B | 文具纸张与材料 | 壁报·黑板 | 纤维·静电·墨 |
| C | 食物与生活化学 | 給食·便当 | 氧化·发酵·扩散 |
| D | 植物昆虫与季节 | 校庭·花坛 | 行为·物候 |
| E | 声音音乐与感知 | 广播·体育馆 | 共振·回声 |
| F | 文字语言与传播 | 校报·投稿 | 谣言·排版·同音 |
| G | 游戏规则与社会 | 商店街·外校 | 规则·群体心理 |
| H | 通学与城市 | 通学路·车站 | 路径·地图 |
| I | 历史旧物与社区 | 葛西店 | 照片·史料 |
| J | 艺术审美与秩序 | 美术室·祭 | 构图·色彩 |

每域约 10 案 · 填表时 **先趣事后域**，不可先填「本学期物理」。
