# 技能路由决策树

```
用户意图？
├─ 「搜集/资料/参考/日语/气候/怪谈素材」
│   └─ academy-research-editor → 更新 09_ → 调度单
├─ 「大纲/分卷/阶段/整体规划/先讨论」
│   └─ academy-series-architect → 卷任务包（不写正文）
├─ 「写第X卷 / 出稿 / 实验设计」
│   ├─ architect 快速确认卷包（若缺）
│   ├─ research-editor 若资料缺口
│   └─ academy-engine → 初稿
│       └─ japan_campus_consultant_agent.html → 定稿
└─ 「校准/像中国学校吗」
    └─ 田中みどり + 知识库 + consultant HTML
```

## 子分身协作（推荐顺序）

**新开一卷**
1. series-architect：卷任务包
2. research-editor：素材调度单 + 补库
3. academy-engine：正文+实验
4. consultant：文化校准
5. architect：验收清单勾选

**用户只给关键词（如「逃げ水」）**
1. research-editor 主动扩展 → 入库
2. architect 标注「可用于卷3–5」写入速查

## 无需外装 skill

以下用项目内文件即可，不必下载第三方 agent：
- 文化校准：`japan_campus_consultant_agent.html`
- 创作规则：`创作红线与原则.txt`
- 人物：`00_项目总览/人物名称定稿.txt`
