## 0. 文档用途

本文件用于指导插画师进行 **原创** 角色设定、封面、内页插图、人物关系图、陆瑆日记页、观察社道具、校园场景和后续商品化视觉开发。

**本文件是视觉创作依据。** 插画师应：

* 仅根据本文 **文字描述** 进行原创设计；
* **禁止** 以任何具体知名漫画家、动画导演、插画师、漫画作品、动画作品、既有角色 IP 的名称或造型作为风格参考或垫图；
* **禁止** 在交付说明、图层命名、参考板中标注外部作者名/作品名；
* 与旧 AI 出图、旧排版图 **脸型冲突时，以本文 + V1.0 年级表为准**。

**本文不是**：简单提示词合集 · 某作品风格模仿指南 · 侦探 cosplay 设定集。

## 1. IP 视觉总定位

### 1.1 正篇视觉气质

**关键词（必须同时满足）**：

| 维度 | 要求 |
|------|------|
| 情感 | 温暖 · 明亮 · 有温度 · 不冰冷 |
| 舞台 | 真实日本小学校园 · 名古屋日常 |
| 叙事 | 日常观察 · 轻推理 · **趣事感**（非破案猎奇） |
| 读者 | 8–11 岁桥梁书 · 有儿童可读性 |
| 边界 | **不恐怖** · **不成人化** · **不低幼化** |
| 商业 | 强符号识别 · 可贴纸/徽章/文具延展 |

**气质句**：

> 像孩子自己班门口发生的事——奇怪，但可以观察、可以验证、可以温柔收束。

### 1.2 三层视觉系统

#### A. 正篇故事画风

**用于**：正文插图 · 封面 · 人物主视觉 · L0 十人排面

| 要求 | 细则 |
|------|------|
| 比例 | 小学生 **真实年龄感**（见 §2.3）· 轻漫画化表情 |
| 场景 | 可探索 · 线索物件 **清楚可辨** |
| 色彩 | 明快 **但不幼稚** · 低–中饱和 |
| 温度 | 水彩/彩铅感暖色 · 不要医疗或法医式冷灰 |

#### B. 陆瑆日记画风

**用于**：篇末日记 · 手账页 · 贴纸页 · 表情包延展

| 要求 | 细则 |
|------|------|
| 简化 | 比正篇 **更圆、更短、更概括** |
| 手绘感 | 允许儿童笔迹、修正痕、贴纸边 |
| 拟人 | 椅子、黑板、书本、便当盒等可画成 **圆圆小物/小怪物**（无恐怖） |
| 商品 | 适合 2–3 cm 贴纸、书签、小图标 |

#### C. 观察社符号系统

**用于**：社团徽章 · 投稿箱 · 壁报 · 线索卡 · 路线图 · 角色小道具

**社团正名**：**学校おもしろ観察クラブ**（校园趣事观察社）

| 要求 | 细则 |
|------|------|
| 简洁 | 单线可识 · 缩小仍可读 |
| 重复 | 全系列同一套几何语言 |
| 印刷 | 适合单色/双色印刷、刺绣、热转印 |

**符号母题建议**：眼睛（观察）· 问号（奇怪）· 路线折线（动线）· 空白栏（待核实）—— **禁止** 放大镜+侦探帽组合。

### 1.3 原创 IP 识别目标

画风目标 **不是**「画得像某个经典 IP」，而是建立 **《学堂趣事录》自有视觉系统**：

> **小学生真实年龄感 + 强符号角色设计 + 温暖校园真实感 + 轻漫画化表情 + 陆瑆日记贴纸化视觉层**

## 2. 基础画风规范

### 2.1 线条

| 要求 | 可执行标准 |
|------|------------|
| 主线条 | 清晰 · 外轮廓 **略重** · 衣褶/内线 **略轻** |
| 线色 | 暖深棕 `#2A1810`（**禁纯黑 `#000000` 勾全身**） |
| 线宽（3840px 宽基准） | 外轮廓 1.0–1.4 px · 五官 0.6–0.9 px · 衣褶 0.5–0.8 px |
| 性格 | 柔和 · 略圆 · 允许 **极轻** 手绘抖动感 · **禁潦草** |
| 儿童线 | 无攻击性尖角 · 下巴线钝圆 |
| 道具线 | 线索物件 **闭合清楚** · 孩子能指认 |

**避免**：过度写实线稿 · 成人漫画式硬阴影线 · 复杂肌肉骨骼 · 华丽装饰线堆砌

### 2.2 上色

| 要求 | 可执行标准 |
|------|------------|
| 整体 | 温暖 · 清新 · 明亮 |
| 季节 | 日本 **春季** 校园 + 名古屋城市生活 |
| 饱和 | **低–中** · 禁荧光 |
| 阴影 | 柔和 · 同色系加深 15–25% · **禁** 强烈暗黑对比 |
| 识别 | 每人 **主色稳定**（§6.2） |

**建议色名与参考 HEX**：

| 色名 | HEX | 用途 |
|------|-----|------|
| 奶油白 | `#F7F3EE` | 纸感/背景 |
| 樱花粉 | `#F5DDE8` | 四月点缀 |
| 校园米黄 | `#E8D8C0` | 走廊/木桌 |
| 草地嫩绿 | `#7AB88A` | 自然/志郎辅色 |
| 晴空蓝 | `#5A82CC` | 珣/窗光 |
| 橘红 | `#E87830` | 光围巾 |
| 橄榄绿 | `#6A7050` | 中谷/户外 |
| 海军蓝 | `#3A5080` | 珣/理紗 |
| 暖棕 | `#8B5A2B` | 木牌/老物 |
| 纸张浅黄 | `#F5EDE0` | 瑆日记 |

**避免**：赛博霓虹 · 黑暗悬疑色 · 少年战斗漫画冲击色 · 过度甜腻糖果色

### 2.3 头身比例

| 身份 | 建议头身 | 备注 |
|------|----------|------|
| 四年级（瑆） | **5.5–6.0** | 全队最矮 · 圆脸 |
| 五年级（观察社核心） | **6.0–6.5** | 仍儿童 · 禁少年尖脸 |
| 六年级（中谷） | **6.5** | **精瘦** · 仍小学生 |
| 成人（父母） | **7.0–7.5** | 肩宽臀线明确 |
| 老人（葛西） | **6.5–7** | 略佝偻 · 亲和 |

**要求**：

* 孩子 **必须像小学生**，不像初中生/高中生；
* 可略漫画化，**L0 正篇禁止 Q 版二头身**（Q 版仅瑆日记轨）；
* 手脚可 **略大** 一点，增强动作感，但禁比例失调。

### 2.4 表情

| 原则 | 细则 |
|------|------|
| 明确度 | 比普通写实插画 **更可读** |
| 个性 | 每人有 **固定表情倾向**（见 §4） |
| 幅度 | 可夸张 · **禁** 低幼恶搞/鬼脸 |
| 类型 | 喜剧与温柔 **并存** |

**通用表情库（可组合）**：

发现异常 · 认真观察 · 想错了 · 恍然大悟 · 不好意思 · 偷偷开心 · 轻微生气 · 温柔理解 · 嘴硬但关心 · 记录时眼睛发亮

## 3. 角色识别总原则

每个角色必须通过 **三项测试**：

### 3.1 黑影测试

涂纯黑剪影后，仍可通过 **发型轮廓 + 服装轮廓 + 道具 + 姿态** 识别。

### 3.2 三秒测试

儿童读者看 **3 秒**，能大致说出其「观察世界的方法」：

| 角色 | 读者应感到 |
|------|------------|
| 陆珣 | 安静观察路线的人 |
| 陆瑆 | 记录日记的人 |
| 伊藤光 | 行动派、先冲出去的人 |
| 加藤慧美 | 温柔采访、倾听的人 |
| 松本志郎 | 查资料验证的人 |
| 中谷琦 | 街区前辈、校外边界 |
| 山本理紗 | 秩序感、对齐细节的人 |

### 3.3 小图标测试

缩至 **贴纸/徽章/头像**（约 32×32 px）仍可识别。

**每人必须拥有固定项**：

脸型 · 眼型 · 发型 · 主色 · 服装轮廓 · 标志道具 · 姿态倾向 · 表情倾向

## 4. 主要儿童角色视觉设定

### 4.1 陆珣（陸珣）

| 项 | 内容 |
|----|------|
| **身份** | **5年2組** · 第一主角 · Vol1 新加入 **学校おもしろ観察クラブ** |
| **功能** | 空间路线 · 观察现场 · 动线还原 · 读者入社入口 |
| **L0 身高** | **142 cm**（槽②） |

**视觉关键词**：安静 · 专注 · 路线感 · 空间感 · 刚入社的微妙距离感

**脸型**：微尖圆脸 · 下巴 **不要太尖** · 轮廓干净 · 略内向 · 略驼背显肩前

**眼型**：中等偏大 · 眼尾 **略下垂** · 专注凝视 · **禁** 热血瞪眼

**发型**：黑色短发 `#1A1818` · 前额 V 形或不规则分叉 · 略乱（非时尚造型）· 剪影可识

**主色**：深蓝 `#3A5080` · 晴空蓝 `#5A82CC` · 少量黄 `#E8C848`（拉链/内衬/包带）

**服装（春·私服）**：

* 藏青/蓝连帽外套 · **黄色拉链或内衬** 露 1–2 cm
* 卡其工装短裤或九分裤 `#A89878`
* 运动鞋 · 上履き（室内场景）
* 深灰窄双肩包 `#505050`
* **禁** 中学生制服 · 潮牌 · 侦探风衣

**标志道具（P0）**：

* **路线观察本**（横线/方格 · **禁**「探案笔记」字样）
* 小指南针挂坠 `#A0A0A0`
* 折叠校园地图 · 铅笔 · 透明小尺

**固定姿态**：低头看地图/本子 · 手指指向地面或路线 · 常站队伍 **侧后方** 观察

**禁止**：普通帅气男主 · 侦探套装 · 西装 · 装酷 · 中学生体态

**一句话**：> 陆珣是「蓝色路线少年」——安静，但能看见别人忽略的空间关系。

**发型**：短波波或齐肩内扣 · 发尾可微翘 · **固定小花/星形/铅笔形发夹** `#F0A8B8`

**主色**：珊瑚粉 `#FFD0C0` · 奶油白 `#F5EDE0` · 少量暖橙

**服装**：

## 11. 禁止事项

### 11.1 风格与引用

* 使用具体知名漫画家、动画导演、插画师、作品名、角色 IP 作为风格参考
* 模仿知名角色脸型、服装、轮廓
* 在交付物中标注外部作者/作品名

### 11.2 叙事与气质

* 侦探帽、风衣、烟斗、放大镜 **主 pose** 等传统侦探套装
* 恐怖 · 血腥 · 超自然视觉
* 暗黑悬疑调色

### 11.3 角色误绘

* 画成中学生/高中生
* 过度 Q 版低幼（正篇/L0）
* 过度青春恋爱化 · 战斗化 · 科幻化
* 全员穿搭同质化 · 无道具 · 无主角中心
* 中谷 → 危险不良少年
* 理紗 → 普通冷淡优等生模板 · Vol1 强 C 位
* 慧美 → 无功能温柔背景
* 志郎 → 被嘲笑的书呆
* 光 → 普通运动男孩（**无橙围巾**）
* 珣 → 普通帅气男主（**无路线本**）
* 瑆 → 低龄卖萌 · 爱虫人设
* 美咲/直人 → 儿童身高
* 葛西 → **全藏/泉蔹误字**

## 13. 最终视觉目标

《学堂趣事录》的视觉目标 **不是**「普通日系小学生群像」，而是：

> **每个孩子都是一种观察世界的方法。**

| 角色 | 观察方法 | 视觉必须落到 |
|------|----------|--------------|
| 陆珣 | 路线 | 蓝黄 · 路线本 · compass · 侧观姿态 |
| 陆瑆 | 日记 | 粉橙 · 日记 · 贴纸轨 |
| 伊藤光 | 行动 | 橙围巾 · 前倾 · 口哨 |
| 加藤慧美 | 倾听 | 黄 · 银框镜 · 采访本 |
| 松本志郎 | 验证 | 绿红 · 圆框镜 · 查证卡 |
| 中谷琦 | 街区 | 橄榄 · 耳机/斜挎 · 精瘦 |
| 山本理紗 | 秩序 | 海军蓝 · 书签尺 · 对齐 |
| 葛西泉藏 | 记忆 | 花帽 · 旧地图 |
| 陆美咲 | 温度 | 围裙 · 便当 |
| 陆直人 | 结构 | 设计稿 · 钢笔 |

请把以上 **抽象功能** 全部落实到：脸型 · 眼型 · 发型 · 服装 · 主色 · 道具 · 姿态 · 表情。
