## 一、核心理念与总体风格

### 1.1 核心定位

本作品是 **「儿童桥梁书 × 科学观察绘本」**（含 **本格公平线索**，但 **非** 少年侦探团、非破案猎奇）。  
画面须同时承载三重功能：

1. **日本名古屋的真实日常生活感**（公立小学 · 私服 · 上履き · 通学路）  
2. **科学现象的严谨可辨性**（结露、静电、动线等须画对）  
3. **双层叙事结构**（正文写实观察 + 陆瑆笔记手账层）

### 1.2 总体风格关键词

写实手绘 · 水彩淡雅 · 铅笔/暖棕线条 · 空气透视 · 日常质感 · 克制的情感表达 · 科学图解清晰 · 手账拼贴质感（笔记层） · **轻符号可读性**（角色识别见 P0）

### 1.3 风格详细描述

画面整体呈现 **「安静观察」** 气质。

* **线条**：铅笔或炭笔手绘感，保留轻微不规则与笔触痕迹；轮廓线暖深棕 `#2A1810`，**禁纯黑 `#000000` 勾全身**；避免机械化矢量平滑。  
* **上色**：透明水彩淡彩，颜色浸润纸纹，有自然浓淡与水痕边缘；阴影同色系加深 15–25%，**禁** 死黑块。  
* **光影**：重视空气透视——远处偏蓝偏灰、边缘模糊，近处清晰温暖。  
* **细节**：丰富但不喧闹，营造「可以盯着看很久」的沉静感。  
* **色调**：低–中饱和，**禁** 荧光色与高对比数码感。  
* **留白**：适度，不填满每一寸，让读者目光有休息余地。  
* **符号层**（P0）：在以上写实基线上，每人保留 **固定主色、道具、姿态** 以便三秒识别——不是变成少年漫画主视觉。

### 1.4 严禁使用的风格参照

禁止模仿或引用任何在世或已故 **具体** 作家/插画师/动画的 **作品风格、角色造型、场景构图** 作为创作依据。  
禁止使用「像某某画风」作为沟通语言。  
所有风格描述必须以 **本手册 + P0** 中的技术性语言为准。

## 二、角色塑造规范

> **脸型、发型符号、主色、道具、禁止方向** → 以 [`00_插画师视觉创作说明书_V1.0.md`](./00_插画师视觉创作说明书_V1.0.md) §4–§6 为准。  
> 本节规定 **技法层** 的共性人体与笔触感。

### 2.1 人物比例与体型

儿童角色采用 **写实比例**，正篇 **非 Q 版**（Q 版仅陆瑆日记/贴纸轨）。

| 身份 | 身高参考（V1.0） | 头身（正典） |
|------|------------------|--------------|
| 四年级（陆瑆） | 132 cm | **5.5–6.0** |
| 五年级（观察社核心） | 142–155 cm | **6.0–6.5** |
| 六年级（中谷琦） | 152 cm · 精瘦 | **6.5** |
| 成人（陆美咲/陆直人） | 163 / 175 cm | **7.0–7.5** |
| 老人（葛西泉藏） | 158 cm · 略佝偻 | **6.5–7.0** |

体型须忠实日本同年龄段儿童数据：**禁** 成人化曲线、**禁** 过度纤细或肥胖 caricature。  
面部保留适度圆润——脸颊有自然过渡，下颌柔和。手脚比例正常；可按 P0 略放大手脚以增强动作感，但 **禁** 比例失调。

### 2.2 五官与表情

**基线（默认）**：

* 眼睛：单眼皮或内双为主，眼型偏细长；**禁** 闪瞳、**禁** 二次元超大瞳。  
* 眉：浓度适中；**禁** 极端柳眉或粗黑浓眉（伊藤光除外，见 P0）。  
* 鼻唇：鼻梁低柔、鼻头圆小；嘴唇偏薄，自然状态下嘴角微平或微扬。  
* 日常：「在观察」的平静——眼中有内容，但 **不** 默认漫画式夸张。

**角色符号例外（P0 §4 · 不得突破「像小学生」底线）**：

| 角色 | 允许的表情/眼型倾向 |
|------|---------------------|
| 陆瑆 | 眼 **略** 大、亮；惊喜、记录时眼睛发亮 |
| 伊藤光 | 上扬眼、粗眉；发现时可 **明显** 大笑、喊叫 |
| 加藤慧美 | 弯眼、倾听脸 |
| 松本志郎 | 圆眼镜；紧张/验证时眼亮 |
| 陆珣 | 专注凝视；少热血夸张 |
| 山本理紗 | 细长眼、克制 |

笑容可以含蓄，也可以按上表 **角色化**——关键是 **同角色跨页一致**。

### 2.3 发型

黑发或深棕，简单自然。男生短发/碎发；女生短发、齐肩或低马尾。  
**禁** 夸张刘海造型、染发、发胶造型感。  
发丝可稍显凌乱（风/刚脱帽），但 **每人轮廓须与 P0 固定发型一致**。

### 2.4 体态与动作

站姿重心稳定，不刻意摆 pose。走路微前倾、步伐轻快。  
陆珣可 **微驼背**；蹲观察时身体紧凑、手指稳定。  
**标志性动作**（如光举手发现、慧美侧耳听、理紗扶正歪物）须跨页 **完全一致**。

### 2.5 四季穿搭原则

符合日本小学生实际穿着：**私服为主**（Vol1），**上履き** 在走廊/地面，**禁** 在实验台/窗台。

* 色：素色为主——白、灰、藏蓝、深绿、卡其、米白、淡粉 + **每人 P0 主色**  
* **禁**：荧光色、大 Logo、卡通图案、中学制服风  
* 材质：棉麻，自然褶皱垂坠  
* 冬：围巾/羽绒厚度与织物纹理可辨  
* 夏：轻薄，风中有轻微飘动

## 三、场景绘制规范

> 建筑与动线细节须对照 [`09_日本参考资料库/02_校园空间与建筑/`](../09_日本参考资料库/02_校园空间与建筑/) · 名古屋条目见 [`05_地域与民俗_名古屋/`](../09_日本参考资料库/05_地域与民俗_名古屋/)。**不自行发明** 中国式/美式校园细节。

### 3.1 学校场景

* 校舍：3–4 层 RC，浅灰外墙  
* 教室门：引き戸；窗：アルミサッシ  
* 走廊：一侧教室、一侧开放窗（片廊下式）  
* 教室内：深绿黑板 · 钢管腿+合板课桌 · 淡绿/米色 PVC 地面 · 荧光灯两排  
* 玄关：下駄箱整面墙 · 名牌 · 白色上履き  

须可藏线索：壁报、投稿箱、值日表、图书标签、便签。

### 3.2 家庭场景

日式一户建：土間下沉 · 室内地板抬高 · 鞋柜/伞立 · 和洋折衷客餐厅 · 厨卫分离。  
儿童房：书桌、台灯、ランドセル/书包、书架。  
**温暖生活感**，禁样板房过度精致。

### 3.3 街景与通学路

名古屋市郊：窄街、电线杆、自贩机、民宅围墙、小公园、便利店、神社鸟居。  
Vol1（4 月）：樱花刚散、新绿；地面沥青/水泥，白色道路标识。  
生活痕迹：靠墙自行车、门口花盆、晾衣杆微动。

## 四、色彩与光影规范

### 4.1 整体色调

低–中饱和、温暖沉静。  
主色：暖灰、米白 `#F7F3EE`、淡绿 `#7AB88A`、浅蓝灰、木色、深蓝 `#3A5080`、墨绿。  
点缀：橘色围巾（光）、银框镜（慧美）、结露水滴反光。  
**禁**：纯黑大面积、荧光色、高饱和原色。

### 4.2 时间与光影（Vol1 = 4 月）

| 时段 | 特征 |
|------|------|
| 清晨 | 窗外冷蓝灰 · 室内暖黄灯 · 窗玻璃薄雾/冷凝痕 · 影长而淡 |
| 正午 | 光偏白 · 影短而浓 |
| 放学后 15:00–16:00 | 暖金黄斜光进教室 · 窗框影拉长 |
| 夜晚（如有） | 台灯暖黄 · 窗外暗蓝 · 墙影柔和 |

按卷月份调整见 `13_插画风格规范_v2.md` §3.1。

### 4.3 空气透视

远处（窗外山/街景）比近处 **偏蓝、偏灰、边缘模糊**。  
从教室窗内向外看、校门口望远方时 **必须** 体现。

## 五、笔触与线条规范

### 5.1 线条风格

铅笔/炭笔手绘感：轻微抖动、粗细变化、起笔收笔顿挫。  
暗部线加重，亮部线轻或断开。  
**人物轮廓线略粗于场景**，突出主体。  
线宽参考（3840 px 宽）：外轮廓 1.0–1.4 px · 五官 0.6–0.9 px · 衣褶 0.5–0.8 px。

### 5.2 上色方式

水彩淡彩，纸纹自然扩散。亮部留白，暗部 2–3 层淡彩叠加。  
色块过渡保留水痕，不刻意抹平。

### 5.3 禁止的数字感

**禁**：软件渐变填充、对称工具、完美直线、克隆笔刷重复贴图（如同一树叶笔刷铺满）。  
所有线条与色块须有 **手绘自然感**。

## 六、双层叙事视觉区分

### 6.1 正文页

写实水彩 + 铅笔/暖棕线条。画面是 **「透过镜头看到的真实世界」**。  
执行 P0 正篇画风 + 本手册 §二–§五。

### 6.2 笔记页（陆瑆笔记）

切换为 **手账拼贴** 风格：

* 底纸：微黄横线/方格 `#F5EDE0`  
* 字体：稚嫩但工整手写体（日文假名 + 少量汉字）· **禁印刷字体**  
* 元素：箭头、便签、胶带、拍立得式小插图  
* 图解：细线勾勒 + 手写标注  
* 氛围：三年级女生（**正典：四年级**）晚上台灯下整理记录  
* **允许**：更圆、更简、拟人小物、二头身贴纸化（**仅此轨**）

## 七、构图与镜头语言

### 7.1 景别

| 景别 | 用途 |
|------|------|
| 全景 | 学校、街道、教室全貌 |
| 中景 | 人物互动、讨论 |
| 近景 | 水珠、温度计、笔记字迹 |
| 特写 | 窗框接触点、铁板水珠分布 |

### 7.2 视角

以 **平视或微俯视** 为主——小学生眼高。  
**禁** 成人式高俯、**禁** 英雄式夸张仰视。  
可用陆珣视角：后排望窗，见伊藤光背影与椅背水膜。

### 7.3 留白与节奏

留白是呼吸。同页可 **一大场景 + 两小特写**（如教室全貌 + 窗框接触点 + 温度计）。

## 九、创作流程与审核标准

### 9.1 创作顺序

1. **Demo 图**：教室窗边四人互动（窗框、结露、铁板与玻璃片实验场景）—— Vol1 案① 机制  
2. Demo 过审 → **角色比例基准表**（并排 + 统一身高线）  
3. **L0 十人合辑**（每人含正/侧/背/道具/动作/表情）  
4. 教室平面图 + 氛围图 + 窗边细节  
5. 学校布局 + 关键场所  
6. 通学路 + 家庭场景  
7. 陆瑆笔记页模板  

### 9.2 每张图审核标准

- [ ] 比例符合 V1.0 年级身高表 + P0 头身  
- [ ] 光影符合故事季节/月份/时段（Vol1 = 4 月）  
- [ ] 场景符合日本小学/家庭/街道规范（对照 09 库）  
- [ ] 科学现象（结露等）机制正确（见 `13_v2` §5）  
- [ ] 正文 vs 笔记层区分正确  
- [ ] 同一角色跨页五官/发型/身高/动作一致  
- [ ] P0 三测试：黑影 / 三秒 / 小图标
