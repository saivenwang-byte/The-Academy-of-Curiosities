## 2. 四选一方案

### 选项 A · 观察クラブ徽章

| 项 | 内容 |
|----|------|
| **描述** | 圆形/盾形社徽 · 「学校おもしろ観察クラブ」 · 放大镜+笔记元素 · 名古屋校园色 |
| **优点** | 系列识别强 · 对标放課後MC 社团感 · 封面/腰封/周边统一 |
| **缺点** | 易做成 generic 社团 logo · A001 单案封面需另加事件钩子 |
| **栏③ 影响** | Sheet 角标 · 尾页社徽 · 六帧 UI 道具统一 |

### 选项 B · 陸瑆笔记本角标

| 项 | 内容 |
|----|------|
| **描述** | 瑆笔记页角标/贴纸 · 手写感 · 「？」「风→」符号 · 双层叙事入口 |
| **优点** | 差异化强 · 绑定妹妹线 · E20 可测「是否翻笔记页」 |
| **缺点** | 主叙事偏陸珣 · 首卷 A001 瑆戏份有限 · 家长层识别弱 |
| **栏③ 影响** | 笔记页版式 LOCK · DC1 壁报栏与角标并置 |

### 选项 C · 侧廊三联构图

| 项 | 内容 |
|----|------|
| **描述** | 固定机位：侧廊+掲示板+樱花/TV塔 · 每案换海报内容 · 系列「同一舞台不同谜」 |
| **优点** | 空间记忆强 · 名古屋 L0 容器 · 与 DA1 锚点一致 |
| **缺点** | 构图重复风险 · 非 A001 案需新场景时弱 |
| **栏③ 影响** | 封面系列模板 · G1 平面立面延续 |

### 选项 D · 风侧 / 翘边事件图形语言

| 项 | 内容 |
|----|------|
| **描述** | 抽象「翘边三角+风向箭头+摩擦屑」 · A001 事件符号可演变为每案一符号 |
| **优点** | **零售钩子最直接** · DA3/DB1 科学可视化 · 单案封面即懂 |
| **缺点** | 单案性强 · 需设计「符号族」以免每卷重新教育 |
| **栏③ 影响** | DB1 机制图风格 LOCK · 封面主视觉 · 腰封小图标 |
