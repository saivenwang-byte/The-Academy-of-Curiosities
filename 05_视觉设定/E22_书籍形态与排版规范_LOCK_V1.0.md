## 四、正文排版 LOCK

### 4.1 文图关系（P0）

- 插图 **混排** · 段落后 · `break-inside: avoid`
- **禁**整页单图（封面 · approved 跨页 spread ≤每案 1 除外）
- 每 **4–6p** 视觉变化 · 连续纯文字 **≤6p**

### 4.2 案首

- **半页标题条**（Q1=A）· **非**整页案件扉

### 4.3 日文书写与 **ルビ**（强制 · 对齐标日）

| 项 | LOCK |
|----|------|
| **书写体系** | 仅 **平假名（ひらがな）** · **片假名（カタカナ）** · **汉字（漢字）** — **无第四种体系** |
| **废止术语** | **「振假名」** · **「全振假名」** 及一切暗示「假名第四种书写类别」的说法 |
| **正确术语** | 读音标注 · **ルビ（Ruby）** · **振り仮名（ふりがな）** |
| **标日规范** | 辞书形·语法 · 小学 **五·六年级配当汉字** 为默读基线 · **尽量减少汉字使用频率**（能假名则不强行汉字化） |
| **ルビ 策略** | **选择性** · 首次 + 案首复现 · **非**竞品式全书每字注音 |
| **排版** | 横排试读 · ルビ 在汉字**上方** · 5–6pt |
| **详则** | [`日文读音标注策略_V1.1.md`](../../00_项目总览/日文读音标注策略_V1.1.md) |

### 4.4 字体（T1–T5）

| 级 | 用途 |
|----|------|
| T4 | 正文 · 11–12pt · **稳定** |
| T5 | 瑆/便签/壁报 · 手写横排 |
| T3+ | **表演字** · 见 §8.3 |

### 4.5 B 轨（§4 之后）

DB1 · CL · 实验 · V-S02 瑆 — **禁** §1–3 插入
