## LOCK 要点（必守）

| 项 | 规则 |
|----|------|
| 坐标 | **东向为上** · 活动准备室在北 · 西=走廊深 · 东=下駄箱 |
| 尺度 | 走廊 W≈2.4m · H≈2.8m · 海报 W≈0.9m |
| vent | 海报**正上方** · 气流**向下** · 翘边与 vent **同侧** |
| 两板 | **海报板**（DA 区）与 **壁报栏**（DC1 区）步行 **≥3m** · **不得同板** |
| 地面 | **上履き** · 浅灰瓷砖 · 全程 |
| 海报文案 | `学校おもしろ観察クラブ` · 简笔眼睛 · 手写感 |
| 壁报 | 三栏满 + **第四栏空** · pencil 草稿感 |
| 光线 | 4月下午 · 西向窗斜光 · 侧廊略冷 |
