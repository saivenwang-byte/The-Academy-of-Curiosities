## 4. 转发话术（复制即用）

### 4.1 标准版（L0 十人排面）

```
【学堂趣事录 · L0 十人排面 · 画师委托】

请按附件两份文字说明进行原创绘制：
1.《插画师视觉创作说明书 V1.0》— 全 IP 画风/角色/禁忌
2.《L0 画师文字创作说明书》— 本张排面槽位/身高/举牌/交付规格

任务：L0 十人温暖水彩定稿 · 3840×2160 · 白底 · 右侧身高线保留

硬性要求：
· 禁止引用任何外部漫画家/动画/既有 IP 名称或垫图
· 禁止沿用旧 AI 出图脸型；与文字说明冲突时以 V1.0 正典为准
· 葛西「泉藏」（禁全藏/泉蔹误字）；父母为明确成人体型
· 陆珣 = 路线观察本（禁侦探风衣/探案笔记 pose）
· 伊藤光 = 橙围巾 P0

回传：CHAR_lineup_L0_定稿_v1.psd + CHAR_lineup_L0_定稿_v1.png（分层见 brief）
首稿建议 7 个工作日内，有问题请先文字确认再动笔。

谢谢！
```

### 4.2 短版（已有长期合作画师）

```
请仅按《插画师视觉创作说明书 V1.0》+《L0 画师文字创作说明书》绘制 L0 十人温暖水彩定稿；
禁止引用外部作者名或旧图脸型；回传 3840 PSD+PNG。
```

### 4.3 标准版（A001 深度锚点 6 张）

```
【学堂趣事录 · Vol1 案① · A001 深度锚点 · 6 张精修】

请按附件执行：
1.《Vol1 插图深度标准_湿椅子抽象 V1.0》— 分镜满 / 线索入画 / A+B 双轨 / 张张能当内页
2.《A001 深度锚点包_6张_画师开工单 V1.0》— 逐帧 brief / 交付规格
3.《插画师视觉创作说明书 V1.0》+ L0 定稿 v1 脸库

任务：6 张出版级 PNG（4A 叙事 + 1B 机制 SUM + 1C 尾钩）
产出至 depth_anchor/ 目录；首稿 v0.1 线稿 → E06 审 → v1.0 定稿

禁止：占位灰框水准 · 侦探/探案 pose · 理紗抢 C 位 · 外部 IP 垫图
谢谢！
```
