## 五层 · 各管一事

| 层 | 学习对象 | 负责什么 | 用在哪 |
|:--:|----------|----------|--------|
| **L1** | 放课后（规格）+ 小屁孩（低阻力） | 章节书骨架 · 短段 · 4–6p 视觉变化 | T4 正文 · T5 瑆/便签/页边 reaction |
| **L2** | 神奇校车 | 知识/线索 **不进正文堆叠** | 路线 · 采访 · 验证 · 瑆（B 轨） |
| **L3** | 老鼠记者 | 字也是表演 | 拟声/案名/物理/光台词 · **≤5–8% 全书** |
| **L4** | 小王子 | 留白与气质 | 扉/空镜/收束/葛西/理紗 · **每案 1–2p** |
| **L5** | 哈利（标准版 subset） | 书是收藏物 | 封面/书脊/卷色/环衬/地图/尾钩 |

**日本直接竞品**《放課後ミステリクラブ》= **L1 规格与年龄** · 不承担 L3–L5 全球气质。

## Do / Don't（每层 3+3）

### L1 · 骨架

| Do | Don't |
|----|-------|
| 单栏横排 · 11–12pt · 短段 | 全书横线笔记本正文 |
| 4–6p 内必有图/栏/变化 | 连续 >6p 纯文字 |
| 瑆/便签用手写 T5 | 第三人称正文改第一人称 |

### L2 · 推理信息

| Do | Don't |
|----|-------|
| 每案 1 线索 + 1 验证/路线 | 每页边注百科 |
| B 轨在 §4 之后 | 图解插入 §1–3 揭晓前剧透 |
| 箭头/表格手绘感 | 教辅 stock 图 |

### L3 · 趣味字

| Do | Don't |
|----|-------|
| 咚/风/翘边等 **关键词** | 正文整段变色 |
| 案名半页条可表演 | >8% 版心文字图形化 |
| 瑆页可略高于 8% | 推理链关键词变形致误读 |

### L4 · 文学留白

| Do | Don't |
|----|-------|
| 一人+一物+大留白 | 空镜页堆箭头线索 |
| 水彩点状 · 低饱和 | 全书稀疏如绘本 |
| 卷首/案末/夜间瑆 | 占用 CL 公平首次线索 |

### L5 · 系列收藏

| Do | Don't |
|----|-------|
| 卷号色 · 书脊线 · 名古屋地图 | 初版复杂机关/档案袋 |
| 环衬校园线稿 | 复制 HP 构图/字体 |
| 封底下一案钩 | 五案名堆封面 |

## 页数 vs 面积（144–160p · 不混算）

| 页数占比 | 类型 |
|:--------:|------|
| 55–60% | 主阅读（含留白节奏） |
| 18–22% | 混排半页/小图 |
| 8–10% | 全页/跨页（**每案跨页 ≤1**） |
| 6–8% | 线索/地图/验证 |
| 5–7% | 瑆+实验 |
| ~5% | 前置/预告 |

**面积 LOCK 不变**：正文 65–70% · 图 15–18% · 瑆 7–10% · 线索 7–10%

## 案内节拍（22–26p · 与 §1–4 对齐）

半页标题条 → 异常 2–3p → 调查 6–8p → 线索 1–2p → 误推 3–4p → 验证 1–2p → 揭晓 2–3p → 收束 1–2p → 尾钩 1p → **B 轨**

**案首 = 半页标题条**（非整页扉）。
