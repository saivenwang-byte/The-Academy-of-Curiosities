## 这是什么

**分支点（Branch Point）** = 在继续大规模执行（写卷、批量出图、对外发布）之前，必须先拍板的一组前提问题。

本讨论回答一个问题：

> **《学堂趣事录 / 学堂奇事録 / The Curious Logbook》在什么条件下，才算「IP 成立」？**

「成立」可能有多种含义（创作正典锁定 / 样章可对外 / 商业可启动 / 法律可保护）。本分支不预设单一答案，先把条件摊开、分级、标状态。

## 讨论规则

1. **先讨论，后扩产** — 本分支 OPEN 期间，不新增「对外发布级」交付物（样章送审、商洽 deck 等），除非主编明确豁免。
2. **不覆盖正典** — 讨论稿中的「待决」项不得写入 `03_故事内容/` 或 L0 红线。
3. **定案即迁移** — 每条前置条件定案后：写入 `01_决策记录.md` → 必要时升格为 `00_项目总览/` 或 `docs/` 正典。
4. **关闭条件** — 主编签字「IP 成立定义 + 必达清单全部 PASS 或 ACCEPTED RISK」→ Status 改为 `CLOSED` → 整包移入 `00_归档/`。
