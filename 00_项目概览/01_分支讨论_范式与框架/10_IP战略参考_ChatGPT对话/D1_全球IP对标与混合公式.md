## 一、两类 IP（先分类再对标）

| 类型 | 是什么 | 代表 | 对本项目的参考价值 |
|------|--------|------|-------------------|
| **IP 平台** | 多角色、多品类、授权与渠道机器 | 迪士尼、漫威、三丽鸥、泡泡玛特 | **商品化、系列化、联名** 机制 |
| **故事/角色宇宙** | 固定世界 + 可复用单元结构 | 哆啦A梦、HP、宝可梦、柯南、小丸子 | **叙事结构、连载节奏、角色关系** |

**边界判断（对话原文）**

- **保利** → 文化产业/机构品牌，非「米老鼠类」角色 IP，不进对标清单。  
- **安徒生 / 西游记 / 福尔摩斯 / 小王子** → 文学/公版或半公版 IP；优势在 **全球再改编**，非单一公司授权链。  
- **泡泡玛特** → 平台；Molly、Labubu 等才是角色 IP。对儿童校园推理 IP，学 **造型资产化、系列化、收藏化、联名化**，不学其弱叙事。

## 二、筛选标准（对话共识）

全球知名度高 · 存活 **>10 年** · **多品类** · **多国家** · **多语言**。

## 三、对标优先级（第一梯队）

| IP | 学什么 | 不学什么 |
|----|--------|----------|
| **Pokémon** | 游戏/动画/卡牌/商品/线下体验闭环 | 战斗收集核心 |
| **Hello Kitty / Sanrio** | 弱叙事角色靠情绪+设计+授权活 50 年 | 完全无故事的极端 |
| **Disney 体系** | 公司型 IP 矩阵、乐园转化 | 美式 fairy tale 模板 |
| **Harry Potter** | 文学 IP → 影视/乐园/舞台/游戏/商品宇宙 | 魔法世界观 |
| **Doraemon** | **单元故事 + 固定角色 + 可扩展道具/机制** | 未来道具万能解 |
| **Crayon Shin-chan** | 儿童/家庭喜剧的低门槛长期传播 | 成人向 humor |
| **Peanuts / Snoopy** | 弱剧情、强性格、强符号化长期授权 | 报刊四格形态 |
| **Sesame Street** | 教育型儿童 IP 跨语言本地化 | 电视课程结构 |
| **熊出没** | 中国侧多国/多语言/多品类成熟案例 | 本土喜剧 tone |
| **POP MART / Labubu** | 当代潮玩：收藏机制、稀缺感、社交传播 | 盲盒伦理争议 |

**第二梯队（结构参考）**：名侦探柯南（推理单元）、樱桃小丸子（校园日常）、线条小狗（轻符号情绪商品化）。

## 四、混合公式（立项用 · 战略定稿）

> **不是抄袭拼盘，而是「结构基因 + 体验分工」清单。**

```
柯南   →  单案推理结构、公平线索、日常中的「不可解释小异常」
小丸子 →  校园/家庭日常质感、低门槛、可打断阅读
哆啦A梦 →  单元剧稳定、固定 cast、每集可独立又归属同一世界
小王子 →  精神余味、温柔真相、不说教的价值层
三丽鸥/线条小狗 →  可商品化的符号化角色与情绪设计
```

**一句话混合定位（内部用）**

> 用 **柯南式结构** 承载 **小丸子式日常**，以 **哆啦A梦式单元稳定** 连载，留 **小王子式余味**，并用 **三丽鸥式符号** 支撑长期商品化——而非做硬科普或成人破案。

与 B1 对齐：**抢「孩子愿意读、家长觉得值」的阅读成长市场**，不抢教辅。

## 五、为什么「有很多零件却长不大」

对话归纳的失败模式（立项自检）：

| 有 | 缺 | 结果 |
|----|-----|------|
| 可爱角色 | 稳定 **单元结构** | 单篇好看，无法系列化 |
| 知识点/谜题 | **可复用体验**（六体验 · 见 A2） | 像上课，孩子不自选 |
| 单卷故事 | **三重用户同时成立**（见 A4） | 孩子/家长/平台只满足其一 |
| 文字成功 | **跨媒介资产**（角色/空间/线索视觉） | 无法动画/商品延展 |
| 一次性爆款 | **习惯回路**（见 A6） | 无回归动机 |

**自检句**：我们是在堆「零件」，还是在建 **结构 + 体验 + 用户 + 资产** 四条增长轴？
