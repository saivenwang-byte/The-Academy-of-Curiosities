## §1 三层模型（战略定稿）

```
┌─────────────────────────────────────────┐
│  新闻社外壳   │  采访、记录、校刊/壁报——合法、低越界 │
├─────────────────────────────────────────┤
│  观察社内核   │  看小细节、记怪现象、温柔还原 │
├─────────────────────────────────────────┤
│  推理故事引擎 │  每则「趣事稿」自然长成单集谜题 │
└─────────────────────────────────────────┘
```

**功能分工**

| 层 | 读者感知 | 创作功能 |
|----|----------|----------|
| 外壳 | 「他们在给学校写有趣稿子」 | 解释为何能在校内活动、为何不像警察 |
| 内核 | 「奇怪的事值得先看一看」 | 对齐 A6 现实观察护城河 |
| 引擎 | 「原来可以这样想清楚」 | 对齐 A7/B4 好奇钩子，非恐怖控制 |

## §2 定位句（对内 / deck 可用）

> 专门发现、记录、整理学校里 **看似小事、实则值得想** 的趣事与怪现象；  
> 可以像 **校刊/班级报纸** 一样对外，内核是 **观察与温柔真相**。

**口号候选（择一或并存）**

- 「学校里的小事，才不是小事。」  
- 「觉得奇怪，就先观察。」

日文口号待 L0 日文品牌文档统一时一并锁定；社名以 **おもしろ観察** 为轴，不用 **ふしぎ** 作主品牌（对话早期选项已弃）。

## §3 陸珣入社理由（战略层 · 与 L0 能力一致）

入社动机：**空间感 / 路线 / 视线还原** —— 能在现场把「谁经过、哪里湿、哪里不对」画出来或说出来。  
不是「天才侦探」或「被赋超能力」。

与 F1 一致：**能力是习惯与练习**，不是超能力（对话八项优化 #6）。

| 允许 | 禁止 |
