## C · 创作类

### C01 第一页钩子

**触发**：开篇 · 第一章 · 封面文案  
**归属**：`academy-engine`

- [ ] 前两页内出现 **小异常**（非长篇设定）  
- [ ] 读者能问「咦？怎么回事？」  
- [ ] 不像教辅/说明书开头  
- [ ] 符合 `10_/B4` 四钩 · 非操控（`S19`）

### C02 校园小异常生成（灵感）

**触发**：新案件 · Vol 目录 · 趣事稿灵感  
**归属**：`series-architect` · `research-editor`

输出：小异常 · 表面现象 · 可观察线索 · 误导 · 温柔真相（各一句）  
**可选**：空间/信息布局线索（Grabenstein · D05）· 团体驱动点（Blyton · D04）

### C03 轻推理结构

**触发**：案件大纲 · 单集完稿  
**归属**：`academy-engine` · 本格公平

- [ ] 谜面清楚  
- [ ] ≥3 公平线索 · 无超自然  
- [ ] 有误导 · 有反转  
- [ ] 真相合理 · 非纯知识点总结

**子项（标杆蒸馏 v0.2 · 新案必跑）**：

#### C03b 读者参与式公平线索（Sobol · D02）

- [ ] 解谜前 **≥3 条线索已在正文出现**，读者可回看  
- [ ] 无「最后一页才出现的关键新信息」  
- [ ] 误导 = 读者也会犯的错，非作者诈欺  
- [ ] 不采用测验/Quiz 口吻（「你猜对了吗？」）  
- [ ] 案⑤ 类未结案：只悬不骗 · 已亮线索不矛盾

#### C03c 卷级/系列钩子（Ron Roy · D03）

- [ ] 单册内 **多数小案完整收束**（Vol1：案①–④ 结 · 案⑤ 钩）  
- [ ] 册末有 **下一谜题承诺**（壁报待续 / Vol2 标题可接）  
- [ ] 小案标题 = 一眼可见异常（可扫读）  
- [ ] 册末情绪 **上扬**（合订/入社/成就仪式）  
- [ ] 重复 cast  comfort · 读者为 **再见观察社** 而来

#### C03d 儿童轻推理尺度（Hayamine + 富安 · D01 D08）

- [ ] 谜面 **校园可遭遇** · 非成人刑侦  
- [ ] 真相 = 小习惯/小疏忽/物理机制 · 非天才罪犯  
- [ ] 允许略神秘开场 · **≤2 页** 内落地（案④ 粉笔圈）  
- [ ] 无恐怖渲染 · 无实锤超自然 · 无夜间独自追疑  
- [ ] 对话是 **聊天中推理** · 非审讯腔

### C04 温柔真相

**触发**：结尾 · 真相设计  
**归属**：`academy-engine`

- [ ] 不只解释现象 · 有 **人的原因**  
- [ ] 有误会被解开 · 有共情 · 有余味  
- [ ] 陸瑆笔记可接（双来源 E1）  
- [ ] **吉竹式余味**（D06）：可问句收束 · 禁止老师式总结段「所以我们学到了…」  
- [ ] scare-then-laugh 后 **安心**（富安 · 案④）

### C05 对话功能密度

**触发**：初稿完成 · 文学审计 · 对话戏改稿  
**归属**：`academy-literary-audit` · `academy-engine`  
**标准**：文学执行标准 §13–§15 · 场级 20 项 #11

- [ ] 每段有效对话完成 **≥1 项**：推进调查 / 暴露性格 / 制造误解 / 改关系 / 给或藏线索 / 调节奏 / 产生幽默  
- [ ] 无复读已知信息（「椅子是湿的」「对，湿的」链）  
- [ ] 科学解释 **不平均分配**；慧美 ≤1 句对照提问 · 禁止三原理口播  
- [ ] 允许不完整句、打断、改口（儿童感）· 禁故意低幼/网络梗  
- [ ] 去掉角色名后 **≥3 句** 仍可辨是谁在说（与 P10 联动）

## P · 人物类

### P05 人物功能重复

**触发**：新增/改角色 · 新关系  
**归属**：`series-architect`

- [ ] 不可替代功能（对照 11_ §7）  
- [ ] 与现有角色不重复  
- [ ] 不能合并则拒绝新增  
- [ ] 不破坏主角最小化（6 人上限）

### P06 人物年龄感

**触发**：对白 · 能力 · 角色图  
**归属**：engine · visual-auditor

- [ ] 像小学 4–6 年级 · 非成人腔  
- [ ] 能力来自习惯/经验 · 非超能力  
- [ ] 对白符合年级（中文/日文分别检）

### P07 人物缺口设计（输出）

**触发**：小传 · 成长线  
**归属**：`series-architect`

输出：表层性格 · 生活来源 · 小缺口 · 触发点 · 补偿行为 · 成长方向

### P08 关系张力

**触发**：社团互动 · 对话戏  
**归属**：`academy-engine`

- [ ] 配对有互补或温和冲突  
- [ ] 有共同目标 · 有成长空间  
- [ ] Vol1 五案各 cast 功能不打架  
- [ ] **俱乐部感**（Blyton · D04）：有「我们的规矩/仪式/成果展示」· 非排他秘密  
- [ ] 入社/成员身份变化 **可读**（案② 珣）

### P10 团队功能不抢位

**触发**：Scene Card 分工 · 初稿 · literary-audit  
**归属**：`academy-character-director` · `academy-literary-audit`  
**标准**：文学执行标准整合附录 C · soul `never_write`

| 角色 | 不可替代 | 禁止抢位 |
|------|----------|----------|
| 加藤慧美 | 采访温度 · 情绪观察 · 证据格化 · 关系刹车 | 唯一测量员/逻辑链讲解员/「冷静理性」标签 |
| 松本志郎 | **第一个动手验证** | 长篇原理口播 |
| 陸珣 | 位置/重复/微观关联 | 社交串联主导 |
| 伊藤光 | 发现/传播/串联 | 微观细节独占 |

- [ ] 同一异常 **四人四种反应**（注意点+行动+语言）  
- [ ] 慧美台词中测量/逻辑 **<60%** 且无关系行为（拉袖子、不写名字、空栏规矩等）→ FAIL  
- [ ] 志郎在验证段 **≥1 次动手**（摸/移/贴/挡风实验）  
- [ ] 无旁白贴标签（「冷静理性的女孩」）  
- [ ] 标志动作（挠头/推眼镜）**限频** · 非 climax 连发（见 L15）

## L · 语言类

### L09 日文儿童文体

**触发**：日文稿 · 中→日翻译后  
**归属**：`academy-jp-voice-editor` · **硬门禁**

- [ ] 非翻译腔 · 非成人说明文  
- [ ] 短句 · 对话像日本小学生  
- [ ] 必要处拟声拟态 · 不过度低幼

### L10 汉字/ルビ

**触发**：日文儿童稿完成  
**归属**：`jp-voice-editor` · textlint（可选）

- [ ] 汉字密度符合目标年级  
- [ ] 难词有ルビ或假名替换  
- [ ] 抽象词不过多

### L11 拟声拟态（增补）

**触发**：动作/情绪/线索场景缺儿童感  
**归属**：`jp-voice-editor`

参考：じーっ · こそこそ · きらり · ぺたっ · しーん · ふわふわ — **按需** · 忌堆砌

### L12 细节有效性

**触发**：文学审计 · 关键场景改稿  
**归属**：`academy-literary-audit`  
**标准**：文学执行标准 §3 · 场级 20 项 #2

- [ ] 重要细节承担 **≥1 功能**：异常 / 性格 / 线索 / 氛围 / 预示 / 真实性  
- [ ] 无纯装饰（「四十张桌子/绿色黑板」式）  
- [ ] 禁抽象独撑（「很湿」「很冷」无身体/物理跟进）  
- [ ] 删除该细节 **不损失** 人物/线索/节奏 → 删

### L13 感官与比较层

**触发**：发现段 · 验证段 · 文学审计  
**归属**：`academy-literary-audit`  
**标准**：文学执行标准 §4–§6

- [ ] 每场景 **1–2 种主感官** · 不五感堆叠  
- [ ] 感官系 **当前视角人物** 会注意到的内容  
- [ ] 「不是…而是…」服务推理 · **非**每段机械重复  
- [ ] 比喻来自儿童经验（水瓶/室内鞋/操场/饭盒）· 物理准确  
- [ ] 与 [`描写变奏登记_VolX.md`](../../03_故事内容/) 上案 **入口不同**

### L14 比喻边界

**触发**：文学审计 · voice-editor 第三遍  
**归属**：`academy-literary-audit` · `academy-voice-editor`

- [ ] 无成人文学典故/过度心理比喻  
- [ ] 单段通常 **≤1 个最有力** 的比喻/拟人/排比  
- [ ] 同一卷内比喻来源 **分区轮换**（校园/食物/自然/运动）  
- [ ] 比喻为理解服务 · 不为炫技歪曲物理

### L15 全书变奏

**触发**：卷级复盘 · 开新案前 · 文学审计  
**归属**：`academy-literary-audit`  
**标准**：变奏标准 §35–§55 · 变奏 §20 checklist

- [ ] 已读 **上两案** `描写变奏登记` · 禁用项未触犯  
- [ ] 无机械近义替换（看→瞧→望 轮播）  
- [ ] 高频动作警戒（皱眉/点头/停步/深吸气）**单角色单章 ≤2**  
- [ ] 章末钩子类型与 **近两案** 不同  
- [ ] 线索发现方式非模板「珣忽然注意到」独占  
- [ ] 有意重复（主题意象）须 **第二次增义** · 否则删  
- [ ] 变奏专项 **≥10/12**（变奏标准 §十一主编表）
