## 0. 一句话

> **专家池不是加东西，是判断什么不该加。**  
> Agent 在关键场景 **主动介入**；Skill 负责 **重复性检查**；人类专家提供 **边界与试读**。

## 5. IP 总控 · 标准输出模板

新增人物 / 设定 / Vol 结构 / 世界观补丁时，总控 **必须先输出**（可简短）：

```markdown
## 总控裁定 · [议题]

| 项 | 结论 |
|----|------|
| 是否保留 | 保留 / 后置 / 降级单集 / 拒绝 |
| 不可替代功能 | （一句话）|
| 与谁重复 | 无 / [角色名] |
| 对主线影响 | |
| 对 8–10 岁市场 | |
| **建议不加的原因**（若拒绝）| |
```

**默认立场**：除非证明 **不可替代**，否则 **不加**。
