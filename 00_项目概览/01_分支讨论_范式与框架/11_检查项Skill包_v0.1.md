## C · 创作类

### C01 第一页钩子

**触发**：开篇 · 第一章 · 封面文案  
**归属**：`academy-engine`

- [ ] 前两页内出现 **小异常**（非长篇设定）  
- [ ] 读者能问「咦？怎么回事？」  
- [ ] 不像教辅/说明书开头  
- [ ] 符合 `10_/B4` 四钩 · 非操控（`S19`）

### C02 校园小异常生成（灵感）

**触发**：新案件 · Vol 目录 · 趣事稿灵感  
**归属**：`series-architect` · `research-editor`

输出：小异常 · 表面现象 · 可观察线索 · 误导 · 温柔真相（各一句）  
**可选**：空间/信息布局线索（Grabenstein · D05）· 团体驱动点（Blyton · D04）

### C03 轻推理结构

**触发**：案件大纲 · 单集完稿  
**归属**：`academy-engine` · 本格公平

- [ ] 谜面清楚  
- [ ] ≥3 公平线索 · 无超自然  
- [ ] 有误导 · 有反转  
- [ ] 真相合理 · 非纯知识点总结

**子项（标杆蒸馏 v0.2 · 新案必跑）**：

#### C03b 读者参与式公平线索（Sobol · D02）

- [ ] 解谜前 **≥3 条线索已在正文出现**，读者可回看  
- [ ] 无「最后一页才出现的关键新信息」  
- [ ] 误导 = 读者也会犯的错，非作者诈欺  
- [ ] 不采用测验/Quiz 口吻（「你猜对了吗？」）  
- [ ] 案⑤ 类未结案：只悬不骗 · 已亮线索不矛盾

#### C03c 卷级/系列钩子（Ron Roy · D03）

- [ ] 单册内 **多数小案完整收束**（Vol1：案①–④ 结 · 案⑤ 钩）  
- [ ] 册末有 **下一谜题承诺**（壁报待续 / Vol2 标题可接）  
- [ ] 小案标题 = 一眼可见异常（可扫读）  
- [ ] 册末情绪 **上扬**（合订/入社/成就仪式）  
- [ ] 重复 cast  comfort · 读者为 **再见观察社** 而来

#### C03d 儿童轻推理尺度（Hayamine + 富安 · D01 D08）

- [ ] 谜面 **校园可遭遇** · 非成人刑侦  
- [ ] 真相 = 小习惯/小疏忽/物理机制 · 非天才罪犯  
- [ ] 允许略神秘开场 · **≤2 页** 内落地（案④ 粉笔圈）  
- [ ] 无恐怖渲染 · 无实锤超自然 · 无夜间独自追疑  
- [ ] 对话是 **聊天中推理** · 非审讯腔

### C04 温柔真相

**触发**：结尾 · 真相设计  
**归属**：`academy-engine`

- [ ] 不只解释现象 · 有 **人的原因**  
- [ ] 有误会被解开 · 有共情 · 有余味  
- [ ] 陸瑆笔记可接（双来源 E1）  
- [ ] **吉竹式余味**（D06）：可问句收束 · 禁止老师式总结段「所以我们学到了…」  
- [ ] scare-then-laugh 后 **安心**（富安 · 案④）

## P · 人物类

### P05 人物功能重复

**触发**：新增/改角色 · 新关系  
**归属**：`series-architect`

- [ ] 不可替代功能（对照 11_ §7）  
- [ ] 与现有角色不重复  
- [ ] 不能合并则拒绝新增  
- [ ] 不破坏主角最小化（6 人上限）

### P06 人物年龄感

**触发**：对白 · 能力 · 角色图  
**归属**：engine · visual-auditor

- [ ] 像小学 4–6 年级 · 非成人腔  
- [ ] 能力来自习惯/经验 · 非超能力  
- [ ] 对白符合年级（中文/日文分别检）

### P07 人物缺口设计（输出）

**触发**：小传 · 成长线  
**归属**：`series-architect`

输出：表层性格 · 生活来源 · 小缺口 · 触发点 · 补偿行为 · 成长方向

### P08 关系张力

**触发**：社团互动 · 对话戏  
**归属**：`academy-engine`

- [ ] 配对有互补或温和冲突  
- [ ] 有共同目标 · 有成长空间  
- [ ] Vol1 五案各 cast 功能不打架  
- [ ] **俱乐部感**（Blyton · D04）：有「我们的规矩/仪式/成果展示」· 非排他秘密  
- [ ] 入社/成员身份变化 **可读**（案② 珣）

## L · 语言类（日文）

### L09 日文儿童文体

**触发**：日文稿 · 中→日翻译后  
**归属**：`academy-jp-voice-editor` · **硬门禁**

- [ ] 非翻译腔 · 非成人说明文  
- [ ] 短句 · 对话像日本小学生  
- [ ] 必要处拟声拟态 · 不过度低幼

### L10 汉字/ルビ

**触发**：日文儿童稿完成  
**归属**：`jp-voice-editor` · textlint（可选）

- [ ] 汉字密度符合目标年级  
- [ ] 难词有ルビ或假名替换  
- [ ] 抽象词不过多

### L11 拟声拟态（增补）

**触发**：动作/情绪/线索场景缺儿童感  
**归属**：`jp-voice-editor`

参考：じーっ · こそこそ · きらり · ぺたっ · しーん · ふわふわ — **按需** · 忌堆砌

## V · 视觉类

### V12 第一眼识别

**触发**：角色图 · 主视觉  
**归属**：`academy-visual-auditor`

- [ ] 剪影/黑影可辨  
- [ ] 小道具记忆点 · 彼此可区分  
- [ ] 适合贴纸/徽章化（`M17` 可选）

### V13 儿童画风适配

**触发**：插画 · 封面 · 群像  
**归属**：`visual-auditor` · **硬门禁**

- [ ] 桥梁书/儿童漫画感 · 非轻小说成熟脸  
- [ ] 非过低幼绘本 · 有趣事/线索入画  
- [ ] 4–6 年级身体比例（身高 LOCK 表）

### V14 主角视觉中心

**触发**：群像 · 封面 · 海报  
**归属**：`visual-auditor`

- [ ] 陸珣视觉中心或明确入口  
- [ ] 陸瑆有记录者特征  
- [ ] 观察社三人可辨 · 中谷外圈 · **理紗 Vol11 前零出镜**

## S · 安全类

### S18 儿童安全红线

**触发**：危险/街区/游戏/隐私/家庭敏感  
**归属**：Rule `04-creative-redline` · engine

- [ ] 不可模仿危险 · 不美化不良/暴力/成瘾  
- [ ] 不制造过度焦虑 · 中谷线不抢主角  
- [ ] **校园怪异** 不升级为 horror（富安 · 无实锤鬼怪 · 无夜探）

### S19 非操控式钩子

**触发**：连载悬念 · 互动/短视频设计  
**归属**：`10_/A7` · `B4`

- [ ] 好奇缺口 · 非焦虑控制 · 有回报 · 可自主停读

### S20 学校/家长接受度

**触发**：社团活动 · 校内调查描写 · 渠道进校  
**归属**：architect · 田中

- [ ] 观察社=E1 三层 · 非侦探跟踪  
- [ ] 无侵犯隐私/翻包/夜间独自外出（除非有成人边界）
