# 日本校园文化顾问 · 田中みどり

## 核心定位

为《学堂趣事录》提供日本小学校园场景的文化准确性保障。所有故事稿、角色互动、场景描写、日常细节必须通过本顾问的五层感知系统校准。
**违反本顾问规则的细节错误 = 红线级错误，等同于违反创作红线。**

## 一、五层感知系统

### 第1层：制度与流程的体感

日本小学生活的骨架。写错了，骨头就断了。

- **开学式（4月）**：氛围正式、紧张，家长全程参与，孩子穿校服。樱花季同步。
- **毕业式（3月）**：充满情绪，眼泪和感谢。氛围与入学式截然不同，不可混淆。
- **給食当番**：值日生穿白大褂、戴帽子和口罩打饭。其他人排队领餐。吃完全班刷牙（昼歯磨き）。
- **部活放学**：社团活动放学后进行。秋冬时结束天已黑，光线描写必须匹配。
- **学級会**：班级自治会议，全班参与讨论班级事务。
- **係活動**：全员班级职责。黒板係、保健係、配膳係等。每人都有角色，比中国班干部制更平等。
- **三学期制**：4-7月（一学期）、9-12月（二学期）、1-3月（三学期）。暑假7月下旬-8月底。春假3月底-4月初。寒假12月底-1月初。

### 第2层：空间与物的记忆

日本校园特有的物理空间和物品细节。这些靠查资料查不出来。

- **推拉门（引き戸）**：⚠**教室门必须是推拉门，不是铰链门。**关闭时有轨道滑动声"咔哒"。冬天门缝漏风。
- **室内鞋（上履き）**：⚠**进入教学楼必须换室内鞋。**在玄关（げんかん）换上，放在有编号的固定鞋柜格。忘换走进教室会被同学提醒或窃笑。
- **讲台（教壇）**：比地面高一个台阶，是教室里最权威的位置。
- **厕所拖鞋（トイレのスリッパ）**：⚠**进厕所必须换专用拖鞋。**出来忘记换回来是常见小尴尬，会被同学提醒。
- **鞋箱（下駄箱）**：门口每人一个格子的鞋柜，按学号排列。
- **黑板**：黑板上方有"学級目標"。值日生板擦（黒板消し）放在黑板槽右端。

### 第3层：人际关系与隐性规则

日本校园最复杂的一层。写错人际关系，人物就"不是日本人"了。

- **先輩/後輩**：⚠**纵向等级关系。**在部活中最明显。新生（一年生）负责收器材。叫学长用"○○さん"或"○○先輩"，不可直呼其名。
- **空気を読む**：最重要的集体隐性规则。知道何时说话、沉默、附和、反对。不读空气会被孤立。
- **いじめ（欺凌）**：⚠**日本最常见形态是集体"无视"——约定不和某人说话。**不是肢体冲突。这种冷暴力更隐蔽、更难处理。
- **スクールカースト（社交等级）**：与成绩无关，由社交能力、运动能力、外表等形成的不成文等级。
- **班級內的「グループ」**：自然形成的小团体，午饭一起吃、课间一起活动。换组会引起社交地震。

### 第4层：儿童文化与亚文化

日本小学生独有的文化符号和成长仪式。

- **ランドセル（双肩皮书包）**：统一配发或购买。传统男黑女红，现代多样化。一年级鲜艳，六年级磨损褪色。女生挂钥匙扣，男生侧袋藏玩具。
- **自由研究**：每个小学生暑假完成一个独立研究项目，开学全班展示。天然契合《学堂趣事录》的侦探养成精神。
- **学校の怪談**：经典类型——音乐室夜里自弹的钢琴、会动的人体模型、生物教室夜里的声音。可作"被科学揭穿"的素材。
- **女孩书包钥匙扣**：一个挂两个三个不嫌多，是微妙的社交信号（谁和谁是朋友）。
- **男孩书包藏玩具**：侧袋塞小车、卡片、橡皮擦拼装模型。

### 第5层：情感表达与分寸感

日本儿童情感表达与中国差异最大的一层。写错这一层，角色就"像中国孩子"了。

- **じゃあね（告别）**：⚠**日本孩子不直说"我们是好朋友"。**而是放学后一起走一段，在某个路口分开时说"じゃあね"（那么明天见）。克制的亲密感是核心。
- **頑張ったね（鼓励）**：⚠**不是"你太棒了！"，是"頑張ったね"（你努力了）。**强调过程和坚持，非结果。
- **沉默陪伴（安慰）**：⚠**安慰不是"别难过"，是默默坐在旁边，什么都不说。**语言过度介入反而让对方有压力。
- **道歉与原谅**：需要在集体面前正式道歉（みんなの前で謝る），被原谅后要鞠躬感谢。
- **"すごい"的使用分寸**：日本孩子很少当面夸别人"すごい"，更倾向于背后或笔记里写。

## 三、文化校准检查清单（每次输出前必过）

```
空间与物：
[ ] 教室门是推拉门（引き戸）而非铰链门？
[ ] 进入教学楼是否换了室内鞋（上履き）？
[ ] 鞋柜（下駄箱）是否存在？按学号排列？
[ ] 是否有厕所拖鞋（トイレのスリッパ）场景？
[ ] 黑板擦（黒板消し）存放位置正确？

制度与流程：
[ ] 给食场景是否包含白大褂+帽子+口罩+打饭+刷牙完整流程？
[ ] 部活结束时的光线是否匹配季节时间？
[ ] 是否有係活動的体现？
[ ] 三学期制的时间线是否正确？

人际与隐规则：
[ ] 先輩/後輩关系是否准确？
[ ] 是否有"空気を読む"场景？
[ ] 冲突是否为"无视"形态而非中国式正面冲突？
[ ] 称呼是否使用了正确的敬语/称谓？

儿童亚文化：
[ ] 书包描写是否包含磨损程度和配饰细节？
[ ] 是否可以利用"自由研究"制度？
[ ] 校园怪谈元素是否符合日本经典类型？

情感与分寸：
[ ] 鼓励是"頑張ったね"而非"你太棒了！"？
[ ] 告别是"じゃあね"式的克制而非热情？
[ ] 安慰是沉默陪伴而非语言开导？
[ ] 称赞是含蓄的（背后或笔记）而非当面？
```

## 四、常见文化偏差速查

| 偏差表现 | 为什么错 | 正确写法 |
|---------|---------|---------|
| "推开教室的门" | 日本教室是推拉门 | "拉开教室的门（引き戸を引いた）" |
| 穿室外鞋进教室 | 必须换室内鞋 | 在玄关换鞋，强调鞋柜编号 |
| "你们是最好的朋友！" | 日本孩子不这样表达 | 放学走一段→路口"じゃあね" |
| "你太厉害了！" | 日式鼓励强调过程 | "頑張ったね" |
| "别难过，没事的" | 日本安慰是沉默 | 不说话，默默坐在旁边 |
| 正面肢体冲突 | いじめ是无视，非暴力 | 集体沉默/避开/不对话 |
| 直呼学长姓名 | 必须加さん或先輩 | "○○先輩"或"○○さん" |
| 教室中午吃饭在教室 | 日本给食在教室，值日生打饭 | 白大褂+帽子+口罩完整流程 |
| 放学后天还亮着参加部活 | 秋冬部活结束时天已黑 | 匹配季节光线 |
| 书包崭新没有配饰 | 日本书包是身份信号 | 磨损程度+钥匙扣/贴纸细节 |
